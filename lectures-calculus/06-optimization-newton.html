1. 왜 최적화인가

한 줄 비유부터

"가장 효율적인 디자인", "가장 빠른 길", "가장 적은 비용". 일상의 모든 의사결정이 사실상 최적화 문제예요. 미분적분학은 이 문제들을 푸는 가장 강력한 도구를 제공합니다 — "극값에서는 도함수가 0"이라는 한 줄 사실입니다.

이번 주의 두 핵심 도구:

  • 해석적 풀이 — $f'(x) = 0$을 손으로 풀고 1계/2계 판정으로 분류 (W5 활용).
  • 수치적 풀이 — 손으로 $f' = 0$이 안 풀리면 뉴턴법으로 빠르게 근사.

2. 닫힌 구간의 절대 최대·최소 — 후보 비교법

$f$가 닫힌 구간 $[a, b]$에서 연속이면 최대값과 최소값이 반드시 존재(극값 정리, EVT). 그 후보는 다음 셋뿐.

  1. 구간 끝점 $a, b$.
  2. $f'(c) = 0$인 임계점 $c \in (a, b)$.
  3. $f$가 미분 불가능한 점 $c \in (a, b)$.

이 모든 후보의 함숫값을 계산하고 가장 큰 것·작은 것을 고르면 끝.

예제 2.1

$f(x) = x^3 - 6x^2 + 9x + 1$ 의 $[0, 4]$ 최대·최소.

  1. $f'(x) = 3x^2 - 12x + 9 = 3(x-1)(x-3)$. 임계점 $x = 1, 3$.
  2. 후보값: $f(0) = 1$, $f(1) = 5$, $f(3) = 1$, $f(4) = 5$.
  3. 최대 5 (at $x = 1, 4$), 최소 1 (at $x = 0, 3$).

3. 응용 최적화 — 모델링이 절반

실제 문제의 80%는 식 세우기. 다음 절차를 외우세요.

  1. 최적화하려는 양을 변수로 (예: 면적·둘레·시간·비용).
  2. 그 양을 한 변수의 함수로 적기 (제약조건을 이용해 변수를 줄이기).
  3. 정의역 (물리적·기하학적 제약).
  4. $f' = 0$ 풀어 임계점.
  5. 1계 또는 2계 판정으로 최대·최소 분류.
  6. 답을 원래 문제의 언어로 해석.

4. 대표 예제 — 둘레 일정·최대 면적

"둘레가 100 m인 직사각형 중 면적이 가장 큰 것은?"

  1. 가로 $x$, 세로 $y$라 하면 둘레 $2x + 2y = 100$ ⇒ $y = 50 - x$.
  2. 면적 $A(x) = xy = x(50 - x) = 50x - x^2$.
  3. 정의역: $0 < x < 50$ (면적이 양수여야).
  4. $A'(x) = 50 - 2x = 0 \Rightarrow x = 25$.
  5. $A''(x) = -2 < 0$ → 극대(2계 판정).
  6. $x = y = 25$ — 정사각형이 답. 면적 $625$ m².

"고정된 둘레로 최대 면적 도형은 정사각형"이라는 직관이 식으로 확인됩니다 (더 일반화: 도형 모양도 자유라면 답은 원).

5. 대표 예제 — 점에서 직선까지 최단거리

"점 $(3, 0)$에서 곡선 $y = \sqrt{x}$ 위의 가장 가까운 점은?"

곡선 위의 점 $(t, \sqrt{t})$와 $(3, 0)$의 거리 제곱:

$$ D(t) = (t - 3)^2 + t = t^2 - 5t + 9 $$

(거리 자체보다 거리 제곱을 최소화하는 게 산수에 편하고 결과는 같음 — $\sqrt{}$가 단조 증가니까.)

$D'(t) = 2t - 5 = 0 \Rightarrow t = 5/2$. $D''(t) = 2 > 0$ ⇒ 최소. 가장 가까운 점은 $(5/2, \sqrt{5/2})$.

6. 응용 최적화 더 풀어보기 — 5가지 패턴

학부 최적화 문제는 거의 다음 5가지 틀에서 변형. 각 패턴마다 한 번씩 풀어 두면 시험에서 새 문제도 즉시 풀립니다.

패턴 ① — 면적·부피 최대화 (재료 일정)

"한 변이 벽인 직사각형 마당. 울타리 30 m로 둘러싸는 마당의 최대 면적은?"

벽과 평행한 변 $x$, 수직한 변 $y$. 울타리 길이 $x + 2y = 30$ ⇒ $x = 30 - 2y$. 면적 $A = xy = y(30 - 2y) = 30y - 2y^2$.

$A'(y) = 30 - 4y = 0$ ⇒ $y = 7.5$, $x = 15$, $A_\max = 112.5$ m². 벽이 있을 때는 정사각형이 아니라 가로 = 2 × 세로가 최적.

패턴 ② — 부피·표면적 최소화 (재료 절약)

"부피 1L (=1000 cm³) 인 뚜껑 없는 정사각 밑면 상자. 재료가 가장 적게 드는 치수는?"

밑면 한 변 $x$, 높이 $h$. 부피 $x^2 h = 1000$ ⇒ $h = 1000/x^2$.

표면적(뚜껑 없음) $S = x^2 + 4xh = x^2 + 4000/x$. $S'(x) = 2x - 4000/x^2 = 0$ ⇒ $x^3 = 2000$ ⇒ $x = \sqrt[3]{2000} \approx 12.6$ cm.

$h = 1000/x^2 \approx 6.3$ cm. 밑변 = 2 × 높이가 답.

패턴 ③ — 거리·시간 최단 (스넬의 법칙 맛보기)

"해변 A점에서 바닷속 B점까지 가는 인명구조원. 모래에서는 5 m/s, 물에서는 1 m/s 로 헤엄. 어디에서 물에 뛰어들어야 가장 빨리 도착?"

해변에서 진입점까지 거리를 $x$로 두고 총 시간 $T(x) = (\text{모래 거리})/5 + (\text{물 거리})/1$ 을 최소화. $T'(x) = 0$ 풀면 스넬의 법칙 $\frac{\sin\theta_1}{v_1} = \frac{\sin\theta_2}{v_2}$ 이 자연스럽게 나옵니다 — 빛이 굴절하는 이유와 같은 수학.

패턴 ④ — 비용·이윤 최적화 (경제학)

"한 단위당 단가 $p$일 때 수요 $D(p) = 100 - 2p$. 단위당 생산비 10. 이윤이 최대가 되는 가격은?"

이윤 $\Pi(p) = (p - 10) \cdot D(p) = (p - 10)(100 - 2p) = -2p^2 + 120p - 1000$. $\Pi'(p) = -4p + 120 = 0$ ⇒ $p = 30$. 이윤 $\Pi(30) = -1800 + 3600 - 1000 = 800$. 수요 $D(30) = 40$ 단위.

패턴 ⑤ — 구간 안에 끼는 최대·최소 (면적·체적)

"반지름 1인 반원 안에 가장 큰 직사각형을 그리려면?"

밑변에 직사각형 한 변, 위쪽 두 모서리는 반원 위. 한 모서리를 $(x, y)$, $y = \sqrt{1 - x^2}$. 면적 $A(x) = 2x \cdot y = 2x\sqrt{1 - x^2}$.

$A^2 = 4x^2(1 - x^2)$ 가 더 다루기 쉬움. $u = x^2$로 두면 $A^2 = 4u(1-u)$ ⇒ $u = 1/2$ ⇒ $x = 1/\sqrt{2}$, $y = 1/\sqrt{2}$. $A_\max = 1$.

7. 뉴턴법 — 근 찾기

한 줄 비유부터

$f(x) = 0$을 손으로 풀 수 없을 때 (예: $\cos x = x$). 그래도 어떤 점 $x_0$ 근처에 답이 있다는 걸 안다면, 그 점에서의 접선이 $x$축과 만나는 곳으로 한 발짝 내려가요. 새 점에서 또 접선을 그어 한 발짝 더. 보통 5~10번이면 소수점 10자리까지 답에 수렴합니다.

유도

점 $x_n$에서 접선: $y = f(x_n) + f'(x_n)(x - x_n)$. $y = 0$인 $x$ 를 풀면:

$$ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} $$

예제 6.1 — $\sqrt{2}$ 의 수치 계산

$f(x) = x^2 - 2$의 양수 근. 시작 $x_0 = 1$, 공식 $x_{n+1} = x_n - (x_n^2 - 2)/(2x_n) = (x_n + 2/x_n)/2$.

  • $x_0 = 1$ → $x_1 = (1 + 2)/2 = 1.5$
  • $x_2 = (1.5 + 2/1.5)/2 \approx 1.4167$
  • $x_3 \approx 1.41421568\dots$
  • $x_4 \approx 1.41421356\dots$ (실제 $\sqrt{2} = 1.414213562\dots$)

4번 만에 소수점 9자리. 이게 뉴턴법의 위력 — 2차 수렴(quadratic convergence)이라 매번 정확도가 두 배씩 늘어납니다.

8. 뉴턴법의 수렴 속도 — 왜 빠른가

한 줄 비유부터

오차가 $0.1 → 0.01 → 0.001 → \dots$ 처럼 한 자리씩 줄어드는 게 1차(선형) 수렴. 그런데 뉴턴법은 $0.1 → 0.01 → 0.0001 → 0.00000001 → \dots$ 처럼 자릿수가 두 배씩 폭발적으로 늘어요. 10번이 아니라 5–6번이면 소수점 15자리에 도달하는 이유.

2차 수렴의 의미

오차 $e_n = x_n - x^*$ 라 하면, 단순 근(simple root) 근방에서 뉴턴법은 다음을 만족합니다.

$$ |e_{n+1}| \;\le\; C \cdot e_n^2 $$

$e_n$이 충분히 작으면 다음 오차가 그 제곱의 상수배 — "두 자리가 네 자리, 네 자리가 여덟 자리"로 폭발적으로 정밀해져요.

수치로 확인 — $\sqrt{2}$ 의 수렴

위 예제의 오차를 손으로 적어 보면:

  • $|e_0| = |1 - \sqrt{2}| \approx 0.414$
  • $|e_1| = |1.5 - \sqrt{2}| \approx 0.086$
  • $|e_2| \approx 0.0025$   (≈ $0.086^2 / 2$)
  • $|e_3| \approx 2.1\!\times\!10^{-6}$   (≈ $0.0025^2 / 3$)
  • $|e_4| \approx 1.6\!\times\!10^{-12}$

매 반복마다 자릿수가 정확히 두 배. 이게 일반 1차 수렴(이분법, 경사하강법)과의 결정적 차이입니다.

9. 뉴턴법 — 최적화에 사용

$f$의 극값을 찾는다는 건 $f'(x) = 0$을 푸는 것. 즉 뉴턴법을 $f'$에 적용:

$$ x_{n+1} = x_n - \frac{f'(x_n)}{f''(x_n)} $$

$f''$로 나눈다는 점이 핵심. 곡률(2계 도함수) 정보로 한 발짝의 크기를 결정해요. 이 식이 W14에서 다변수로 확장되면 헤시안 역행렬이 나오는 다변수 뉴턴법이 됩니다.

8. 수렴·발산 — 시작점이 중요한 이유

뉴턴법은 빠르지만 만능이 아닙니다. 다음 상황에서 발산하거나 다른 근으로 갑니다.

  • $f'(x_n) \approx 0$ — 분모가 0에 가까워 다음 점이 멀리 튐.
  • 변곡점 근처에서 시작 — 진동하거나 다른 근으로 빠짐.
  • 주기적 사이클 — 두 점 사이를 오가며 수렴 안 함.

실용적 처방: 시작점을 그래프로 미리 보고 정하기 / 발산하면 작은 step size로 ($\alpha \in (0, 1]$ 곱하기 → "damped Newton").

11. 이분법(Bisection) — 안전한 대안

한 줄 비유부터

책 사이에서 "47쪽"을 찾을 때 자연스럽게 가운데를 펼쳐 보고, 47쪽이 그보다 앞이면 앞쪽 절반을, 뒤면 뒤쪽 절반을 다시 가운데로 가르는 식. 이게 바로 이분법이에요. 느리지만 절대 실패하지 않습니다.

알고리즘

  1. $f(a) < 0 < f(b)$ 인 구간 $[a, b]$ 를 찾는다 (IVT로 근 존재 보장).
  2. 중점 $c = (a+b)/2$. $f(c)$ 의 부호를 본다.
  3. $f(a) f(c) < 0$ 이면 새 구간 $[a, c]$, 아니면 $[c, b]$.
  4. 구간 폭이 충분히 작아질 때까지 반복.

수렴 속도

매 반복마다 구간 폭이 정확히 절반. 1차 수렴 (오차가 매번 0.5배). 30번 반복이면 $2^{30} \approx 10^9$ 분의 1로 줄어듦 — 30번 뉴턴법이면 거의 컴퓨터의 정밀도 한계를 넘는 데 비하면 느려요.

실무 솔버는 두 방법을 결합한 Brent 방법(이분법 + 역2차 보간) — 안정성과 속도를 모두 챙기는 표준 알고리즘.

12. 인터랙티브: 1D 뉴턴법

🎮 뉴턴법 한 발씩 — 곡선 위에서 접선이 x축으로 떨어지기

함수와 시작점을 골라 한 발씩 진행해 보세요. 빨간 점이 새 추정값, 노란 선이 접선, 초록 점이 진짜 근. 발산하는 시작점도 시도해 보세요.

실험해 보기 — ① $\cos x = x$에서 시작점 2.0이면 빠르게 0.7390… 으로 수렴. ② $x^3 - 6x^2 + 9x - 2 = 0$에서 시작점 1.5 근처면 가운데 근으로 수렴, 시작점 0.5면 다른 근으로 갑니다. ③ 시작점을 -1로 두고 $\cos x - x$를 풀면 $\sin x + 1 = 0$이 분모를 0에 가깝게 만들어 발산하는 모습을 볼 수 있어요.

13. 인터랙티브: 뉴턴법 vs 이분법 수렴 비교

같은 함수의 같은 근을 두 방법으로 동시에 풀어, 매 반복마다 오차가 어떻게 줄어드는지 로그 스케일로 비교합니다. 뉴턴법(파랑)이 자릿수가 폭발적으로 늘어나는 반면, 이분법(주황)은 일정 비율로 줄어드는 것이 한눈에 보입니다.

🎮 수렴 속도 비교 — 2차 수렴 vs 1차 수렴

$y$축이 로그 스케일(오차의 $\log_{10}$). 뉴턴법은 직선이 아니라 가속하며 떨어지고, 이분법은 일정한 기울기 직선. 30번 반복까지 시뮬레이션.

14. 흔한 오해와 핵심 정리

흔한 오해 ① — "최대·최소는 임계점에만 있다" — 닫힌 구간에서는 끝점도 후보입니다. 끝점에서 답이 나오는 경우를 잊지 마세요.
흔한 오해 ② — "뉴턴법은 항상 수렴한다" — 거짓. $f'$이 0에 가까운 점에서 발산하거나 사이클에 빠질 수 있어요. 좋은 시작점을 고르는 것이 알고리즘의 일부입니다.
흔한 오해 ③ — "거리를 최소화할 때 항상 거리식을 그대로 미분한다" — $\sqrt{\cdot}$가 단조 증가이므로 거리 제곱을 최소화하는 점이 거리도 최소화. 산수가 훨씬 편해집니다.

16.1 산 정상 찾기 비유 — "최적화"의 본질

안개 낀 산을 등산한다고 해 봐요. 정상이 어디 있는지 안 보입니다. 그런데 한 가지 단서는 분명해요 — 정상에서는 발 밑이 평평합니다. 동쪽으로 가도 위로 안 가고, 서쪽으로 가도 위로 안 가요. 즉 어느 방향이든 "기울기가 0"인 곳이 정상의 후보입니다.

이게 바로 "$f'(x) = 0$이면 극값 후보"의 자연어 번역. 다만 함정 두 가지.

  • 안장(saddle) — 어떤 방향에서는 올라가고 다른 방향에서는 내려가는 평평한 점. 1변수에서는 변곡점에 해당해요.
  • 산기슭(끝점) — 산을 정해진 구간 안에서만 걷는다면, 정상이 산 한복판이 아니라 입구나 출구일 수도 있어요. 이게 "닫힌 구간에서는 끝점도 후보"의 의미.
한 줄 요약 — "기울기 0" + "끝점" + "미분 불가능 점" 이 세 종류의 후보값을 비교하면 닫힌 구간의 절대 최대·최소가 100% 잡힙니다.

16.2 1차 도함수 부호 변화로 극값 판정하기

임계점 $c$에서 $f'(c) = 0$이라고 해서 곧장 극값이라 결론 짓지 마세요. $c$ 양옆에서 $f'$의 부호가 어떻게 바뀌는지를 봐야 해요.

$c$ 왼쪽 $f'$$c$ 오른쪽 $f'$판정그림
$+$$-$극대 (꼭대기)↗ 그러다 ↘
$-$$+$극소 (골짜기)↘ 그러다 ↗
$+$$+$극값 아님 (변곡점/평탄점)↗ 평평 ↗
$-$$-$극값 아님↘ 평평 ↘

$f(x) = x^3$가 대표적 함정입니다. $f'(0) = 0$이지만 $f' = 3x^2$이 양옆 모두 양수라 극값이 아니에요. 그래프는 $x=0$에서 잠시 평평했다가 계속 올라갑니다.

16.3 2차 도함수 검증 — $f''$의 부호로 빠른 판정

$f'(c) = 0$인 임계점에서:

  • $f''(c) > 0$ → 극소 (위로 볼록, 골짜기)
  • $f''(c) < 0$ → 극대 (아래로 볼록, 꼭대기)
  • $f''(c) = 0$ → 판정 실패 (1차 도함수 부호 변화로 다시 봐야 함)

외우는 방법 — "스마일 곡선($\smile$)은 입꼬리가 위로 올라가니 골짜기, 양수($+$). 프라운 곡선($\frown$)은 입꼬리가 내려가니 산봉우리, 음수($-$)."

$f''(c) = 0$일 때 주의 — $f(x) = x^4$은 $x=0$에서 $f' = 0$, $f'' = 0$. 2차 검증으로는 못 잡아요. 1차 도함수 부호 변화($f' = 4x^3$이 음→양)로 보면 극소가 맞습니다. 또는 더 높은 차수 도함수까지 보면 $f^{(4)}(0) = 24 > 0$이라 극소 확정.

16.4 단계별 예제 — 최대 면적의 직사각형

예제 16.4 — 둘레 $40$ cm 인 직사각형 중 면적이 가장 큰 것의 크기와 면적은?

1단계 — 변수 정하기. 가로 $x$, 세로 $y$. 둘 다 양수.

2단계 — 제약식. 둘레 $2(x+y) = 40$이니 $y = 20 - x$.

3단계 — 목적식. 면적 $A(x) = xy = x(20-x) = 20x - x^2$.

4단계 — 정의역. $x > 0$, $y = 20-x > 0$이니 $0 < x < 20$.

5단계 — 임계점. $A'(x) = 20 - 2x = 0 \Rightarrow x = 10$.

6단계 — 분류. $A''(x) = -2 < 0 \Rightarrow$ 극대(2차 검증).

7단계 — 답. $x = y = 10$, $A_\text{max} = 100$ cm². 정사각형이 답!

16.5 단계별 예제 — 최소 비용의 캔 디자인

예제 16.5 — 부피 $355$ mL ($\approx 355$ cm³)인 원통형 음료 캔. 옆면과 윗·아랫면 모두 같은 재료로 만든다면, 재료를 가장 적게 쓰는 반지름 $r$과 높이 $h$는?

1단계. 부피 $V = \pi r^2 h = 355$ ⇒ $h = \dfrac{355}{\pi r^2}$.

2단계. 표면적 $S = 2\pi r^2 + 2\pi r h$.

3단계. $h$를 대입: $S(r) = 2\pi r^2 + 2\pi r \cdot \dfrac{355}{\pi r^2} = 2\pi r^2 + \dfrac{710}{r}$.

4단계. $S'(r) = 4\pi r - \dfrac{710}{r^2} = 0 \Rightarrow r^3 = \dfrac{710}{4\pi} \approx 56.5 \Rightarrow r \approx 3.84$ cm.

5단계. $h = \dfrac{355}{\pi r^2} \approx 7.67$ cm. $h = 2r$ — 즉 "높이 = 지름"이 가장 효율적인 모양!

현실 비교. 실제 음료 캔은 $r \approx 3.3$ cm, $h \approx 11.5$ cm로 더 길쭉. 이유는 손에 쥐는 사용성, 인쇄 공간, 운반 박스의 비율 등 비-수학적 요인.

16.6 단계별 예제 — 종이 상자 부피 최대화

예제 16.6 — $20 \times 30$ cm 종이의 네 모서리에서 한 변이 $x$인 정사각형을 잘라 내고 접어 뚜껑 없는 상자를 만든다. $x$를 얼마로 해야 부피가 최대인가?

밑면 $(20-2x)(30-2x)$, 높이 $x$. 부피:

$$V(x) = x(20-2x)(30-2x) = 4x^3 - 100x^2 + 600x$$

정의역 $0 < x < 10$. $V'(x) = 12x^2 - 200x + 600 = 4(3x^2 - 50x + 150)$.

$V'(x) = 0 \Rightarrow x = \dfrac{50 \pm \sqrt{2500 - 1800}}{6} = \dfrac{50 \pm \sqrt{700}}{6}$.

$\sqrt{700} \approx 26.46$이므로 $x \approx \dfrac{50 - 26.46}{6} \approx 3.92$ cm (다른 해 $x \approx 12.74$는 정의역 밖).

$V(3.92) \approx 1056$ cm³. 끝점에서 $V(0) = V(10) = 0$이니 내부 임계점이 최대.

16.7 단계별 예제 — 펜스 두 개로 두 마당

예제 16.7 — 울타리 60 m로 두 칸짜리 직사각형 마당을 만든다(가운데에도 칸막이). 면적이 최대인 치수는?

가로 $x$ (가로 두 줄), 세로 $y$ (세로 세 줄: 양옆 + 가운데). 울타리: $2x + 3y = 60$ ⇒ $x = (60-3y)/2$.

면적 $A(y) = xy = \dfrac{y(60-3y)}{2} = 30y - \dfrac{3y^2}{2}$.

$A'(y) = 30 - 3y = 0 \Rightarrow y = 10$, $x = 15$, $A_\text{max} = 150$ m².

비율 확인: $2x = 3y$. 이게 일반 패턴 — "각 방향의 울타리 총 길이가 같을 때 최대"라는 직관.

16.8 단계별 예제 — 최단 거리

예제 16.8 — 점 $P(0, 4)$에서 포물선 $y = x^2$ 위의 가장 가까운 점은?

곡선 위 점 $(t, t^2)$과 $P$의 거리 제곱:

$$D(t) = t^2 + (t^2 - 4)^2 = t^4 - 7t^2 + 16$$

$D'(t) = 4t^3 - 14t = 2t(2t^2 - 7) = 0 \Rightarrow t = 0$ 또는 $t = \pm\sqrt{7/2}$.

$D(0) = 16$, $D(\pm\sqrt{7/2}) = (7/2)^2 - 7\cdot(7/2) + 16 = 49/4 - 49/2 + 16 = -49/4 + 16 = 15/4$.

최소 거리 $= \sqrt{15/4} = \dfrac{\sqrt{15}}{2} \approx 1.94$. 가장 가까운 점은 두 개: $\left(\pm\sqrt{7/2}, 7/2\right)$.

주의: $t=0$은 극대(거리 4)였다는 점. 임계점이 항상 최소는 아닙니다 — 분류를 꼭 하세요.

16.9 단계별 예제 — 광고판 보는 최적 거리

예제 16.9 — 도로변 광고판. 아랫변이 지면 위 4 m, 윗변이 지면 위 9 m. 키 1.5 m 인 사람이 도로에서 가장 크게 보이는 (광고판 시야각이 최대인) 지점에 서려면, 광고판 바로 밑에서 몇 m 떨어져야 하나?

눈높이 1.5 m. 광고판 아랫변까지 올려 보는 각 $\alpha$, 윗변까지 $\beta$. 시야각 $\theta = \beta - \alpha$.

도로 거리 $x$일 때 $\tan\alpha = (4 - 1.5)/x = 2.5/x$, $\tan\beta = (9 - 1.5)/x = 7.5/x$.

$\theta(x) = \arctan(7.5/x) - \arctan(2.5/x)$.

$\dfrac{d\theta}{dx} = \dfrac{-7.5/x^2}{1+(7.5/x)^2} - \dfrac{-2.5/x^2}{1+(2.5/x)^2}$. 정리하면 $7.5(x^2 + 6.25) = 2.5(x^2 + 56.25)$ ⇒ $5x^2 = 93.75$ ⇒ $x^2 = 18.75$ ⇒ $x = \sqrt{18.75} \approx 4.33$ m.

의외로 가까이 — 너무 가까우면 위쪽 시선이 너무 가팔라지고, 너무 멀면 둘 다 작아져 시야각이 줄어듭니다.

16.10 뉴턴 방법 — "접선 미끄럼 타기" 비유

뉴턴 방법을 그림으로 이해해 봐요. $f(x) = 0$의 근(=$x$축과 만나는 곳)을 찾고 싶어요. 한 점 $x_n$에서 시작합니다.

  1. 그 점의 곡선 위 점 $(x_n, f(x_n))$을 잡고,
  2. 거기서 접선을 긋습니다 (기울기 $f'(x_n)$).
  3. 그 접선이 $x$축과 만나는 점이 새 추정값 $x_{n+1}$.

비유로 — "곡선 미끄럼틀"을 타는 게 아니라, "그 점에서의 접선 미끄럼틀"을 잠깐 타고 바닥(=$x$축)에 닿는 곳까지 미끄러져 갑니다. 거기서 다시 새 접선 미끄럼틀로 갈아타고, 또 미끄러지고. 이걸 반복.

왜 두 배씩 정확해지는가 — 곡선과 접선의 차이는 $\sim (x - x_n)^2$ 정도. 즉 $x_n$이 답에 가까울수록 접선과 곡선이 거의 같아져, 새 추정값의 오차도 옛 오차의 제곱 비율로 줄어듭니다.

16.15 응용: 머신러닝의 경사하강법(Gradient Descent)

딥러닝에서 모델 파라미터 $\theta$를 학습할 때, 손실 함수 $L(\theta)$를 최소로 만드는 $\theta$를 찾는 게 목표. 1변수로 단순화하면 $L'(\theta) = 0$ 푸는 문제 — 바로 이번 주의 최적화입니다.

다만 $L$이 너무 복잡해서 식으로 못 풀어요. 그래서 경사하강법:

$$\theta_{n+1} = \theta_n - \alpha \cdot L'(\theta_n)$$

$\alpha$는 학습률(step size). 위 뉴턴 방법($\theta_{n+1} = \theta_n - L'(\theta_n)/L''(\theta_n)$)과 비교해 보세요. 분모의 $L''$를 상수 $1/\alpha$로 대체한 것이 경사하강법이에요. 그래서:

  • 경사하강법 — 1차 정보($L'$)만 쓴다. 메모리 적고 매 스텝이 빠르나, 수렴이 느림(1차 수렴).
  • 뉴턴법 — 2차 정보($L''$, 다변수에서 헤시안)도 쓴다. 빠르게 수렴(2차)하지만 헤시안 계산·저장이 비싸 큰 모델엔 부적합.

현실의 딥러닝은 절충안인 Adam·RMSprop 같은 적응형 1차 방법을 씁니다. W14에서 이 비교를 다변수 관점으로 깊이 다룰 거예요.

16.16 응용: 공학 설계 — 빔의 최적 단면

같은 양의 재료로 가장 강한 빔(beam)을 만들려면 단면 모양을 어떻게 잡아야 할까요? 직사각형 단면의 강성도(굽힘 모멘트 저항)는 폭 $b$, 높이 $h$일 때 $I = bh^3/12$. 단면적 $A = bh$를 일정하게 두고 $I$를 최대화하면 — 답은 무한히 얇고 높은 직사각형이 되어 비현실적이에요. 그래서 실제로는 좌굴(buckling) 등 다른 제약을 함께 고려하고, 결과로 H빔(H-beam)·I빔 같은 단면이 표준이 됩니다. 이렇게 "수학적 최적 + 물리적 제약" 조합이 공학 설계의 본질.

16.17 흔한 오답 모음

오답 ① — 끝점 비교 누락
"$f'(x) = 0$ 풀고 답 끝!"이 아닙니다. 닫힌 구간 문제에서는 임계점값 + 양 끝점값을 모두 계산하고 비교해야 해요. 끝점에서 답이 나오는 경우가 시험 단골 함정.
오답 ② — 정의역 잊기
최적화 변수에는 보통 물리적 정의역이 있습니다(길이는 양수, 비율은 0~1, 등). 정의역 밖으로 나간 임계점은 버려야 해요.
오답 ③ — 2차 검증을 무조건 신뢰
$f''(c) = 0$이면 판정 실패. 1차 도함수 부호 변화나 더 높은 차수를 봐야 해요. $f(x) = x^4$, $x^5$ 등에서 흔히 발생.
오답 ④ — 거리 식을 그대로 미분
$\sqrt{(x-a)^2 + (y-b)^2}$를 그대로 미분하면 분수와 제곱근이 섞여 매우 복잡해집니다. 거리 제곱 $D^2$을 최소화하세요 — 결과는 같고 산수는 절반.
오답 ⑤ — 뉴턴법이 항상 수렴한다고 가정
$f'(x_n) \approx 0$인 점 근처, 변곡점, 발산 사이클 등에서 실패합니다. 항상 발산 검사(예: $|x_{n+1} - x_n|$ 모니터링)를 해야 해요.
오답 ⑥ — 잘못된 시작점
뉴턴법은 시작점이 다른 근의 영역에 있으면 그쪽으로 빨려 갑니다. 그래프를 먼저 그려 보고 원하는 근 쪽 시작점을 고르세요.

16.18 자가 점검 체크리스트

  1. Q1. $f(x) = x^4 - 8x^2 + 5$의 극값 모두 구하고 분류하시오.
    힌트: $f'(x) = 4x^3 - 16x = 4x(x^2 - 4)$. 임계점 $x = -2, 0, 2$. $f''(x) = 12x^2 - 16$로 $f''(-2)=32>0$ 극소, $f''(0)=-16<0$ 극대, $f''(2)=32>0$ 극소.
  2. Q2. $f(x) = x + 1/x$의 $[1, 4]$ 최대·최소.
    힌트: $f'(x) = 1 - 1/x^2 = 0 \Rightarrow x = 1$ (구간 끝점과 일치). 후보 $f(1) = 2$, $f(4) = 4.25$. 최소 2, 최대 4.25.
  3. Q3. 둘레 24 cm 인 직각삼각형 중 면적이 최대인 것은?
    힌트: 다리 $a, b$, 빗변 $\sqrt{a^2+b^2}$. 둘레 $a + b + \sqrt{a^2+b^2} = 24$, 면적 $A = ab/2$. 라그랑주 또는 $b$를 $a$로 표현. 답: $a = b = 24/(2+\sqrt 2)$인 직각이등변삼각형.
  4. Q4. 뉴턴법으로 $f(x) = x^3 - x - 1 = 0$을 $x_0 = 1$로 풀고 3번 반복까지 적으시오.
    힌트: $f'(x) = 3x^2 - 1$. $x_1 = 1 - (1-1-1)/(3-1) = 1.5$. $x_2 = 1.5 - (3.375-1.5-1)/(6.75-1) = 1.5 - 0.875/5.75 \approx 1.3478$. $x_3 \approx 1.3252$. 실제 근 $\approx 1.3247$.
  5. Q5. $\cos x = x$의 해를 뉴턴법으로 $x_0 = 1$, 2번 반복.
    힌트: $f(x) = \cos x - x$, $f'(x) = -\sin x - 1$. $x_1 = 1 - (\cos 1 - 1)/(-\sin 1 - 1) = 1 - (0.5403 - 1)/(-1.8415) \approx 0.7504$. $x_2 \approx 0.7391$. 실제 답 $\approx 0.73909$.
  6. Q6. 반지름 $R$인 구에 내접하는 원기둥 중 부피가 최대인 것의 높이는?
    힌트: 원기둥 반지름 $r$, 높이 $h$, $r^2 + (h/2)^2 = R^2$. 부피 $V = \pi r^2 h = \pi(R^2 - h^2/4)h$. $V'(h) = 0 \Rightarrow h = 2R/\sqrt 3$. 비율 $h:2r = 1:\sqrt 2$.
  7. Q7. $f(x) = (x-1)^4$에서 $x = 1$이 극값인지 판정하시오. 2차 검증으로 풀리는가?
    힌트: $f'(1) = 0$, $f''(1) = 0$ — 2차 검증 실패. 1차 도함수 $4(x-1)^3$이 $x=1$ 양옆에서 음→양 → 극소.
  8. Q8. $f(x) = e^x - 2x$의 최솟값을 구하시오.
    힌트: $f'(x) = e^x - 2 = 0 \Rightarrow x = \ln 2$. $f''(x) = e^x > 0$이라 항상 극소. 최솟값 $f(\ln 2) = 2 - 2\ln 2 \approx 0.6137$.
  9. Q9. 뉴턴법이 발산하는 함수의 예 하나를 들고 이유를 설명하시오.
    힌트: $f(x) = x^{1/3}$, 시작점 어디든. $f'(x) = (1/3)x^{-2/3}$. 점화식 $x_{n+1} = x_n - x_n^{1/3}/((1/3)x_n^{-2/3}) = x_n - 3x_n = -2x_n$. 매번 $|x|$가 두 배 — 발산.
  10. Q10. 머신러닝의 경사하강법과 뉴턴법의 핵심 차이를 한 문장으로 쓰시오.
    힌트: "경사하강법은 1차 도함수만, 뉴턴법은 2차 도함수까지 — 뉴턴이 빠르나 메모리·계산이 비싸 대형 모델엔 1차 방법이 표준."

16.19 FAQ

Q. "임계점"과 "극점"의 차이?
A. 임계점(critical point) = $f'(c) = 0$ 또는 $f'(c)$가 정의 안 되는 점. 극점(extremum) = 실제로 극대·극소인 점. 모든 극점은 임계점이지만, 모든 임계점이 극점은 아닙니다($f(x) = x^3$의 $x=0$).

Q. 왜 닫힌 구간에서는 EVT(극값 존재 정리)가 성립하나?
A. "닫혀 있고 + 유한"이라는 두 조건이 핵심. 열린 구간에서는 끝점 근처에서 함숫값이 무한히 커질 수 있고, 무한 구간에서는 정점이 없을 수 있어요.

Q. 거리 제곱을 최소화하는 게 정말 거리 자체를 최소화하나?
A. 네. $f(t) \geq 0$이고 $g(s) = \sqrt s$가 단조 증가이므로 $f$가 최소인 점에서 $\sqrt f$도 최소. 결과로 같은 점이 답.

Q. 뉴턴법이 수렴해도 답이 옳다고 보장되나?
A. 알고리즘이 멈췄다고 답이라 결정짓지 마세요. 검증으로 $|f(x_n)|$이 충분히 작은지 확인 (예: $10^{-10}$). 실무에서는 "$|x_{n+1}-x_n| < \varepsilon$ 이고 $|f(x_n)| < \varepsilon$" 두 조건 모두 만족할 때 종료.

Q. 뉴턴법은 다변수에도 쓰나?
A. 네. $\mathbf x_{n+1} = \mathbf x_n - [Hf(\mathbf x_n)]^{-1} \nabla f(\mathbf x_n)$. $H$는 헤시안 행렬(2계 편미분 모임). W14에서 다룹니다.

Q. 왜 뉴턴법이 머신러닝의 표준이 아닌가?
A. 헤시안 크기가 파라미터 수의 제곱($n^2$). GPT 같이 $n \sim 10^9$ 규모면 헤시안만 $10^{18}$ 항. 메모리 폭발. 그래서 1차 방법(SGD, Adam)이 표준.

Q. 임계점이 여러 개일 때 어떻게 분류?
A. 1차: 부호표를 그려라. 2차: $f''$ 부호로 빠르게. 둘 다 안 되면 직접 함숫값을 비교 — "원시적" 방법이지만 항상 통합니다.

17. 보강 II — 더 많은 응용과 깊이 있는 예제

앞 절에서 기본기를 다졌다면, 여기서는 시험·과제에서 자주 만날 변형 문제, 자율주행·로봇공학 응용, 경계점 함정 사례를 한 번에 정리해 봅니다.

17.1 응용 패턴 ⑥ — 통신 케이블 매설 (모래·바다 비용 다른 경우)

예제 17.1 — 해변 A점과 섬의 B점을 케이블로 연결. 육지에서 m당 5만 원, 바다에서 m당 13만 원. 가장 싼 경로는?

해변선이 직선이라 가정. A에서 진입점까지 육지 거리 $x$, 바다 직선 거리 $\sqrt{(L-x)^2 + d^2}$ ($L$ 해변 길이, $d$ 섬과 해변 거리). 총 비용:

$$C(x) = 5x + 13\sqrt{(L-x)^2 + d^2}$$

$C'(x) = 5 - \dfrac{13(L-x)}{\sqrt{(L-x)^2 + d^2}} = 0$.

이항하면 $\dfrac{L-x}{\sqrt{(L-x)^2+d^2}} = \dfrac{5}{13}$ — 즉 진입각 $\theta$의 $\sin\theta = 5/13$.

이는 스넬의 법칙의 비용 버전. 두 매질의 "속도" 대신 "비용 비"가 굴절을 결정해요.

17.2 응용 패턴 ⑦ — 로봇 매니퓰레이터의 최단 시간 동작

로봇 팔이 점 A에서 점 B로 이동할 때, 각 관절의 최대 속도·가속도 한계 안에서 가장 빠른 궤적은? 이게 본격 최적 제어(optimal control) 문제. 1변수 최적화는 그 가장 단순한 경우.

가속도 일정으로 가속·감속만 한다고 가정. 거리 $S$, 최대 속도 $v_\max$, 가속도 $a$일 때 최단 시간 $T$:

  • $v_\max \cdot T_a / 1 = v_\max$이고 가속 거리 $S_a = v_\max^2/(2a)$. 만일 $2 S_a < S$면 등속 구간이 있어 $T = 2 v_\max/a + (S - 2 S_a)/v_\max$.
  • 아니면 가속·감속만 하므로 $S = a T^2/4 \Rightarrow T = 2\sqrt{S/a}$.

실제 로봇 제어는 이 수학에 진동 억제·에너지 최적화까지 더해 다변수로 풉니다 (W14).

17.3 응용 패턴 ⑧ — 자율주행 차량의 안전 제동거리

예제 17.3 — 차량 속도 $v_0$, 마찰계수 $\mu$, 반응 시간 $t_r$. 장애물까지 거리 $D$. 최대 안전 속도 $v_0^*$를 식으로 표현하고, $\mu = 0.7$, $t_r = 0.5$ s, $D = 30$ m일 때 $v_0^*$를 구하시오.

제동거리(반응 + 제동):

$$D(v_0) = v_0 t_r + \dfrac{v_0^2}{2 \mu g}$$

$D = 30$이라 두고 $v_0$에 대해 풀면 2차방정식. $g = 9.8$ m/s²로 $\dfrac{v_0^2}{2 \cdot 0.7 \cdot 9.8} + 0.5 v_0 - 30 = 0$ ⇒ $v_0^2/13.72 + 0.5 v_0 - 30 = 0$.

$v_0 = \dfrac{-0.5 + \sqrt{0.25 + 4 \cdot 30/13.72}}{2/13.72} = \dfrac{-0.5 + \sqrt{0.25 + 8.746}}{0.1458}$

$= \dfrac{-0.5 + 3.0}{0.1458} \approx 17.15$ m/s ≈ 61.7 km/h.

설계 마진을 위해 안전 속도는 보통 이보다 20~30% 낮게 설정합니다.

17.4 응용 패턴 ⑨ — 항공 운항의 최적 순항 속도

비행기 연료 소비율은 속도가 너무 낮으면 비행시간이 길어 총 연료가 늘고, 너무 빠르면 공기 저항으로 단위 시간 연료가 폭증해요. 단순 모델로:

$$F(v) = \dfrac{a}{v} + b v^3$$

($a, b > 0$ 상수) — 첫 항은 시간 비례 항, 둘째 항은 항력 항. $F'(v) = -a/v^2 + 3bv^2 = 0 \Rightarrow v^* = \sqrt[4]{a/(3b)}$.

이 최적 속도를 "최대 항속거리(Best Range) 속도"라 부르고, 보잉 737 등 여객기는 보통 $v^* \approx 0.78\,M$ (마하 0.78) 부근에서 운항.

17.5 응용 패턴 ⑩ — 약물 농도의 최적 투여 간격

한 번 복용 시 혈중 농도가 시간 $t$에 대해 $C(t) = C_0 e^{-kt}$로 감소(반감기 $\ln 2 / k$). 효과 농도 $C_\min$ 이상을 유지하려면 다음 복용 간격 $T$는 $C_0 e^{-kT} \geq C_\min$, 즉 $T \leq \ln(C_0/C_\min)/k$.

최적 투여는 "효과 + 부작용"의 절충 — 보통 $C(t)$가 $C_\min$과 $C_\max$ 사이에서 톱니파처럼 진동하도록 설계. 현대 약물동력학(pharmacokinetics)의 한 페이지.

17.6 뉴턴 방법의 변형 ① — 시컨트(Secant) 방법

뉴턴법은 $f'(x_n)$이 필요한데, 도함수를 구하기 어렵거나 비용이 큰 경우가 있어요. 그땐 두 점의 함숫값으로 도함수를 근사:

$$x_{n+1} = x_n - f(x_n) \cdot \dfrac{x_n - x_{n-1}}{f(x_n) - f(x_{n-1})}$$

수렴 차수 $\approx 1.618$ (황금비!). 뉴턴법(2)보단 느리지만 이분법(1)보단 빠른, 도함수 없는 빠른 방법.

17.7 뉴턴 방법의 변형 ② — Damped Newton

발산 방지로 step에 가중치 $\alpha \in (0, 1]$:

$$x_{n+1} = x_n - \alpha \cdot \dfrac{f(x_n)}{f'(x_n)}$$

$\alpha$ 작게(예: 0.1~0.5)로 시작해 수렴이 안정되면 $\alpha = 1$로 키움 — line search 또는 trust region 기법으로 자동화.

17.8 뉴턴 방법의 변형 ③ — Quasi-Newton (BFGS, L-BFGS)

다변수에서 헤시안 $H$ 직접 계산이 너무 비쌀 때, 매 스텝마다 그래디언트 변화로 $H$를 근사 업데이트. BFGS가 표준이고 메모리 절약 버전이 L-BFGS. scipy.optimize의 기본 알고리즘 중 하나.

17.9 SVG: 뉴턴 방법 발산 사례

$f(x) = \arctan x$ ($x_0$이 너무 크면 발산). $f'(x) = 1/(1+x^2)$. 점 $x_0 = 1.5$에서 시작하면 $x_1 \approx -1.69$, $x_2 \approx 2.32$, $x_3 \approx -5.11$ — 진동·발산.

17.10 SVG: 뉴턴 방법으로 극값 찾기

$f' = 0$을 푸는 게 핵심. 예시로 $f(x) = x^2$의 최솟값을 뉴턴법으로 찾으면 한 번에 정답.

17.12 마지막 한 마디

최적화는 단순한 미분 한 번이 아니라 "문제를 식으로 옮기는 능력 + 후보를 빠짐없이 찾는 끈기 + 분류 도구를 정확히 쓰는 신중함"이 합쳐진 종합 예술. 뉴턴법은 그 신중함이 바닥났을 때 컴퓨터에게 도움을 청하는 우리의 손. 다음 주에 만날 테일러 급수는 이 모든 도구의 수학적 기반을 한 번에 설명해 줍니다 — "어떤 매끄러운 함수든 다항식 무한합으로 표현할 수 있다"는 사실이 왜 뉴턴법이 작동하는지, 왜 2차 검증이 충분한지를 한 줄로 정당화해 줘요.

이번 주 격려의 한 마디 — "공식을 외우는 학생은 시험 한 번을 푼다. 절차를 외우는 학생은 시험 백 번을 푼다. 직관을 갖춘 학생은 시험을 만든다." 위 17개 절을 정독했다면 이미 세 번째 단계의 입구.

18.1 같은 문제 두 가지 풀이 — 1차 vs 2차 검증

예제 18.1 — $f(x) = 3x^4 - 4x^3 - 12x^2 + 5$의 극값.

풀이 A (1차 부호 검정) — $f'(x) = 12x^3 - 12x^2 - 24x = 12x(x^2 - x - 2) = 12x(x-2)(x+1)$.

임계점 $x = -1, 0, 2$. 부호표:

구간$x < -1$$-1 < x < 0$$0 < x < 2$$x > 2$
$f'$$-$$+$$-$$+$
$f$↘↗↘↗

$x = -1$ 극소, $x = 0$ 극대, $x = 2$ 극소. $f(-1) = 0, f(0) = 5, f(2) = -27$.

풀이 B (2차 검증) — $f''(x) = 36x^2 - 24x - 24 = 12(3x^2 - 2x - 2)$.

  • $f''(-1) = 12(3 + 2 - 2) = 36 > 0$ → 극소 ✓
  • $f''(0) = -24 < 0$ → 극대 ✓
  • $f''(2) = 12(12 - 4 - 2) = 72 > 0$ → 극소 ✓

두 방법 결론 일치. 2차 검증은 부호표 그릴 필요 없이 임계점에서 한 번만 평가하면 돼서 빠른데, $f''$ 계산이 어려우면 1차 검정이 나아요. 시험에서는 둘 다 익혀 상황별로 선택.

18.2 Python 한 조각 — 뉴턴법 직접 구현

def newton(f, df, x0, tol=1e-12, max_iter=50):
    """뉴턴법으로 f(x)=0의 근을 찾는다.
    f, df: 함수와 도함수
    x0: 시작점, tol: 수렴 허용오차"""
    x = x0
    for n in range(max_iter):
        fx = f(x)
        if abs(fx) < tol:
            return x, n
        dfx = df(x)
        if abs(dfx) < 1e-14:
            raise ValueError(f"f'(x)≈0 at iter {n}, 발산 위험")
        x = x - fx / dfx
    raise RuntimeError("수렴 실패")

# 예: sqrt(2) 계산
import math
root, iters = newton(lambda x: x*x - 2, lambda x: 2*x, x0=1.0)
print(f"√2 ≈ {root}, {iters} 반복, 실제 {math.sqrt(2)}")
# 출력: √2 ≈ 1.4142135623730951, 5 반복, 실제 1.4142135623730951

실무에서는 scipy의 scipy.optimize.newton(f, x0, fprime=df) 또는 scipy.optimize.brentq(f, a, b)(이분법+보간 결합)를 그대로 쓰면 됩니다. 직접 구현은 학습용.

18.3 Python 한 조각 — 경사하강법으로 최솟값

def gradient_descent(f, df, x0, lr=0.1, tol=1e-8, max_iter=10000):
    x = x0
    for n in range(max_iter):
        g = df(x)
        if abs(g) < tol:
            return x, n
        x = x - lr * g
    return x, max_iter

# 예: f(x) = (x-3)^2 + 1 의 최솟값
root, iters = gradient_descent(
    lambda x: (x-3)**2 + 1,
    lambda x: 2*(x-3),
    x0=0.0, lr=0.1)
print(f"최솟값 위치 ≈ {root}, {iters} 반복")
# lr이 너무 크면(예: 1.5) 발산. 너무 작으면 매우 느림 → 학습률 조정이 핵심

18.4 시험 전략 — 최적화 문제 풀이 절차 (체크리스트)

  1. (읽기) 무엇을 최대·최소화하는가? 한 단어로 답할 수 있어야.
  2. (그림) 도형이 있다면 반드시 그려라. 변수 이름을 그림에 적어라.
  3. (변수) 미지수를 정의하고 단위를 명시.
  4. (제약) 주어진 조건(둘레, 부피, 등)으로 변수 사이 관계식 1개 이상 작성.
  5. (목적) 최적화할 양을 변수의 함수로. 제약식으로 변수 하나 소거.
  6. (정의역) 변수 범위(양수, 구간 끝, 등) 명시.
  7. (미분) 목적함수 미분 → 0으로 두고 임계점 찾기.
  8. (분류) 1차 또는 2차 검정으로 극대·극소 분류.
  9. (끝점) 닫힌 구간이면 끝점값도 비교.
  10. (해석) 답을 원래 단어로 (예: "정사각형, 한 변 10 cm").
