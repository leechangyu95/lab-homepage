1. 롤 정리 (Rolle's Theorem)

한 줄 비유부터

등산을 시작한 곳과 도착한 곳의 고도가 같다고 합시다. 그러면 도중에 반드시 한 번은 평지(지역적 정상 또는 분지)를 지났을 거예요. 그 평지에서 순간 기울기가 0 — 그게 롤 정리입니다.

정리 진술

$f$가 $[a, b]$에서 연속, $(a, b)$에서 미분 가능하고 $f(a) = f(b)$이면, $f'(c) = 0$인 $c \in (a, b)$가 적어도 하나 존재한다.

2. 평균값 정리 (MVT, Mean Value Theorem)

한 줄 비유부터

출발점부터 도착점까지의 평균 속력이 60 km/h였다면, 도중에 적어도 한 번은 정확히 60 km/h로 달린 순간이 존재합니다. 평균을 한 점에서 실제로 달성한 순간 — 그게 MVT.

정리 진술

$f$가 $[a, b]$에서 연속, $(a, b)$에서 미분 가능하면, 다음을 만족하는 $c \in (a, b)$가 적어도 하나 존재한다.

$$ f'(c) = \frac{f(b) - f(a)}{b - a} $$

그래프 위에서: 두 끝점을 잇는 할선과 평행한 접선이 도중에 반드시 한 번은 그어진다는 뜻.

왜 강력한가?

다음 두 결과가 MVT의 직접 따름.

  • $f' \equiv 0$이면 $f$는 상수. ("변하지 않는데 어떻게 다른 값?")
  • $f'$이 두 함수에서 같으면 두 함수는 상수 차이. 적분의 정의와 직결되는 결과 (W9).

3. 1차 도함수 → 단조성

$$ f'(x) > 0 \;\text{on } I \Rightarrow f \text{ 증가}, \quad f'(x) < 0 \;\text{on } I \Rightarrow f \text{ 감소} $$

예제 3.1

$f(x) = x^3 - 3x$의 단조 구간.

$f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$. 부호:

  • $x < -1$: $f' > 0$ → 증가
  • $-1 < x < 1$: $f' < 0$ → 감소
  • $x > 1$: $f' > 0$ → 증가

그러므로 $x = -1$에서 극대, $x = 1$에서 극소.

4. 극값과 1·2계 판정법

1계 판정법 (First Derivative Test)

임계점 $c$ ($f'(c) = 0$ 또는 미분 불가능)에서:

  • $f'$이 $c$ 직전에 양수, 직후에 음수 → 극대.
  • $f'$이 $c$ 직전에 음수, 직후에 양수 → 극소.
  • 부호가 바뀌지 않음 → 극값 아님.

2계 판정법 (Second Derivative Test)

$f'(c) = 0$이고 $f''(c) \ne 0$이면:

  • $f''(c) > 0$ → 극소 (그릇 모양).
  • $f''(c) < 0$ → 극대 (역그릇 모양).
  • $f''(c) = 0$ → 판정 불가, 1계로 다시 봐야 함.

5. 2차 도함수 → 볼록성

정의

구간 $I$에서 $f'' > 0$이면 볼록(convex, concave up), $f'' < 0$이면 오목(concave down)이라 합니다.

  • 볼록 함수: 그릇 모양. 임의의 두 점을 잇는 할선이 곡선보다 위.
  • 오목 함수: 역그릇 모양. 할선이 곡선보다 아래.

왜 중요한가?

최적화에서 결정적입니다. 볼록 함수는 지역 최저점이 곧 전역 최저점이라 골치가 없어요. 오목/볼록이 섞여 있으면 가짜 극값에 속을 수 있습니다.

예제 5.1 — AM-GM 부등식 ($\ln x$의 오목성에서)

$\ln x$의 도함수 $1/x$, 2차 도함수 $-1/x^2 < 0$이므로 $\ln x$는 $x > 0$에서 오목.

오목 함수의 정의(할선이 곡선 아래)로부터 $\ln\!\bigl(\tfrac{a+b}{2}\bigr) \ge \tfrac{1}{2}(\ln a + \ln b) = \ln\sqrt{ab}$. 즉 $\dfrac{a+b}{2} \ge \sqrt{ab}$ — 산술-기하 평균 부등식이 한 줄에 떨어집니다.

6. 변곡점 (Inflection Point)

$f$가 그 점에서 연속이고, 그 점을 지나면서 $f''$의 부호가 바뀌면 그 점을 변곡점이라 합니다.

주의: $f''(c) = 0$이라고 무조건 변곡점이 되는 게 아닙니다. 부호가 바뀌어야 합니다 (예: $f(x) = x^4$의 $x = 0$은 $f''(0) = 0$이지만 변곡점 아님).

7. 그래프 분석 종합 — 7단계 절차

  1. 정의역과 점근선.
  2. 대칭성 (우함수·기함수·주기) 검토.
  3. $x$절편·$y$절편.
  4. $f'$의 부호로 단조 구간.
  5. 임계점에서 1계 또는 2계 판정으로 극값 분류.
  6. $f''$의 부호로 볼록 구간과 변곡점.
  7. 표를 그리고 그래프 스케치.

10. 흔한 오해와 핵심 정리

흔한 오해 ① — "$f'(c) = 0$이면 극값이다" — 거짓. $f(x) = x^3$의 $x = 0$은 $f'(0) = 0$이지만 극값 아님. 1계 판정으로 부호 변화가 있어야 극값.
흔한 오해 ② — "$f''(c) = 0$이면 변곡점이다" — 거짓. $f(x) = x^4$의 $x = 0$은 $f''(0) = 0$이지만 변곡점 아님 (양쪽이 모두 볼록). $f''$의 부호 변화가 있어야 변곡점.
흔한 오해 ③ — "MVT는 닫힌 구간 끝점에서도 성립한다" — $c$는 열린 구간 $(a, b)$ 안에 있어야 합니다. 끝점에서는 한쪽 도함수만 정의되니 MVT의 보장이 닿지 않아요.

이번 주의 핵심 한 문장

"$f'$의 부호로 증감을, $f''$의 부호로 볼록성을 읽는다. 임계점과 변곡점을 찾아 모양을 분류하면 어떤 함수의 그래프든 그릴 수 있다."

12. 친절 보강 — 비유로 다시 배우는 MVT·단조성·볼록성

아래는 1학년 학부생 눈높이에서 다시 정리한 보충 강의입니다. 같은 내용이라도 비유 → 정의 → 예제 → 자가 점검의 순서로 한 번 더 다지면, 시험장에서 흔들리지 않습니다.

12.1 평균값 정리, 운전 이야기로 한 번 더

한 줄 비유 — "공주에서 천안까지 평균 60 km/h였다면, 도중 어느 한 순간의 속도계는 정확히 60 km/h를 가리켰다." 이게 평균값 정리의 전부입니다.

속도위반 단속의 구간 단속을 생각해 봅시다. A 지점과 B 지점 사이의 평균 속도가 110 km/h였다면, 두 지점 사이 어딘가에서 그 110 km/h를 한 번은 진짜로 달성한 순간이 있다는 게 MVT의 직관입니다. 운전자는 "저는 항상 100 km/h만 밟았어요"라고 우길 수 없어요. 평균이 110이면 어느 순간엔 110이 등장해야 하니까요.

수학으로 옮기면 거리 함수 $s(t)$가 닫힌 구간 $[a, b]$에서 연속이고 열린 구간에서 미분 가능하면, 평균 속도 $\dfrac{s(b)-s(a)}{b-a}$와 같은 순간 속도 $s'(c)$가 적어도 한 점에서 등장한다는 뜻이 됩니다.

12.2 롤 정리는 MVT의 특수 사례

롤 정리는 MVT에서 양 끝점 값이 같은($f(a)=f(b)$) 특수 경우입니다. 그 때 평균 변화율이 0이니, MVT가 보장하는 $c$에서 $f'(c)=0$이 됩니다. 즉:

비교롤 정리평균값 정리
가정$f$ 연속/미분 + $f(a)=f(b)$$f$ 연속/미분
결론$\exists c\in(a,b),\ f'(c)=0$$\exists c\in(a,b),\ f'(c)=\dfrac{f(b)-f(a)}{b-a}$
비유고도가 같으면 도중 어느 평지를 지난다평균 속력이 60이면 도중 한 순간 60
그래프수평 접선이 적어도 한 번할선과 평행한 접선이 적어도 한 번
꼭 기억 — MVT는 "존재"만 보장합니다. 어디 있는지 정확히 찾아주는 게 아니에요. 그래도 부등식 증명에 굉장히 강력합니다(예: $\sin$의 립시츠 성질, $|\sin x - \sin y|\le |x-y|$).

12.4 단조성 — 부호로 산 모양을 읽기

$f'$의 부호는 그래프의 "올라감/내려감"을 알려 주는 신호등입니다.

부호해석비유
$f'(x) > 0$$f$ 단조 증가오르막. 등산하는 발걸음마다 고도가 올라감.
$f'(x) = 0$임계점(평지/산정/계곡 가능)잠시 평평한 능선. 더 봐야 어떤 곳인지 안다.
$f'(x) < 0$$f$ 단조 감소내리막. 한 발짝마다 고도가 떨어짐.
주의 1 — "$f'>0$이면 증가"는 구간 전체에서 성립할 때만 깔끔합니다. 한 점에서만 양수라고 그 점 주변에서 증가가 보장되는 건 아닙니다. 항상 구간 전체의 부호를 살피세요.

12.5 단계별 예제 — 단조성 분석

예제 12.5.1. $f(x) = x^4 - 4x^3$의 단조 구간을 구하라.

1단계. 도함수: $f'(x) = 4x^3 - 12x^2 = 4x^2(x-3)$.

2단계. $f'(x)=0$ → $x=0$(중복근), $x=3$.

3단계. 부호표:

  • $x<0$: $4x^2>0$, $(x-3)<0$ → $f'<0$ → 감소
  • $00$, $(x-3)<0$ → $f'<0$ → 감소
  • $x>3$: $4x^2>0$, $(x-3)>0$ → $f'>0$ → 증가

결론. 감소 구간 $(-\infty, 3)$, 증가 구간 $(3, \infty)$. $x=0$은 임계점이지만 부호가 바뀌지 않으므로 극값이 아닙니다(평지가 잠깐 등장). $x=3$이 극소.

예제 12.5.2. $g(x) = xe^{-x}$의 단조 구간.

곱 미분: $g'(x) = e^{-x} - xe^{-x} = (1-x)e^{-x}$. $e^{-x}>0$이므로 부호는 $1-x$가 결정.

$x<1$: $g'>0$ → 증가; $x>1$: $g'<0$ → 감소. 즉 $x=1$이 극대(최댓값 $1/e$).

예제 12.5.3. $h(x) = \dfrac{x}{1+x^2}$의 단조 구간.

몫 미분: $h'(x) = \dfrac{(1+x^2)-x\cdot 2x}{(1+x^2)^2} = \dfrac{1-x^2}{(1+x^2)^2}$.

분모는 항상 양수. 분자 $1-x^2$의 부호:

  • $|x|<1$ → $h'>0$ → 증가
  • $|x|>1$ → $h'<0$ → 감소

$x=-1$ 극소(=$-1/2$), $x=1$ 극대(=$1/2$). 양 끝에서 0으로 수렴.

12.6 볼록성 — 그릇과 엎은 그릇

2차 도함수 $f''$는 곡선이 "위로 굽었는지(볼록)" 또는 "아래로 굽었는지(오목)"를 알려 줍니다. 비유 한 줄:

비유 — 비가 와서 곡선 위에 물을 부어 봅시다. 물이 곡선 안에 고이면 볼록(그릇 모양, $f''>0$). 물이 양쪽으로 흘러내리면 오목(엎어 놓은 그릇, $f''<0$).
2차 도함수이름그림예
$f''(x) > 0$볼록(convex, concave up)U 모양 그릇$x^2$, $e^x$, $-\ln x$
$f''(x) < 0$오목(concave down)역U 그릇$\ln x$, $-x^2$, $\sqrt{x}$
$f''(x) = 0$변곡 후보S자 가운데$x^3$의 $x=0$

12.8 단계별 예제 — 볼록성 판정

예제 12.8.1. $f(x) = x^3 - 6x^2 + 9x + 1$의 볼록 구간과 변곡점.

$f'(x) = 3x^2 - 12x + 9$, $f''(x) = 6x - 12 = 6(x-2)$.

  • $x<2$: $f''<0$ → 오목
  • $x>2$: $f''>0$ → 볼록

$x=2$에서 부호가 바뀌므로 변곡점. $f(2) = 8 - 24 + 18 + 1 = 3$. 변곡점 $(2, 3)$.

예제 12.8.2. $g(x) = x^4$의 $x=0$은 변곡점이 아닌 이유.

$g''(x) = 12x^2 \ge 0$. $x=0$에서 $g''(0)=0$이지만 양쪽 모두 $g''>0$(볼록). 부호가 바뀌지 않았으니 변곡점이 아닙니다. 단지 "잠깐 평평한 그릇 바닥".

예제 12.8.3. $h(x) = \sin x$의 볼록·오목 구간 (한 주기 $[0, 2\pi]$ 안에서).

$h''(x) = -\sin x$.

  • $0 < x < \pi$: $\sin x > 0$이므로 $h''<0$ → 오목 (산봉우리 쪽)
  • $\pi < x < 2\pi$: $\sin x < 0$이므로 $h''>0$ → 볼록 (계곡 쪽)

$x = 0, \pi, 2\pi$에서 변곡점. 사인 곡선이 위·아래로 번갈아 굽는 이유입니다.

예제 12.8.4. $\phi(x) = e^{-x^2}$ (정규분포 종 모양)의 변곡점.

$\phi'(x) = -2x e^{-x^2}$, $\phi''(x) = (-2 + 4x^2) e^{-x^2} = 2(2x^2 - 1)e^{-x^2}$.

$\phi''=0$ → $x = \pm 1/\sqrt{2}$. 양쪽에서 부호가 바뀌므로 변곡점 두 개. 종 모양 그래프의 어깨 부분이 정확히 이 위치입니다 — 정규분포 표준편차의 위치와도 같습니다.

12.9 변곡점 한 번 더

변곡점은 두 가지 조건이 함께 필요합니다.

  1. 그 점에서 $f$가 연속.
  2. 그 점을 지나며 $f''$의 부호가 바뀜.
흔한 실수 — "$f''(c)=0$만 보고 변곡점이라고 적기". 부호 변화 검증을 빼먹으면 $x^4$ 같은 함수에서 틀립니다. 부호표를 그리세요.

12.11 안장점 주의 — $f'(c)=0$이라고 다 극값은 아니다

$f(x)=x^3$의 $x=0$은 $f'(0)=0$이지만, 좌우 모두 증가. 극값이 아니라 안장(saddle) 같은 평지일 뿐입니다. 다변수에서는 진짜 안장점이 더 자주 등장하지만, 1변수에서도 이런 모양은 종종 나옵니다.

핵심 — 임계점은 "후보"일 뿐. 1계 판정법으로 부호 변화를 확인해야 극값입니다.

12.12 그래프 손으로 그리기 — 자세한 7단계

  1. 정의역 — 분모 0, 로그 음수, 제곱근 음수 등 제외.
  2. 대칭성 — $f(-x)=f(x)$(우함수, $y$축 대칭) / $f(-x)=-f(x)$(기함수, 원점 대칭) / 주기성.
  3. 절편 — $x$절편($f(x)=0$), $y$절편($f(0)$).
  4. 점근선 — 수평($\lim_{x\to\pm\infty}f$), 수직($x\to a$에서 발산), 사선(다항/유리에서 차수 차).
  5. $f'$ 부호표 — 단조 구간, 임계점 위치.
  6. $f''$ 부호표 — 볼록 구간, 변곡점.
  7. 표 → 스케치 — 모든 정보를 한 표로 모은 뒤 손으로 곡선을 잇는다.

12.13 종합 예제 — 그래프 그리기

예제 12.13.1. $f(x) = \dfrac{x^2}{x^2-1}$의 그래프를 그려라.

① 정의역: $x \ne \pm 1$.

② 대칭성: $f(-x) = f(x)$ → $y$축 대칭.

③ 절편: $f(0)=0$ → 원점 통과.

④ 점근선: 수직 점근선 $x = \pm 1$, 수평 점근선 $y = 1$ ($x\to\pm\infty$).

⑤ 도함수: $f'(x) = \dfrac{2x(x^2-1) - x^2 \cdot 2x}{(x^2-1)^2} = \dfrac{-2x}{(x^2-1)^2}$.

  • $x<0$: 분자 $>0$ → $f'>0$ → 증가
  • $x>0$: 분자 $<0$ → $f'<0$ → 감소

임계점 $x=0$ 극대 ($f(0)=0$).

⑥ 2차 도함수: 계산하면 $f''(x) = \dfrac{2(3x^2+1)}{(x^2-1)^3}$. 분자는 항상 양수, 분모는 $|x|>1$에서 양수, $|x|<1$에서 음수.

  • $|x|<1$: $f''<0$ → 오목 (점근선 사이의 가운데 봉우리)
  • $|x|>1$: $f''>0$ → 볼록

$x=\pm 1$에서 함수가 정의되지 않으므로 변곡점은 없음.

⑦ 스케치: 가운데 영역에서 0에서 시작해 양쪽으로 발산(아래로). 양 옆에서 $+\infty$로 시작해 1로 수렴(위에서). 표로 정리하면 손으로 그릴 수 있습니다.

12.16 응용 ② — 머신러닝의 손실 함수와 볼록성

딥러닝의 학습은 결국 손실 함수 $L(\theta)$를 최소화하는 일입니다. 만약 $L$이 볼록이라면 지역 최저점이 곧 전역 최저점이라 경사하강법(SGD)이 안전합니다. 선형회귀의 평균제곱오차, 로지스틱 회귀의 교차엔트로피가 그 예입니다.

반면 신경망 전체는 일반적으로 비볼록입니다. 골짜기와 안장점이 잔뜩 섞여 있어 학습률·초기화·정규화 같은 기법이 필요해요. 학기 후반(딥러닝 강의)에서 다시 만납니다.

한 줄 메시지 — "볼록 = 안전한 그릇, 비볼록 = 산악 지형." 그릇이면 굴려서 바닥을 찾고, 산악이면 길을 잘 골라야 합니다.

12.17 응용 ③ — 부등식 증명

볼록성은 부등식 증명의 강력한 도구입니다.

예 1 (예전에 본 AM-GM). $\ln x$의 오목성 → $\ln\!\bigl(\tfrac{a+b}{2}\bigr)\ge\tfrac{\ln a+\ln b}{2}$ → $\dfrac{a+b}{2}\ge\sqrt{ab}$.

예 2 (옌센 부등식). $\phi$가 볼록이면 $\phi\!\bigl(\sum p_i x_i\bigr)\le\sum p_i\phi(x_i)$ ($p_i\ge 0$, $\sum p_i=1$). 확률론·정보이론에서 평생 만납니다.

예 3 ($e^x \ge 1 + x$). $f(x)=e^x - 1 - x$로 두면 $f'(x)=e^x-1$. $x>0$이면 $f'>0$, $x<0$이면 $f'<0$. 따라서 $x=0$이 최솟값, $f(0)=0$. 그러므로 모든 $x$에서 $e^x\ge 1+x$.

12.18 흔한 오답 모음

오답 1. "$f'(x_0)=0$이니까 극값" — 안장 가능. 1계 판정으로 부호 변화 확인.
오답 2. "$f''(x_0)=0$이니까 변곡점" — $x^4$의 $x=0$ 반례. 부호 변화 확인 필수.
오답 3. "도함수가 미분 불가능한 점은 극값이 될 수 없다" — 거짓. $f(x)=|x|$의 $x=0$은 미분 불가능하지만 극소.
오답 4. "MVT에서 $c$를 항상 식으로 정확히 풀 수 있다" — 일반적으로 불가능. 존재만 보장.
오답 5. "단조 증가 = 항상 $f'>0$" — $f(x)=x^3$은 단조 증가지만 $f'(0)=0$. 도함수가 한 점에서 0이어도 전체적으로 증가일 수 있음.
오답 6. "볼록 = 위로 솟은 모양" — 한국어 "볼록"이 직관과 반대로 느껴질 수 있어요. 수학에서 convex(볼록)은 "그릇 모양($\cup$)"입니다. "위로 볼록(concave up)"이라 풀어 부르기도 합니다.

12.22 FAQ

Q1. MVT의 $c$를 항상 풀 수 있나요?
특수한 함수에서만 닫힌 식으로 풀립니다. 일반적으로는 "존재"만 보장. 그래도 부등식 증명·근사·테일러 전개 등에서 이 존재성이 결정적 역할을 합니다.
Q2. "볼록"과 "위로 볼록", "concave up"의 관계?
같은 말입니다. 영어권은 "concave up = 위로 볼록 = $f''>0$"으로 부르고, 한국 교과서는 "볼록(convex)"이라고 줄여 씁니다. "오목(concave) = 위로 오목 = concave down = $f''<0$"입니다.
Q3. 변곡점에서 접선은 어떤 모양인가요?
접선이 곡선을 "통과"합니다(곡선이 접선의 한쪽에서 다른 쪽으로 넘어감). 일반 점에서는 접선이 곡선을 한쪽에서만 스쳐 지나갑니다.
Q4. 1계와 2계 중 어느 게 더 강력한가요?
보편적으로는 1계가 더 안전합니다(부호 변화로 항상 결론). 2계는 빠르지만 $f''(c)=0$일 때 무력해집니다.
Q5. 단조 증가와 강한 단조 증가의 차이?
강한 단조 증가는 $a0$ on $I$이면 강한 단조 증가가 따름.
Q6. $f''>0$만으로 전역 최솟값을 보장하나요?
볼록 함수가 정의역 전체에서 최솟값을 갖는다는 보장은 없습니다(예: $e^x$). 임계점이 존재하고 정의역이 닫힌 구간이거나, 함수가 무한대에서 발산할 때 보장됩니다.
Q7. 미분 불가능한 점에서도 극값이 가능한가요?
네. $f(x)=|x|$의 $x=0$은 극소. 임계점의 정의에 "$f'$ 미분 불가능한 점"도 포함되는 이유입니다.

12.25 마무리 점검

핵심 점검 — 외워야 할 6가지
  • 롤 정리: $f(a)=f(b)$ → $f'(c)=0$ 존재.
  • MVT: $f'(c) = \dfrac{f(b)-f(a)}{b-a}$ 존재.
  • $f'>0$ → 증가, $f'<0$ → 감소.
  • $f''>0$ → 볼록(∪), $f''<0$ → 오목(∩).
  • 임계점에서 1계로 부호 변화 확인 → 극값 분류.
  • 변곡점 = $f$ 연속 + $f''$ 부호 변화.

13. 더 많은 예제 모음 — 손으로 풀어 보기

아래 예제들은 시험 직전에 한 번씩 손으로 다 풀어 보면 좋습니다. 풀이의 단계마다 "왜 이 단계가 필요한가?"를 한 줄씩 옆에 적으면서 진행하세요.

13.1 평균값 정리 — 부등식 증명 응용

예제 13.1.1. 모든 $x>0$에서 $\ln(1+x) < x$임을 MVT로 보여라.

$f(t) = \ln(1+t)$를 $[0, x]$에서 살펴보면, $f$는 연속·미분 가능. MVT에 의해 어떤 $c\in(0, x)$가 있어 $$\frac{\ln(1+x) - 0}{x - 0} = f'(c) = \frac{1}{1+c}.$$ $c>0$이므로 $\dfrac{1}{1+c} < 1$. 양변에 $x>0$ 곱: $\ln(1+x) < x$.

예제 13.1.2. $|\sin a - \sin b| \le |a - b|$ (사인 함수의 1-립시츠 성질).

$f(x)=\sin x$. MVT로 어떤 $c$에서 $\sin a - \sin b = (a-b)\cos c$. $|\cos c|\le 1$이므로 $|\sin a - \sin b| \le |a-b|$.

예제 13.1.3. 모든 실수 $x$에서 $e^x \ge 1 + x$, 등호는 $x=0$.

$g(x)=e^x - 1 - x$. $g'(x) = e^x - 1$. $x>0$이면 $g'>0$ 증가, $x<0$이면 $g'<0$ 감소. 따라서 $g$는 $x=0$에서 최솟값 $g(0)=0$. 즉 $e^x \ge 1+x$.

13.2 단조성 — 추가 연습

예제 13.2.1. $f(x) = x - \ln(1+x^2)$의 단조 구간.

$f'(x) = 1 - \dfrac{2x}{1+x^2} = \dfrac{1+x^2 - 2x}{1+x^2} = \dfrac{(x-1)^2}{1+x^2} \ge 0$.

모든 $x$에서 $f'\ge 0$이고, $f'(1)=0$만 한 점. 따라서 $f$는 전체에서 단조 증가 (한 점에서 평지).

예제 13.2.2. $f(x) = x^3 - 3x^2 + 4$의 단조 구간 및 극값.

$f'(x) = 3x^2 - 6x = 3x(x-2)$. 부호:

  • $x<0$: $f'>0$ 증가
  • $0
  • $x>2$: $f'>0$ 증가

$x=0$ 극대 ($f(0)=4$), $x=2$ 극소 ($f(2)=0$).

예제 13.2.3. $f(x) = \sqrt{x} - \dfrac{x}{2}$의 최댓값.

정의역 $x\ge 0$. $f'(x) = \dfrac{1}{2\sqrt{x}} - \dfrac{1}{2} = \dfrac{1-\sqrt{x}}{2\sqrt{x}}$.

$00$ 증가; $x>1$: $f'<0$ 감소. 따라서 $x=1$에서 최댓값 $f(1)=1/2$.

13.3 볼록성 — 추가 연습

예제 13.3.1. $f(x) = x \ln x$ ($x>0$)의 볼록성.

$f'(x)=\ln x + 1$, $f''(x)=\dfrac{1}{x}>0$. 모든 $x>0$에서 볼록. 따라서 옌센 부등식이 성립: $\dfrac{a\ln a + b\ln b}{2} \ge \dfrac{a+b}{2}\ln\dfrac{a+b}{2}$ — 정보이론의 핵심 부등식.

예제 13.3.2. $f(x) = \arctan x$의 볼록 구간.

$f'(x) = \dfrac{1}{1+x^2}$, $f''(x) = \dfrac{-2x}{(1+x^2)^2}$.

  • $x<0$: $f''>0$ 볼록
  • $x>0$: $f''<0$ 오목

$x=0$이 변곡점. $\arctan$ 그래프는 왼쪽이 그릇, 오른쪽이 엎은 그릇.

예제 13.3.3. $f(x) = e^{-x}\sin x$의 변곡점 (한 주기 안에서 위치 형태만).

$f'(x) = e^{-x}(\cos x - \sin x)$, $f''(x) = e^{-x}(-2\cos x)$ (정리하면).

$f''=0$ → $\cos x = 0$ → $x = \pi/2 + k\pi$. 양 옆에서 부호가 바뀌므로 모두 변곡점. 진동이 점점 줄어드는 감쇠 사인의 굴곡 위치를 정확히 알려 줍니다.

13.4 그래프 그리기 — 추가 종합 예제

예제 13.4.1. $f(x) = \dfrac{e^x}{x}$ ($x\ne 0$).

정의역 $x\ne 0$. 점근선: $x=0$ (수직), $x\to-\infty$일 때 $e^x/x \to 0^-$, $x\to\infty$일 때 $\to\infty$.

$f'(x) = \dfrac{e^x(x-1)}{x^2}$. 부호:

  • $x<0$: $e^x>0$, $x-1<0$, $x^2>0$ → $f'<0$ 감소
  • $0
  • $x>1$: $f'>0$ 증가

$x=1$에서 극소 $f(1)=e$.

$f''(x) = \dfrac{e^x(x^2-2x+2)}{x^3}$. 분자 항상 양수 ($x^2-2x+2 = (x-1)^2+1>0$), 분모 $x^3$의 부호: $x>0$이면 양, $x<0$이면 음. 따라서 $x>0$에서 볼록, $x<0$에서 오목. 변곡점은 없음 (점근선에서 분리).

예제 13.4.2. $f(x) = x^{2/3}(x-5)$.

정리: $f(x) = x^{5/3} - 5x^{2/3}$. $f'(x) = \dfrac{5}{3}x^{2/3} - \dfrac{10}{3}x^{-1/3} = \dfrac{5(x-2)}{3x^{1/3}}$.

$f'$가 0 또는 미정인 곳: $x=2$ ($f'=0$), $x=0$ (미분 불가능). 이 두 곳이 임계점.

  • $x<0$: $x-2<0$, $x^{1/3}<0$ → $f'>0$ 증가
  • $00$ → $f'<0$ 감소
  • $x>2$: $f'>0$ 증가

$x=0$에서 극대(첨점), $x=2$에서 극소.

이 예제의 교훈 — 미분 불가능한 점도 임계점이 될 수 있고, 거기서 극값이 등장할 수 있습니다.

13.5 코시 평균값 정리 — 보너스

일반화된 평균값 정리(코시 MVT)는 두 함수의 비를 다룹니다. $f, g$가 연속·미분 가능하고 $g'\ne 0$이면 어떤 $c\in(a, b)$에서 $$\frac{f(b)-f(a)}{g(b)-g(a)} = \frac{f'(c)}{g'(c)}.$$

이 정리가 결정적으로 쓰이는 곳: 로피탈의 규칙 증명. $0/0$ 또는 $\infty/\infty$ 형태의 극한을 도함수 비로 바꿔 풀 때 쓰입니다.

한 컷 요약 — MVT가 "한 함수의 평균-순간 일치"라면, 코시 MVT는 "두 함수의 평균비-순간비 일치". 로피탈은 이 정리의 따름.

13.6 옌센 부등식 한 번 더

볼록 함수 $\phi$와 가중치 $p_1, \dots, p_n \ge 0$ ($\sum p_i = 1$)에 대해 $$\phi\!\left(\sum_{i=1}^n p_i x_i\right) \le \sum_{i=1}^n p_i \phi(x_i).$$ 오목 함수면 부등호 방향이 반대입니다. 활용 예:

  • $\phi(x)=x^2$ 볼록 → $\bigl(\sum p_i x_i\bigr)^2 \le \sum p_i x_i^2$ → 분산이 음수가 아님.
  • $\phi(x)=-\ln x$ 볼록 → $-\ln\!\bigl(\sum p_i x_i\bigr) \le -\sum p_i \ln x_i$ → 산술-기하 평균(가중) 부등식.
  • 정보이론의 KL 발산이 0 이상임을 보장.

14. 부록 — 시험 직전 체크리스트와 미니 사전

14.1 시험 직전 한 페이지 요약

외워서 가는 6공식
  1. 롤: $f(a)=f(b)$ → $\exists c, f'(c)=0$.
  2. MVT: $\exists c, f'(c) = \dfrac{f(b)-f(a)}{b-a}$.
  3. 코시 MVT: $\dfrac{f(b)-f(a)}{g(b)-g(a)} = \dfrac{f'(c)}{g'(c)}$.
  4. $f' \gtrless 0$ → 증감 / $f''\gtrless 0$ → 볼록·오목.
  5. 1계 판정: 임계점 부호 변화로 극대·극소.
  6. 변곡점 = $f$ 연속 + $f''$ 부호 변화.

14.2 시험 직전 5분 점검 체크리스트

  • □ "롤 정리"의 3가지 가정을 즉시 댈 수 있다.
  • □ MVT의 결론 식을 막힘 없이 쓸 수 있다.
  • □ 단조 증가가 $f'>0$의 충분조건임은 알지만 필요조건은 아님을 안다.
  • □ "$f'(c)=0$ ≠ 극값"의 반례($x^3$)를 댈 수 있다.
  • □ "$f''(c)=0$ ≠ 변곡점"의 반례($x^4$)를 댈 수 있다.
  • □ 1계와 2계 판정법의 적용 우선순위를 안다.
  • □ 그래프 7단계 절차를 순서대로 외운다.
  • □ 부호표를 표로 그릴 줄 안다.
  • □ 옌센 부등식의 방향(볼록 vs 오목)을 헷갈리지 않는다.
  • □ 미분 불가능한 점도 임계점에 포함됨을 안다.

한국어 "볼록"과 "오목"은 일상적으로 "위로 솟다 / 움푹 패다"로 쓰이지만, 수학에서는 다음과 같이 약속합니다.

표현의미그림$f''$
볼록 (convex) = 위로 볼록 = concave up그릇 모양∪$f''>0$
오목 (concave) = 위로 오목 = concave down엎은 그릇 모양∩$f''<0$
주의 — "위로 볼록"이라는 말이 "솟아 있다(∩)"는 일상 직관과 반대로 들릴 수 있습니다. 수학에서는 "위쪽에서 봤을 때 그릇처럼 들어간 곡선"을 볼록이라 합니다. 헷갈리면 그림 ∪/∩과 $f''$ 부호로 기억하세요.

14.5 자주 받는 질문 — 학생 사례

사례 1. "$f(x)=|x|$의 $x=0$은 임계점인가요?" — 네. $f$가 $x=0$에서 미분 불가능하지만 정의역 안의 점이고 극소이므로 임계점입니다. 임계점의 정의에 미분 불가능 점이 포함되는 것을 잊지 마세요.
사례 2. "MVT를 쓰려면 항상 양 끝점에서 미분 가능해야 하나요?" — 아닙니다. 열린 구간 $(a, b)$에서만 미분 가능하면 됩니다. 닫힌 구간에서는 연속이면 충분.

14.7 학습 보너스 — 한 줄 정의 카드 12장

다음 12장의 카드를 종이에 옮겨 적고, 한 면에는 이름·다른 면에는 정의·예를 적은 뒤 셔플해 가며 외우면 시험 직전 30분 만에 전체 정리가 됩니다.

  1. 롤 정리 — 양 끝 같으면 도함수 0인 점 존재.
  2. 평균값 정리 — 평균 변화율 = 어떤 순간 도함수.
  3. 코시 MVT — 두 함수의 평균비 = 어떤 점의 도함수 비.
  4. 임계점 — $f'=0$ 또는 미분 불가능한 정의역 내부 점.
  5. 1계 판정법 — 임계점 부호 변화: +→− 극대, −→+ 극소.
  6. 2계 판정법 — $f'=0$이고 $f''\ne 0$일 때, $f''$ 부호로 분류.
  7. 볼록 — $f''>0$, ∪ 모양, 할선이 곡선 위.
  8. 오목 — $f''<0$, ∩ 모양, 할선이 곡선 아래.
  9. 변곡점 — $f$ 연속 + $f''$ 부호 변화.
  10. 안장점 — 임계점인데 극값 아님 (예: $x^3$의 0).
  11. 옌센 부등식 — 볼록 함수의 평균과 함숫값 평균 비교.
  12. 립시츠 성질 — $|f(a)-f(b)|\le L|a-b|$, MVT의 따름.
