1. 적분이란 — 미분의 역

한 줄 비유부터

미분이 "위치를 시간으로 나누어 속도로 바꾸는" 작업이라면, 적분은 "속도에 시간을 곱해 다시 위치로 모으는" 작업이에요. 도함수가 주어졌을 때 본 함수가 무엇이었는지 거꾸로 추적합니다.

정의 — 부정적분

$F'(x) = f(x)$ 인 함수 $F$ 를 $f$의 부정적분(antiderivative) 또는 원시함수라 합니다. 표기:

$$ \int f(x)\, dx = F(x) + C $$

$+ C$ 는 적분 상수. 도함수가 0인 함수는 상수 차이만큼 자유롭기 때문에 부정적분도 무한히 많고, 그 차이는 항상 상수예요 (W5 MVT의 결과).

2. 기본 부정적분 표

다음 12개를 외워 둡니다.

$$ \int x^n\, dx = \frac{x^{n+1}}{n+1} + C \quad (n \ne -1), \qquad \int \frac{1}{x}\, dx = \ln|x| + C $$ $$ \int e^x\, dx = e^x + C, \qquad \int a^x\, dx = \frac{a^x}{\ln a} + C $$ $$ \int \sin x\, dx = -\cos x + C, \qquad \int \cos x\, dx = \sin x + C $$ $$ \int \sec^2 x\, dx = \tan x + C, \qquad \int \csc^2 x\, dx = -\cot x + C $$ $$ \int \frac{1}{\sqrt{1-x^2}}\, dx = \sin^{-1} x + C, \qquad \int \frac{1}{1+x^2}\, dx = \tan^{-1} x + C $$ $$ \int \sec x \tan x\, dx = \sec x + C, \qquad \int \csc x \cot x\, dx = -\csc x + C $$

이 표는 미분 공식 표(W3·W4)의 거울상. 미분 공식이 머리에 있다면 적분 공식은 자동으로 따라옵니다.

3. 치환적분 ($u$-치환)

한 줄 비유부터

식 안에 "내가 알고 있는 적분 모양"이 숨어 있는데, 변수 이름만 바꾸면 그 모양이 드러난다 — 그게 $u$-치환의 정신이에요. 미분의 연쇄 법칙을 거꾸로 풀어내는 도구.

공식

$u = g(x)$ 로 두면 $du = g'(x) dx$. 그러면

$$ \int f(g(x)) g'(x)\, dx = \int f(u)\, du $$

예제 3.1 — $\displaystyle \int x \cos(x^2)\, dx$

$u = x^2$ → $du = 2x\, dx$ → $x\, dx = \tfrac{1}{2} du$.

$$ \int x \cos(x^2)\, dx = \int \cos u \cdot \tfrac{1}{2}\, du = \tfrac{1}{2}\sin u + C = \tfrac{1}{2}\sin(x^2) + C $$

검산: 답을 미분해 보면 $\tfrac{1}{2}\cos(x^2) \cdot 2x = x\cos(x^2)$ ✓.

예제 3.2 — $\displaystyle \int \frac{2x}{x^2 + 1}\, dx$

$u = x^2 + 1$ → $du = 2x\, dx$.

$$ \int \frac{2x}{x^2+1}\, dx = \int \frac{1}{u}\, du = \ln|u| + C = \ln(x^2 + 1) + C $$

치환 패턴 알아보기

안쪽 함수의 도함수가 식 어딘가에 곱해져 있다 → 치환 가능. 익숙해지려면 "$g(x)$와 $g'(x)$가 짝지어 보이는지"를 묻는 연습이 가장 좋습니다.

4. 치환 패턴 6가지 — 빠른 매칭표

학부 시험에서 자주 등장하는 치환 패턴을 정리. 식을 보자마자 어떤 치환을 시도할지 한 줄에 떠오르도록 연습하세요.

식 모양치환이유
$\int f(g(x)) g'(x) dx$$u = g(x)$가장 기본 — 안 쪽 도함수가 보임
$\int x \cdot f(x^2) dx$$u = x^2$, $du = 2x dx$$x$가 따라옴 → $x^2$ 치환
$\int \frac{f'(x)}{f(x)} dx$$u = f(x)$$\ln|f|$ 답
$\int \sin^n x \cos x\,dx$$u = \sin x$한 인수가 도함수
$\int e^{f(x)} f'(x) dx$$u = f(x)$$e^u$ 표 적분
$\int (\ln x)^n / x \, dx$$u = \ln x$$1/x = (\ln x)'$

예제 4.1 — 패턴 ③ (로그)

$\displaystyle \int \frac{\cos x}{\sin x}\,dx$. 분자 $= (\sin x)'$, 분모 $= \sin x$ → $u = \sin x$.

$$ = \ln|\sin x| + C $$

모든 $\int \tan, \int \cot$ 적분이 이 패턴.

예제 4.2 — 패턴 ⑥ (로그의 거듭제곱)

$\displaystyle \int \frac{(\ln x)^2}{x}\,dx$. $u = \ln x$, $du = dx/x$.

$$ = \int u^2\,du = \frac{u^3}{3} + C = \frac{(\ln x)^3}{3} + C $$

5. 부분적분 (Integration by Parts)

유도 — 곱의 미분의 역

$(uv)' = u' v + u v'$ 양변을 적분.

$$ uv = \int u'v\, dx + \int u v'\, dx \;\Rightarrow\; \int u v'\, dx = uv - \int u' v\, dx $$

흔히 $u$ 와 $dv$ 로 표기:

$$ \int u\, dv = uv - \int v\, du $$

LIATE — $u$ 우선순위

두 인수 중 어느 것을 $u$ 로 둘지 헷갈릴 때, 이 순서대로 위에 있는 쪽을 $u$:

  • Logarithm — 로그 ($\ln x$)
  • Inverse trig — 역삼각 ($\sin^{-1}, \tan^{-1}$)
  • Algebraic — 다항 ($x^n$)
  • Trigonometric — 삼각 ($\sin x, \cos x$)
  • Exponential — 지수 ($e^x$)

왜? 위쪽일수록 미분하면 쉬워지고, 아래쪽일수록 적분이 쉬운 함수라서 — 식이 단순해지는 방향.

예제 4.1 — $\displaystyle \int x e^x\, dx$

LIATE: A(다항)이 E(지수) 위. $u = x$, $dv = e^x\, dx$.

$du = dx$, $v = e^x$.

$$ \int x e^x\, dx = x e^x - \int e^x\, dx = x e^x - e^x + C = (x-1)e^x + C $$

예제 4.2 — $\displaystyle \int \ln x\, dx$

$u = \ln x$, $dv = dx$. $du = dx/x$, $v = x$.

$$ \int \ln x\, dx = x \ln x - \int x \cdot \tfrac{1}{x}\, dx = x \ln x - x + C $$

예제 4.3 — 두 번 부분적분

$\displaystyle \int x^2 e^x\, dx$. $u = x^2$, $dv = e^x dx$ → $\int = x^2 e^x - \int 2x e^x\, dx$. 안쪽 적분도 부분적분 한 번 더.

$$ = x^2 e^x - 2[(x-1)e^x] + C = (x^2 - 2x + 2) e^x + C $$

6. 부분적분 — 순환 트릭과 표 방식

순환(boomerang) 트릭

두 번 부분적분 후 원래 적분이 다시 등장하면, 양변에서 같은 항을 모아 풀 수 있어요. 대표 예제:

예제 6.1 — $\displaystyle I = \int e^x \cos x\, dx$

$u = e^x, dv = \cos x\,dx$ → $du = e^x\,dx, v = \sin x$.

$$ I = e^x \sin x - \int e^x \sin x\, dx $$

오른쪽 적분도 한 번 더 부분적분 ($u = e^x, dv = \sin x\,dx$): $\int e^x \sin x\,dx = -e^x\cos x + I$. 대입:

$$ I = e^x\sin x - (-e^x\cos x + I) = e^x(\sin x + \cos x) - I $$ $$ \Rightarrow\; 2I = e^x(\sin x + \cos x) \;\Rightarrow\; I = \tfrac{1}{2} e^x(\sin x + \cos x) + C $$

두 번 만에 끝. 부분적분만으로 풀이가 무한히 늘어날 위험을 막는 트릭.

표 방식 (Tabular Integration) — $x^n$ 곱하기 다른 함수

$x^n$ 처럼 미분하면 결국 0이 되는 인수와 다른 인수의 곱은, 표를 그려 부호를 번갈아 곱하면 됩니다.

예: $\displaystyle \int x^3 e^{2x}\, dx$. 표:

  미분 (D)        적분 (I)        부호
   x^3            e^(2x)           +
   3x^2           e^(2x)/2         −
   6x             e^(2x)/4         +
   6              e^(2x)/8         −
   0              e^(2x)/16

대각선으로 곱하고 부호 번갈아:

$$ \int x^3 e^{2x}\,dx = \tfrac{1}{2}x^3 e^{2x} - \tfrac{3}{4}x^2 e^{2x} + \tfrac{6}{4} \cdot \tfrac{1}{2} x e^{2x} - \tfrac{6}{8} e^{2x} + C $$

표를 그리는 게 부분적분을 4번 따로 적는 것보다 빠릅니다. 시험에서 시간을 절약하는 단골 기법.

7. 삼각치환 입문

$\sqrt{a^2 - x^2}$, $\sqrt{a^2 + x^2}$, $\sqrt{x^2 - a^2}$ 가 들어 있는 식은 다음 치환으로 풉니다.

  • $\sqrt{a^2 - x^2}$: $x = a \sin\theta$ → $\sqrt{a^2 - a^2 \sin^2\theta} = a\cos\theta$.
  • $\sqrt{a^2 + x^2}$: $x = a \tan\theta$ → $\sqrt{a^2 + a^2\tan^2\theta} = a \sec\theta$.
  • $\sqrt{x^2 - a^2}$: $x = a \sec\theta$.

예제 5.1 — $\displaystyle \int \sqrt{1 - x^2}\, dx$

$x = \sin\theta$, $dx = \cos\theta\, d\theta$, $\sqrt{1-x^2} = \cos\theta$.

$$ \int \cos^2\theta\, d\theta = \int \frac{1 + \cos 2\theta}{2}\, d\theta = \frac{\theta}{2} + \frac{\sin 2\theta}{4} + C $$

$\theta = \sin^{-1} x$ 로 되돌리고, $\sin 2\theta = 2\sin\theta\cos\theta = 2x\sqrt{1-x^2}$ 로 정리.

$$ = \frac{1}{2}\sin^{-1} x + \frac{x\sqrt{1-x^2}}{2} + C $$

이 적분은 단위 반원의 면적 공식과 직결됩니다 — $\int_{-1}^{1}\sqrt{1-x^2}\,dx = \pi/2$.

8. 부분분수 분해 입문

유리함수 $\frac{P(x)}{Q(x)}$ 의 적분은 부분분수로 분해해서 표 적분으로 환원합니다.

예제 6.1 — $\displaystyle \int \frac{1}{x^2 - 1}\, dx$

$\dfrac{1}{x^2 - 1} = \dfrac{1}{(x-1)(x+1)} = \dfrac{1/2}{x-1} - \dfrac{1/2}{x+1}$ (계수는 미정계수법으로).

$$ \int \frac{1}{x^2-1}\, dx = \tfrac{1}{2}\ln|x-1| - \tfrac{1}{2}\ln|x+1| + C = \tfrac{1}{2}\ln\left|\tfrac{x-1}{x+1}\right | + C $$

9. 적분이 닫힌 식으로 안 풀리는 함수들

한 줄 비유부터

모든 함수가 닫힌 식으로 적분되지는 않습니다. 도함수는 항상 어떤 함수든 적을 수 있지만, 부정적분은 그렇지 않아요. 닫힌 식이 없을 때는 (1) 정의로 받아들이고 새 이름을 붙이거나 (2) 수치적분으로 푸는 게 보통.

대표 예 — 닫힌 식 없는 적분

  • $\displaystyle \int e^{-x^2}\,dx$ — 닫힌 식 없음. 오차함수 $\mathrm{erf}(x) = \frac{2}{\sqrt{\pi}} \int_0^x e^{-t^2}dt$ 로 정의 (통계·신호처리의 기초).
  • $\displaystyle \int \frac{\sin x}{x}\,dx$ — 닫힌 식 없음. 사인 적분 함수 $\mathrm{Si}(x)$.
  • $\displaystyle \int \frac{1}{\ln x}\,dx$ — 로그 적분 $\mathrm{li}(x)$ (소수 분포 정리에 등장).
  • $\displaystyle \int e^{x^2}\,dx, \int \sin(x^2)\,dx$ (Fresnel) 등.

왜 알아 둬야 하나?

시험에서 풀리지 않는 적분을 만나 시간을 낭비하지 않으려면 — "이건 닫힌 식이 없을 수도 있다"는 신호를 알아채는 감각이 중요. $e^{-x^2}, \sin(x^2), \cos(x^2)$ 같은 합성을 만나면 의심해 보세요. 실무에서는 W10에서 배울 정적분이 닫힌 식 없이도 수치적으로 정확히 계산되니까 큰 문제는 안 됩니다.

10. 인터랙티브: $u$-치환 시각화

1차원 변수 변환에서 면적이 어떻게 변하는지 시각적으로 보여 줍니다. $u = g(x)$ 변환은 $x$-축의 작은 구간 $dx$ 를 $u$-축의 작은 구간 $du = g'(x) dx$ 로 늘이거나 줄여요. 이게 다변수에서 자코비안 행렬식의 1차원 버전이에요.

🎮 변수 변환과 면적의 비례

변환 $u = g(x)$ 를 골라 보세요. 같은 폭의 작은 막대가 $x$-축에서 $u$-축으로 어떻게 늘이거나 줄어드는지가 보입니다. 이 늘림비율이 정확히 $g'(x)$.

11. 인터랙티브: 적분 기법 분류 퀴즈

적분 식을 보고 어떤 기법(치환·부분적분·삼각치환·부분분수)을 써야 하는지 한 줄에 떠올리는 연습. 8문제 중 답을 골라 즉시 정답을 확인합니다.

🎮 적분 기법 매칭 퀴즈

각 적분에 가장 적합한 첫 기법을 선택. 정답 표기와 짧은 풀이 힌트가 함께 나옵니다.

12. 흔한 오해와 핵심 정리

흔한 오해 ① — "$\int dx/x = \ln x$" — 정확히는 $\int dx/x = \ln |x| + C$. 절댓값을 빼먹으면 음수 영역에서 정의가 깨집니다.
흔한 오해 ② — "치환할 때 $dx$ 만 바꾸면 된다" — $dx$ 를 $du/g'(x)$ 로 바꾸고, 피적분함수의 모든 $x$도 $u$로 표현해야 합니다. 그렇게 안 되면 치환을 잘못 잡은 것.
흔한 오해 ③ — "부분적분에서 $u, dv$ 의 선택은 아무거나 좋다" — 잘못 잡으면 식이 더 복잡해집니다. LIATE 우선순위를 따라 위에 있는 함수를 $u$로 두는 게 안전.

이번 주의 핵심 한 문장

"미분의 역을 풀려면 12개 표 적분 + 치환(연쇄 법칙의 역) + 부분적분(곱 법칙의 역). 이 셋이 학부 적분 기술의 90%."
