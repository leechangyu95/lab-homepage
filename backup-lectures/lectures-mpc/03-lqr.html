Week 3. LQR — 무제약 최적 제어

제약을 잠시 잊으면 최적 제어 문제는 우아한 닫힌형 해를 갖습니다. 그게 LQR(Linear Quadratic Regulator)이에요. 벨만의 동적계획법으로 한 줄씩 유도해보고, 무한 호라이즌이 어떻게 매트릭스 한 번 풀어 끝나는지 봅니다. MPC의 출발점이자 종단 비용 설계의 기준점.

목차

  1. 왜 LQR을 먼저 배우는가
  2. 유한 호라이즌 LQR — 벨만 + DP
  3. 리카티 차분방정식 (Riccati Difference Equation)
  4. 무한 호라이즌 — DARE
  5. 안정성과 게인 마진

1. 왜 LQR을 먼저 배우는가

MPC는 결국 "제약이 있는 LQR"입니다. 제약을 빼면 정확히 LQR이 되고, 제약이 활성화되지 않은 상태에서는 LQR과 동일한 입력을 냅니다. 따라서 LQR을 모르고 MPC를 짜는 건 사칙연산 없이 미적분을 배우는 것과 같습니다. 또한 MPC의 종단 비용 행렬 $P$는 보통 LQR 무한 호라이즌 해(DARE)로 두는 것이 표준입니다 (W6에서 안정성 보장과 함께 다시 등장).

2. 유한 호라이즌 LQR

문제 정식화

이산 LTI 시스템 $x_{k+1} = A x_k + B u_k$에 대해, 다음 비용을 최소화하는 입력 시퀀스를 찾습니다:

$$ J(x_0) = \sum_{k=0}^{N-1} \left( x_k^\top Q x_k + u_k^\top R u_k \right) + x_N^\top P_f x_N $$
  • $Q \succeq 0$ — 상태 가중치. 보통 $Q = \text{diag}(q_1, \ldots, q_n)$.
  • $R \succ 0$ — 입력 가중치. $R$이 양정치여야 해가 unique.
  • $P_f \succeq 0$ — 종단 비용. "$N$ 스텝 후의 상태에 얼마나 벌점을 줄까".

벨만의 원리 — 거꾸로 풀기

벨만 원리: "최적 정책의 끝부분도 그 자체로 최적이다". 즉 $k$에서의 최적 입력은 $k+1$ 이후의 최적 비용 함수만 알면 결정됩니다. 이를 이용해 거꾸로 시간(backward in time) 재귀를 세웁니다.

"value function" $V_k(x) = $ "시점 $k$에서 상태 $x$일 때 남은 최적 비용"을 정의:

$$ V_N(x) = x^\top P_f x $$ $$ V_k(x) = \min_{u}\ \{x^\top Q x + u^\top R u + V_{k+1}(Ax + Bu)\} $$

핵심 추측: $V_k$가 이차형 $V_k(x) = x^\top P_k x$ 라고 가정하면, 위 미분 = 0으로 풀어:

$$ u_k^* = -(R + B^\top P_{k+1} B)^{-1} B^\top P_{k+1} A\, x_k = -K_k x_k $$

대입하면 $P_k$는 다음 재귀를 만족:

$$ \boxed{\; P_k = Q + A^\top P_{k+1} A - A^\top P_{k+1} B (R + B^\top P_{k+1} B)^{-1} B^\top P_{k+1} A \;} $$

이걸 리카티 차분방정식 (Riccati Difference Equation)이라 부릅니다. $P_N = P_f$에서 시작해 거꾸로 $P_{N-1}, P_{N-2}, \ldots, P_0$를 계산한 뒤, 게인 $K_k$를 얻으면 끝.

핵심 사실 — LQR의 최적 입력은 시점에 따라 달라지는 선형 상태 피드백 $u_k = -K_k x_k$입니다. 시변(time-varying) 게인을 N 스텝 거꾸로 한 번만 계산해두면 끝.

3. 무한 호라이즌 — DARE

$N \to \infty$로 보내면, 호라이즌이 충분히 길어진 시점부터 $P_k$는 고정값 $P_\infty$로 수렴합니다. 이 고정점을 만족하는 행렬 방정식이 이산 대수 리카티 방정식 (Discrete Algebraic Riccati Equation, DARE):

$$ P = Q + A^\top P A - A^\top P B (R + B^\top P B)^{-1} B^\top P A $$

이를 푸는 함수가 scipy.linalg.solve_discrete_are(A, B, Q, R) 또는 MATLAB dlqr(A,B,Q,R). 결과로 시간 불변(time-invariant) 게인:

$$ K = (R + B^\top P B)^{-1} B^\top P A,\quad u_k = -K x_k $$

이 한 게인 행렬을 매 스텝마다 그대로 쓰면 됩니다 — 단순한 선형 상태 피드백.

존재·안일성 조건

  • $(A, B)$가 안정화 가능 — 불안정 모드는 입력으로 제어 가능.
  • $(A, Q^{1/2})$가 검출 가능 — 비용에 반영되지 않는 불안정 모드가 없음.

이 둘이 만족되면 DARE의 양정치 해가 존재하고 유일하며, 닫힌 루프 $A - BK$가 안정합니다.

🎮 인터랙티브: LQR 닫힌 루프 응답

2차 시스템 $\ddot x = u$를 이산화해 LQR로 0으로 보냅니다. $Q/R$ 비율을 키우면 빠르게 수렴하지만 입력이 거칠어지고, 줄이면 부드럽지만 느립니다. 초기 위치도 조절 가능.

4. 안정성과 게인 마진

LQR이 매력적인 또 다른 이유: 훌륭한 강건성 보장.

  • 게인 여유 [1/2, ∞) — 입력 채널의 게인을 0.5배 ~ 무한배 곱해도 안정 유지.
  • 위상 여유 ≥ 60° — 위상 변화에 매우 강함.

이는 LQR의 "내장된 강건성" 결과로, PID는 훨씬 작은 마진을 가집니다. 그래서 LQR은 모델 오차에 자연스럽게 강합니다.

하지만 LQG (LQR + 칼만 필터)에서는 이 마진이 깨질 수 있다는 것이 1978년 Doyle의 유명한 결과 ("Guaranteed Margins for LQG Regulators" = "보장 없음"). 그래서 강건 제어 (H∞)가 발전했습니다.

5. LQR vs PID — 언제 무엇을

6. 코드 예제

import numpy as np
from scipy.linalg import solve_discrete_are

# 2차 적분기 시스템 (이산화 dt=0.1)
A = np.array([[1, 0.1],[0, 1]])
B = np.array([[0],[0.1]])
Q = np.diag([10, 1])     # 위치 10, 속도 1 가중치
R = np.array([[1]])       # 입력 1

# DARE 풀기
P = solve_discrete_are(A, B, Q, R)

# LQR 게인
K = np.linalg.inv(R + B.T @ P @ B) @ B.T @ P @ A
print("LQR gain K =", K)

# 폐루프 시뮬레이션
x = np.array([3.0, 0.0])
for _ in range(50):
    u = -K @ x
    x = A @ x + B.flatten() * u[0]
    print(f"x = {x}, u = {u[0]:.3f}")

7. 핵심 정리

  • LQR은 무제약 이차 비용 + 선형 시스템 = 닫힌형 해. 매트릭스 한 번 풀어 끝.
  • 유한 호라이즌은 시변 게인 $K_k$, 무한 호라이즌은 시불변 게인 $K = $ dlqr.
  • 닫힌 루프는 자동 안정 + 좋은 강건성 마진 (게인 1/2~∞, 위상 60°+).
  • MPC의 종단 비용 $P$는 보통 LQR DARE 해로 두는 것이 표준.
  • 제약을 추가하면 닫힌형 해가 사라지고 → 다음 주의 CFTOC + QP로.

📚 보강 1 — LQR을 직관으로

LQR(Linear Quadratic Regulator)은 가장 단순한 최적 제어. 무한 호라이즌 + 이차 비용 + 선형 모델 + 무제약. 닫힌 해가 존재 — 그게 매력입니다.

1.1 친절 비유

  • 주차장 자동 정렬 — 차를 가능한 빠르게(작은 $R$), 그러나 너무 급격하지 않게(큰 $R$) 주차 위치로 이동.
  • 난방 제어 — 빠르게 목표 온도(작은 $R$) vs 부드러운 온도 변화(큰 $R$).
  • 드론 자세 — 빠른 자세 추종 vs 모터 RPM 한계.

1.2 핵심 식 한 줄

$$J = \sum_{k=0}^\infty (x_k^T Q x_k + u_k^T R u_k)$$

최적 제어 $u_k = -K x_k$, 게인 $K$는 DARE(Discrete Algebraic Riccati Equation) 해로 결정.

📚 보강 2 — Riccati 식 단계별 유도

2.1 Bellman 방정식

최적 비용 함수 $V^*(x) = x^T P x$로 가정.

$$V^*(x_k) = \min_{u_k} (x_k^TQx_k + u_k^TRu_k + V^*(x_{k+1}))$$

2.2 대입과 전개

$x_{k+1} = Ax_k + Bu_k$ 대입. 미분 = 0:

$$u_k^* = -(R + B^TPB)^{-1}B^TPA\, x_k = -K x_k$$

2.3 DARE

$P$가 만족하는 식:

$$P = A^TPA - A^TPB(R + B^TPB)^{-1}B^TPA + Q$$

이게 이산 Riccati 식. scipy.linalg.solve_discrete_are로 한 줄 풀이.

📚 보강 3 — 단계별 예제

3.1 예제 — 1차원 LQR

$x_{k+1} = x_k + u_k$, $Q = 1, R = 1$. 최적 $K$?

DARE: $P = P - P^2/(1+P) + 1$ → $P(1+P) - P^2 + (1+P) = P + P^2 - P^2 + 1 + P = 2P + 1$. 같으면 $P(1+P) = 2P+1$ → $P = (1+\sqrt{5})/2 \approx 1.618$ (golden ratio!).
$K = P/(1+P) \approx 0.618$. 닫힌 루프 $A - BK \approx 0.382$, 안정.

3.2 예제 — 2차원 LQR (도립진자 1축)

$A = \begin{pmatrix}1 & 0.01 \\ 0 & 1\end{pmatrix}$, $B = \begin{pmatrix}0 \\ 0.1\end{pmatrix}$, $Q = I, R = 1$. scipy로 풀이.

from scipy.linalg import solve_discrete_are
A = [[1, 0.01], [0, 1]]
B = [[0], [0.1]]
Q = [[1, 0], [0, 1]]
R = [[1]]
P = solve_discrete_are(A, B, Q, R)
K = inv(R + B.T @ P @ B) @ B.T @ P @ A

3.3 예제 — Q/R 비율 변화

$Q$ 키우면 → 빠른 추적. $R$ 키우면 → 부드러운 입력. 1차원에서 직관 확인.

📚 보강 4 — LQR 안정성과 강건성

  • 안정성: $A - BK$ 모든 고유값 $|\lambda| < 1$.
  • 게인 마진: $1/2 \le k \le \infty$ (무한 게인 마진).
  • 위상 마진: 60° 이상.
  • 이런 자연스러운 강건성이 LQR의 매력.

📚 보강 6 — 흔한 오답

  1. DARE를 손으로 풀려 함 (3차원 이상은 비현실적).
  2. $Q, R$ 단위 무시.
  3. $Q \ge 0, R > 0$ 조건 빠뜨림.
  4. 가제어 안 되는 시스템에 LQR 적용.

📚 보강 8 — FAQ

Q1. LQR이 안정성을 보장하는 이유?

$P > 0$이 Lyapunov 함수 → 닫힌 루프 시스템의 에너지가 감소. 따라서 안정.

Q2. $Q, R$ 어떻게 선택?

대각 행렬로 시작 → Bryson 규칙: 대각 원소 = 1/(허용 최대값)². 그 후 시뮬로 튜닝.

Q3. 무한 호라이즌이 현실적인가?

이론. 실무는 정착 후 비용이 거의 0이라 무한이나 유한이나 같음.

Q4. LQR 외 다른 무한 호라이즌 제어?

$H_\infty$ 제어, LQG (LQR + Kalman). MPC도 무한 호라이즌으로 근사 가능.

Q5. MPC 종단 비용 $P_f$는 보통 LQR DARE 해?

네. 무한 호라이즌 cost-to-go의 근사. 안정성 보장에도 사용.

📚 보강 13 — 추가 워크북 5문제

워크 1

$A = 0.9, B = 1, Q = R = 1$. $P, K$?

DARE: $P = 0.81P - 0.81P^2/(1+P) + 1$. 풀면 $P \approx 1.236$, $K = 0.9P/(1+P) \approx 0.498$.

워크 2

$Q = 100, R = 1$. 응답은?

큰 $K$, 빠른 추적, 입력 큼.

워크 3

$Q = 1, R = 100$. 응답은?

작은 $K$, 느린 추적, 부드러운 입력.

워크 4

가관측 안 되는 모드가 $Q$로 보강 안 되면?

DARE에 영향 없음 — 그러나 옵저버에서 그 모드를 못 보면 안정성 손상.

워크 5

가제어 안 되는 시스템에 LQR?

DARE 무한대 발산. 가제어 부분만 제어 가능.

📚 보강 15 — Bryson 규칙 — Q/R 첫 선택

$Q_{ii} = 1/(x_i^{\max})^2$, $R_{jj} = 1/(u_j^{\max})^2$. 각 변수의 허용 최대값 역수 제곱. 그 후 시뮬로 미세 조정.

📚 보강 16 — LQR 특수 케이스

  • $R \to \infty$: $K \to 0$, 입력 0, 시스템 자유 응답.
  • $R \to 0$: $K$ 매우 큼, deadbeat 응답.
  • $Q$ 일부만 0: 그 상태는 자유, 다른 것만 정착.

import numpy as np
from scipy.linalg import solve_discrete_are

A = np.array([[1, 0.1], [0, 1]])
B = np.array([[0], [0.1]])
Q = np.eye(2)
R = np.array([[1]])

P = solve_discrete_are(A, B, Q, R)
K = np.linalg.inv(R + B.T @ P @ B) @ (B.T @ P @ A)

# Closed loop simulation
x = np.array([[1.0], [0.0]])
xs = []
for k in range(50):
    u = -K @ x
    x = A @ x + B @ u
    xs.append(x.flatten())

📚 깊이 1 — Riccati 식의 의미: 동적 프로그래밍

DARE는 단순히 행렬 식이 아니라, Bellman의 최적성 원리를 한 줄로 표현한 것입니다.

$$V^*(x_k) = \min_{u_k}\Big[\underbrace{x_k^TQx_k + u_k^TRu_k}_{\text{현재 비용}} + \underbrace{V^*(x_{k+1})}_{\text{미래 최적 비용}}\Big]$$

$V^*(x) = x^TPx$로 가정하고 양변에 대입한 후 $u$로 미분=0을 풀면 정확히 DARE가 나옵니다. 즉 DARE = Bellman 방정식의 닫힌 형태.

왜 $V^* = x^TPx$ 형태로 가정?

이차 비용 + 선형 시스템 → 최적 가치 함수도 이차형. 이 가정이 맞다는 것은 Bellman 방정식에 대입해서 self-consistent임을 보이면 증명.

📚 깊이 2 — Lyapunov 안정성 증명 단계별

$V(x) = x^TPx$가 닫힌 루프 시스템 $x_{k+1} = (A-BK)x_k$의 Lyapunov 함수임을 보입니다.

  1. 양정값(positive definite): $P > 0$이므로 $V(x) > 0\ \forall x \ne 0$, $V(0) = 0$.
  2. 감소(decrescent): 한 스텝 변화량 $\Delta V = V(x_{k+1}) - V(x_k) = -x^TQx - u^TRu \le 0$.
  3. 가관측 가정: $(A, Q^{1/2})$가 가관측이면 $\Delta V = 0 \Leftrightarrow x = 0$.

3가지 조건 → 점근 안정성. LQR이 자동 안정성을 갖는 이유의 정확한 증명.

📚 깊이 3 — 게인 마진 1/2~∞ 유도

LQR의 유명한 강건성 결과. 입력 채널에 게인 $\alpha$가 곱해져도 ($\alpha I$의 다중 입력 perturbation) 폐루프가 안정한 $\alpha$ 범위가 $[1/2, \infty)$.

증명 핵심: Return Difference equation $|I + L(z)|^2 \ge 1$ (단변수 경우) — 즉 LQR 폐루프의 sensitivity 함수는 항상 $\le 1$. 이 성질이 게인/위상 마진을 보장.

주의: 이 성질은 LQR (full state feedback)에 한해 성립. LQG (관측기 + LQR)에서는 일반적으로 성립 안 함 (Doyle 1978).

📚 깊이 4 — 무한 호라이즌의 의미

유한 호라이즌 $J_N = \sum_{k=0}^{N-1}(\cdot) + x_N^T P_f x_N$에서 $N \to \infty$이면 $P_k \to P_\infty$로 수렴. 이 $P_\infty$가 DARE의 안정 해.

실용적 의미: 한 번 DARE 풀이 = 임의 시간 시작점의 최적 비용 결정. 시간 가변 게인이 아닌 상수 게인으로 충분.

📚 깊이 5 — Q, R 튜닝 — Bryson 규칙 자세히

Bryson 규칙(1975): 각 변수의 허용 최대 편차 역수의 제곱.

$$Q_{ii} = \frac{1}{(x_i^{\max})^2},\quad R_{jj} = \frac{1}{(u_j^{\max})^2}$$

이러면 비용 함수의 각 항이 정규화되어 같은 스케일. 그 후 시뮬레이션으로 미세 조정.

예시 — 도립진자

  • 카트 위치 허용 ±1 m → $Q_{11} = 1$.
  • 카트 속도 허용 ±2 m/s → $Q_{22} = 0.25$.
  • 진자 각도 허용 ±0.1 rad → $Q_{33} = 100$.
  • 진자 각속도 허용 ±0.5 rad/s → $Q_{44} = 4$.
  • 입력 힘 허용 ±10 N → $R = 0.01$.

진자 각도에 큰 가중치 → 균형 우선.

📚 깊이 6 — Cross 항 $S$ 있는 비용

일반 비용:

$$J = \sum_{k=0}^\infty \begin{pmatrix}x_k \\ u_k\end{pmatrix}^T \begin{pmatrix} Q & S \\ S^T & R \end{pmatrix} \begin{pmatrix}x_k \\ u_k\end{pmatrix}$$

유도 단계에서 $S \ne 0$이면 게인 식이 약간 바뀜:

$$K = (R + B^TPB)^{-1}(B^TPA + S^T)$$

출력 정규화나 시간지연 처리 시 $S$가 자연스럽게 등장.

📚 깊이 7 — 출력 LQR (Output LQR)

전체 상태가 아닌 출력 $y = Cx$에 대한 비용:

$$J = \sum_{k=0}^\infty (y_k^T \tilde Q y_k + u_k^TRu_k) = \sum_{k=0}^\infty (x_k^T C^T \tilde Q C x_k + u_k^TRu_k)$$

즉 $Q = C^T \tilde Q C$. 여전히 표준 DARE 풀이로 환원. 단, $Q$가 양반정값일 수 있어 가검출 조건 점검 필요.

📚 깊이 8 — Tracking LQR

참조 신호 $x_{ref}$를 추종:

$$\tilde x_k = x_k - x_{ref},\ \tilde u_k = u_k - u_{ref}$$

$x_{ref}$가 평형점이면 ($Ax_{ref} + Bu_{ref} = x_{ref}$ 만족) $\tilde x$ 동역학 = 표준 LQR. $x_{ref}$가 시간 가변이면 시간 가변 LQR (피드포워드 추가).

📚 깊이 9 — 적분 LQR (Integral Action)

정상상태 오차를 0으로 만들려면 적분 상태 추가:

$$\tilde x = \begin{pmatrix} x \\ \int e\,dt \end{pmatrix}$$

이 증강된 상태에 LQR 적용 → 적분 액션을 가진 LQR. PID의 I 항과 같은 효과.

📚 깊이 10 — LQR과 MPC의 관계

  • 제약 비활성 영역에서는 MPC = LQR (같은 닫힌 해).
  • 제약 활성 영역에서는 MPC = LQR + 제약 보정.
  • MPC의 종단 비용 $P_f = $ LQR DARE 해로 두면 안정성 보장.
  • 종단 제약 $x_N \in \mathcal X_f$ ($\mathcal X_f$ = LQR 안정 영역)이 표준 트릭.

📚 깊이 11 — 단계별 풀이 — 1차원 LQR 손계산

$x_{k+1} = ax_k + bu_k$, $Q = q, R = r$ (모두 스칼라). DARE:

$$p = a^2 p - \frac{a^2 b^2 p^2}{r + b^2 p} + q$$

분모 곱: $p(r + b^2p) = a^2p(r + b^2p) - a^2b^2p^2 + q(r + b^2p)$

전개: $rp + b^2p^2 = a^2pr + a^2b^2p^2 - a^2b^2p^2 + qr + qb^2p$

정리: $b^2 p^2 + (r - a^2 r - q b^2) p - qr = 0$

이차방정식. 양의 해가 안정 해. 게인 $K = a b p / (r + b^2 p)$.

수치 예: $a = 1, b = 1, q = 1, r = 1$

$p^2 + (1 - 1 - 1)p - 1 = 0$ → $p^2 - p - 1 = 0$ → $p = (1+\sqrt{5})/2 \approx 1.618$ (황금비!).

$K = 1.618/(1+1.618) \approx 0.618$. 닫힌 루프 $a - bK = 0.382$, $|0.382| < 1$ → 안정.

📚 깊이 12 — LQG와 분리 정리 (Separation Principle)

실제 시스템은 $x$를 직접 못 봄. 칼만 필터로 추정 $\hat x$ 후 $u = -K\hat x$ 사용. 분리 정리:

  • LQR 게인 $K$를 $A, B, Q, R$로 따로 결계산.
  • 칼만 게인 $L$을 $A, C$, 노이즈 공분산으로 따로 계산.
  • 두 게인이 독립적으로 결정되어도 폐루프 안정성 보장.

이 분리 덕분에 LQG 설계가 모듈화 가능.

📚 깊이 13 — Doyle's Counterexample (1978)

"LQG가 안 좋은 강건성을 가질 수 있다"의 유명한 예. LQR과 칼만 필터 각각이 좋은 마진을 가져도, 결합한 LQG의 게인 마진이 임의로 작아질 수 있음. 해결: LTR (Loop Transfer Recovery).

실무 시사점: 항상 LQG 폐루프의 강건성을 별도 분석.

📚 깊이 15 — DARE 풀이 알고리즘 비교

  • Schur 분해: 표준 직접 방법. scipy 기본. $O(n^3)$.
  • Newton-Kleinman: 반복. 초기값 필요. 큰 문제에 효율.
  • Doubling 알고리즘: 스파스 큰 문제에 효율.
  • Hamiltonian 행렬 고유값 분해: 이론적으로 가장 깨끗한 접근.

📚 깊이 16 — 응용 사례 1 — 도립진자 균형

4상태 도립진자에서 LQR로 카트 위치 + 진자 균형 동시 안정화.

import numpy as np
from scipy.linalg import solve_discrete_are

# 카트(M=1kg) + 진자(m=0.1kg, L=0.5m) 선형화 + 이산화
A = np.array([
    [1, 0.02, 0, 0],
    [0, 1, -0.196, 0],
    [0, 0, 1, 0.02],
    [0, 0, 0.392, 1]
])
B = np.array([[0], [0.02], [0], [-0.04]])
Q = np.diag([1, 1, 100, 1])  # 진자 각도에 큰 비중
R = np.array([[0.1]])

P = solve_discrete_are(A, B, Q, R)
K = np.linalg.inv(R + B.T @ P @ B) @ B.T @ P @ A
print("LQR gain K =", K)

📚 깊이 17 — 응용 사례 2 — DC 모터 위치 제어

2상태 (위치 $\theta$, 속도 $\dot\theta$). $A = [[1, T_s], [0, 1-T_s/\tau]]$, $B = [0, T_s K_m/\tau]^T$.

$Q = \text{diag}(100, 1), R = 0.1$ → 빠른 위치 정착, 부드러운 입력. 산업용 서보 표준.
