5. 왜 배워야 할까? — 모든 공학·과학의 공통 언어

선형대수학 = "여러 개의 숫자를 한꺼번에 다루는 법"

자연·공학에서 다루는 거의 모든 양은 하나의 숫자가 아니라 여러 개의 숫자 묶음입니다.

  • 위치 → $(x, y, z)$ 세 숫자
  • 힘 → 방향과 크기를 가진 화살표
  • 사진 한 장 → 수백만 개 픽셀의 모음
  • 엑셀 표 → 행과 열로 정리된 숫자 배열

이걸 효율적으로 다루려면 새로운 언어가 필요한데, 그 언어가 바로 벡터(vector) 와 행렬(matrix) 입니다. 미적분학이 "변하는 양"의 언어라면, 선형대수학은 "여러 양을 묶어 다루는" 언어 입니다.

한 줄 요약 — 미적분학 + 선형대수학 = 거의 모든 공학·과학·AI의 두 기둥.

6. 왜 배워야 할까? — 일상 속의 선형대수

🎮 컴퓨터 그래픽스 · 게임

게임 캐릭터를 회전·이동·확대 하는 모든 동작 = 행렬 곱셈. 3D 모델을 화면에 그리는 과정 = 벡터·행렬 연산의 연속.

📷 이미지 · 사진

흑백 사진 한 장 = 픽셀 밝기 값으로 채워진 거대한 행렬. 사진 회전·확대·필터링 = 행렬 변환으로 표현.

📍 GPS · 내비게이션

내 위치 = $(x, y)$ 좌표 벡터. 두 지점 사이 거리 = 벡터의 길이(노름) 계산.

📊 데이터 · 통계

엑셀 표 = 행렬. 평균·분산·상관관계 모두 벡터 연산으로 표현됨.

7. 왜 배워야 할까? — 공학 전공에서

🤖 기계 · 로봇 — 운동의 표현

물체의 위치·속도·힘 은 모두 벡터. 로봇 팔이 어떻게 움직이는지 계산하려면 회전 행렬이 필요함.

⚡ 전기·전자 — 회로 해석

여러 개의 전압·전류 방정식 = 연립방정식 → 행렬로 정리해 한 번에 풂.

🧠 데이터 · AI

머신러닝의 거의 모든 알고리즘이 벡터·행렬 연산으로 작동. 딥러닝 모델 한 층 = 행렬 곱 + 비선형 함수.

이 수업은 후속 전공과목(동역학, 자동제어, 신호처리, 머신러닝)을 듣기 위한 필수 도구상자입니다.

8. 이 수업의 목표 — 직관 + 계산 + 활용

  1. 직관 (Intuition) — "벡터·행렬이 도대체 무엇인지" 그림으로 이해. 공식 암기 전에 기하학적 의미를 먼저.
  2. 계산 (Computation) — 손으로 가우스 소거, 행렬식, 고윳값을 정확히 계산할 수 있어야 함. 시험은 이 부분에서 출제.
  3. 활용 (Application) — 실제 공학 문제를 벡터·행렬로 모델링하는 감각. 후속 과목에서 "어, 이거 선형대수에서 본 건데?" 하는 순간이 오도록.
"공식만 외워서 계산하는 학생"이 아니라, "왜 이렇게 푸는지 그림으로 설명할 수 있는 학생"이 되는 것이 목표입니다.

9. 강의 방식 / 권장 도구

시간 구성 — 2시간 20~30분 수업

  • 1/3 : 개념 강의 (직관 + 정의)
  • 1/3 : 예제 풀이 (손계산)
  • 1/3 : 시각화 / 간단한 코드 실습 (노트북·태블릿 지참 권장)

실습 도구

  • 3Blue1Brown 영상 — 핵심 개념을 그림으로 다시 보기.
  • GeoGebra (geogebra.org) — 벡터·행렬 변환을 마우스로 드래그.
  • Python (NumPy) — 큰 행렬 계산은 컴퓨터에게 맡기기 (선택).
시험은 손계산 위주로 출제됩니다. 도구는 직관을 키우는 보조 수단일 뿐, 손으로 푸는 연습을 반드시 병행하세요.

10. 선수 지식 / 학습 팁

선수 지식

  • 고등학교 수학 수준이면 충분 (벡터의 기초, 연립방정식).
  • 미적분학 동시 수강 권장 (직접 의존하진 않음).

학습 팁

  • 그림을 그려라. 2D·3D에서 벡터를 직접 그려보면 추상적인 정의가 살아난다.
  • 손으로 계산하라. 가우스 소거, 행렬식은 직접 풀어보지 않으면 절대 익숙해지지 않는다.
  • 누적 과목이다. 1~3주차 벡터를 놓치면 9주차 이후가 매우 힘들어짐. 밀리지 말 것.
선형대수학은 이해 → 계산 → 활용 순서로 쌓이는 과목입니다. 한 단계씩 차근차근.
