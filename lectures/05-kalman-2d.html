Lecture 5. 칼만 필터 (2D)

로봇이 평면 위에서 움직일 때 위치를 추정해봅시다.

목차

  1. 2D 상태 벡터
  2. 공분산 타원
  3. 오도메트리 + GPS 융합
  4. EKF / UKF / 파티클 필터

1. 2D 상태 벡터

2D 평면에서 움직이는 로봇의 상태는 보통 위치와 속도로 4차원입니다.

$$ \mathbf{x} = \begin{bmatrix} x \\ y \\ \dot{x} \\ \dot{y} \end{bmatrix}, \quad A_d = \begin{bmatrix} 1 & 0 & \Delta t & 0 \\ 0 & 1 & 0 & \Delta t \\ 0 & 0 & 1 & 0 \\ 0 & 0 & 0 & 1 \end{bmatrix} $$

1D 칼만 필터의 모든 식이 그대로 행렬 버전으로 확장됩니다. 차이는 단지 스칼라 → 행렬뿐입니다.

2. 공분산 타원 (Covariance Ellipse)

2D에서 불확실성은 한 숫자가 아니라 2×2 행렬 $P$로 표현됩니다. 이 행렬을 시각화하면 타원이 됩니다. 타원이 클수록 모르겠다는 뜻입니다.

  • 예측 단계: 타원이 커집니다 (시간이 지나면 불확실해짐).
  • 갱신 단계: 측정이 들어오면 타원이 줄어듭니다.

🎮 인터랙티브: 2D 로봇 측위

로봇(녹색)이 원을 그리며 움직이고, GPS(빨강)는 노이즈가 큽니다. 칼만 필터(파랑)와 공분산 타원을 관찰하세요.

3. 비선형 시스템: EKF / UKF

기본 칼만 필터는 시스템이 선형일 때만 정확합니다. 자동차의 자전거 모델처럼 $\sin, \cos$가 들어가면 선형이 아닙니다.

  • EKF (Extended KF) — 매 스텝마다 야코비안 행렬로 1차 근사 선형화. 가장 흔히 씁니다.
  • UKF (Unscented KF) — 평균 주위에 "시그마 포인트"들을 흩뿌려 비선형 함수에 통과시킨 뒤 평균·공분산을 다시 계산. 미분이 필요 없어서 까다로운 시스템에 강합니다.
  • 파티클 필터 — 분포 자체를 수많은 점(파티클)으로 표현. 분포가 가우시안이 아니어도 OK.

4. 코드 예제 (Python)

import numpy as np
dt = 0.1
A = np.array([[1,0,dt,0],[0,1,0,dt],[0,0,1,0],[0,0,0,1]])
H = np.array([[1,0,0,0],[0,1,0,0]])    # GPS measures (x,y)
Q = 0.01*np.eye(4)
R = 4.0*np.eye(2)

x = np.zeros(4); P = np.eye(4)*10
def step(z):
    global x,P
    x = A @ x
    P = A @ P @ A.T + Q
    S = H @ P @ H.T + R
    K = P @ H.T @ np.linalg.inv(S)
    x = x + K @ (z - H @ x)
    P = (np.eye(4) - K @ H) @ P
    return x
