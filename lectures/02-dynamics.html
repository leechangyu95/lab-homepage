1. 운동학 vs 동역학

운동학(Kinematics)은 "어떻게 움직이는가"의 기하학적 묘사입니다. 힘은 따지지 않습니다.
동역학(Dynamics)은 "왜 그렇게 움직이는가"입니다. 힘과 질량을 다룹니다.

고등학교에서 배운 가장 친숙한 동역학 식이 바로 뉴턴의 제2법칙입니다.

$$ F = m a, \quad \tau = I \alpha $$

병진운동(직선)과 회전운동(돌기)에 똑같이 적용됩니다.

2. 좌표계

로봇은 항상 두 좌표계를 동시에 씁니다.

  • 관성 좌표계 (Inertial frame) — 지도 위 고정된 좌표. 움직이지 않는 기준.
  • 본체 좌표계 (Body frame) — 로봇과 함께 움직이는 좌표. 차의 "앞" 방향.

속도는 본체 좌표에서 측정하기 쉽지만(속도계), 위치는 관성 좌표에서 알아야 합니다(지도 위 점). 둘 사이를 변환하는 회전행렬이 필요합니다.

3. 자전거 모델 (Bicycle Model)

실제 자동차는 바퀴 4개지만, 좌우 대칭이므로 앞바퀴 1개·뒷바퀴 1개로 단순화한 자전거 모델이 자율주행 표준입니다.

$$ \dot{x} = v\cos\theta, \quad \dot{y} = v\sin\theta, \quad \dot{\theta} = \frac{v}{L}\tan\delta $$

여기서 $v$는 속도, $\theta$는 차의 방향(heading), $\delta$는 조향각, $L$은 앞·뒷바퀴 사이 거리(축거)입니다. 이 세 식만 있으면 핸들과 페달로 차를 움직일 수 있습니다.

4. 수치 적분: 오일러 vs 룽게-쿠타

위 미분방정식을 컴퓨터로 풀려면 작은 시간 간격 $\Delta t$로 나누어 한 스텝씩 전진해야 합니다.

오일러 (가장 단순)

$$ x_{k+1} = x_k + f(x_k, u_k) \,\Delta t $$

한 스텝의 변화량을 그대로 더합니다. 빠르지만 부정확합니다.

룽게-쿠타 4차 (RK4, 표준)

$$ x_{k+1} = x_k + \frac{\Delta t}{6}(k_1 + 2k_2 + 2k_3 + k_4) $$

한 스텝 안에서 4번 기울기를 구해 평균합니다. 4배 느리지만 오차는 수백 배 작습니다.

코드 예제 (Python)

import numpy as np
import matplotlib.pyplot as plt

def f(x, u, L=2.5):
    v, delta = u
    return np.array([v*np.cos(x[2]), v*np.sin(x[2]), v/L*np.tan(delta)])

def euler(x, u, dt): return x + f(x,u)*dt

def rk4(x, u, dt):
    k1 = f(x, u)
    k2 = f(x + dt/2*k1, u)
    k3 = f(x + dt/2*k2, u)
    k4 = f(x + dt*k3, u)
    return x + dt/6*(k1 + 2*k2 + 2*k3 + k4)

x = np.array([0., 0., 0.])
traj = [x.copy()]
for _ in range(200):
    x = rk4(x, (5.0, 0.1), 0.05)   # v=5 m/s, δ=0.1 rad
    traj.append(x.copy())
traj = np.array(traj)
plt.plot(traj[:,0], traj[:,1]); plt.axis('equal'); plt.show()
핵심 직감 — $\Delta t$가 작을수록 두 방법 모두 정확해집니다. 하지만 RK4는 큰 $\Delta t$에서도 잘 버팁니다. 실시간 시스템에서 계산량을 아낄 때 매우 중요합니다.

5. 차륜형 로봇의 세 가지 모델

같은 "바퀴 달린 로봇"이라도 구동 방식에 따라 운동학이 다릅니다. PythonRobotics의 차량 모델을 참고해 세 가지를 비교합니다.

모델예시운동학
유니사이클 (Unicycle)외바퀴, 단순 이동 로봇$\dot x = v\cos\theta,\;\dot y = v\sin\theta,\;\dot\theta = \omega$
차동 구동 (Differential)룸바, TurtleBot$v=(v_R+v_L)/2,\;\omega=(v_R-v_L)/L$
자전거 (Bicycle)자동차, 자전거$\dot\theta = (v/L)\tan\delta$

🎮 인터랙티브: 차동 구동 로봇

좌·우 바퀴 속도를 따로 조절합니다. 두 속도가 같으면 직진, 다르면 회전. 한쪽이 음수면 제자리 회전이 가능합니다 (자동차는 못 함!).

🎮 인터랙티브: 오일러 vs RK4 정확도

회전하는 단진동 시스템 $\ddot x = -x$(원형 궤도)를 두 방법으로 풉니다. 진짜 답은 단위원이지만 큰 $\Delta t$에서는 오일러가 점점 바깥으로 새 나가고 RK4는 거의 정확합니다.

$\Delta t = 0.4$로 두면 오일러는 한 바퀴도 못 돌아 폭발합니다. RK4는 $\Delta t$를 키워도 단위원 근처에 머무릅니다.
