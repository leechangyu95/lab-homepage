시뮬레이션 갤러리

연구실 알고리즘의 시뮬레이션 결과와 실험 장면을 모아둔 페이지입니다. 아래 예시는 인라인 SVG로 그린 placeholder이며, 실제 matplotlib/CARLA/Gazebo 결과 이미지·GIF로 교체할 수 있습니다.

이미지 추가 방법: 결과 이미지(PNG/GIF/MP4)를 assets/img/ 폴더에 넣고 아래 <img src="assets/img/파일명"> 형태로 교체하면 됩니다. GitHub Pages는 그대로 호스팅됩니다 (개별 파일 100MB 제한, 저장소 1GB 권장).

실제 결과 추가 가이드

matplotlib에서 이미지 저장

fig.savefig("assets/img/lqr_response.png", dpi=150, bbox_inches="tight")

matplotlib animation을 GIF로

from matplotlib.animation import FuncAnimation, PillowWriter
ani = FuncAnimation(fig, update, frames=200, interval=50)
ani.save("assets/img/mpc_demo.gif", writer=PillowWriter(fps=20))

HTML에 삽입

<figure>
  <img src="assets/img/mpc_demo.gif" alt="MPC demo">
  <figcaption>Tube-MPC 시뮬레이션</figcaption>
</figure>
CARLA / Gazebo / 실선 실험 영상은 YouTube에 업로드한 뒤 <iframe> 임베드 코드를 붙이는 방식이 가장 가볍습니다.
