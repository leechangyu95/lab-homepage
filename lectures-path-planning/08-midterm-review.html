2. 꼭 외울 7개 수식

  1. Kinematic Bicycle 요레이트: $\dot\psi = \dfrac{v}{L}\tan\delta$
  2. 회전반경: $R = \dfrac{L}{\tan\delta}$
  3. 선형 타이어 횡력: $F_y = C_\alpha\,\alpha$
  4. Understeer Gradient: $K_{us} = \dfrac{l_r m g}{L\,C_{\alpha f}} - \dfrac{l_f m g}{L\,C_{\alpha r}}$
  5. 오일러 적분: $x_{k+1} = x_k + \dot x_k\,\Delta t$
  6. Frenet ↔ World: $\mathbf{p}(s,d) = \mathbf{r}(s) + d\,\mathbf{n}(s)$
  7. A* 비용: $f(n) = g(n) + h(n)$

이 7개로 이론 파트 70%는 해결됩니다. 시험장 직전에는 1·2·5·7번을 한 번 더 음미하세요.

4. 인터랙티브: A* 5×5 손풀이 연습

🎮 A* 손풀이 — 단계별 확장 따라가기

5×5 격자에서 S(시작)→G(목표) 경로를 A*가 어떻게 찾는지 한 단계씩 진행해 보세요. 파랑 = 닫힌 집합, 주황 = 열린 집합, 빨강 = 벽.

휴리스틱을 0으로 바꾸면 Dijkstra와 같은 동작을 합니다 — 닫힌 집합이 훨씬 더 많이 펼쳐지는 걸 볼 수 있어요.

5. 주차별 핵심 한 장 정리

W1 — SAE 등급과 파이프라인

  • Lv0(전수동) — Lv1(보조 1축) — Lv2(보조 2축) — Lv3(조건부, 책임 전환) — Lv4(영역 무인) — Lv5(완전).
  • Perception → Planning → Control → Actuator. 본 강의는 가운데 두 개.

W2 — 좌표계 & Bicycle

$\dot x = v\cos\psi,\;\dot y = v\sin\psi,\;\dot\psi = \dfrac{v}{L}\tan\delta$. 후륜 기준이 Planning 표준. 변환: $T^W_S = T^W_B\,T^B_S$.

W3 — 동역학

$F_y = C_\alpha\alpha$ (선형 영역, $|\alpha|\le 5°$). $K_{us} > 0$ → Understeer(안전), < 0 → Oversteer(위험). $A$ 행렬에 $v_x$가 들어가 속도별 동역학이 다름.

W4 — 적분

Euler $O(\Delta t)$, RK4 $O(\Delta t^4)$. 자율주행 실무 $\Delta t = 10\sim 50$ ms. 큰 $\Delta t$는 발산 위험.

W5 — 지도와 Frenet

Occupancy Grid + 거리 변환 + 인플레이션 = Costmap. Frenet $(s,d)$: $s$ 진행거리, $d$ 횡오차. $d>0$이면 경로 왼쪽.

W6 — RRT

5단계 = S→N→S→C→A. Goal Bias 5–10%. RRT*는 Choose Parent + Rewire로 점근 최적.

W7 — A*

$f = g + h$. Admissible($h \le h^*$)이면 최적 보장. Manhattan(4-방향), Euclidean/Octile(8-방향). 대각 비용은 1이 아닌 $\sqrt 2$.

7. 예시 문제 (해답 토글)

Q1. SAE Level 3의 설명으로 가장 올바른 것은?
① 운전자가 모든 상황을 감시한다 / ② 차량이 조건부로 운전 주체가 되고 요청 시 운전자가 인수한다 / ③ 특정 영역 무인 / ④ 모든 환경 완전 주행.
정답: ②. Lv3은 시스템이 운전 주체가 되는 첫 등급으로, 책임 주체 전환이 핵심 경계입니다.
Q2. 휠베이스 $L = 2.8$ m, 조향각 $\delta = 20°$일 때 순간 회전반경 $R$은?
$R = L/\tan\delta = 2.8/\tan 20° \approx 2.8/0.364 \approx 7.69$ m
Q3. Occupancy Grid 셀 한 변 0.2 m, 격자 50×50의 실제 면적은?
한 변 = 50 × 0.2 = 10 m. 면적 = 100 m².
Q4. Goal Bias = 100%로 설정하면 RRT는 어떻게 동작하는가?
매번 목표점만 샘플로 사용 → 목표 직선 방향으로만 트리가 자람. 장애물이 있으면 우회를 못 해 실패합니다.
Q5. A*에서 휴리스틱 $h(n) = 0$이라면 어떤 알고리즘과 같은가?
Dijkstra. $f = g$가 되어 균일한 확장이 됩니다.
Q6. $v = 5$ m/s, $\delta = 15°$, 초기 $(0,0,0)$, $L = 2.8$, $\Delta t = 0.1$, 1스텝 후 $(x,y,\psi)$는?
$x = 0 + 5\cos 0\cdot 0.1 = 0.5$, $y = 0$, $\psi = 0 + (5/2.8)\tan 15°\cdot 0.1 \approx 0.0479$ rad ≈ 2.74°.
