2. 꼭 외울 7개 수식

  1. Kinematic Bicycle 요레이트: $\dot\psi = \dfrac{v}{L}\tan\delta$
  2. 회전반경: $R = \dfrac{L}{\tan\delta}$
  3. 선형 타이어 횡력: $F_y = C_\alpha\,\alpha$
  4. Understeer Gradient: $K_{us} = \dfrac{l_r m g}{L\,C_{\alpha f}} - \dfrac{l_f m g}{L\,C_{\alpha r}}$
  5. 오일러 적분: $x_{k+1} = x_k + \dot x_k\,\Delta t$
  6. Frenet ↔ World: $\mathbf{p}(s,d) = \mathbf{r}(s) + d\,\mathbf{n}(s)$
  7. A* 비용: $f(n) = g(n) + h(n)$

이 7개로 이론 파트 70%는 해결됩니다. 시험장 직전에는 1·2·5·7번을 한 번 더 음미하세요.

4. 인터랙티브: A* 5×5 손풀이 연습

🎮 A* 손풀이 — 단계별 확장 따라가기

5×5 격자에서 S(시작)→G(목표) 경로를 A*가 어떻게 찾는지 한 단계씩 진행해 보세요. 파랑 = 닫힌 집합, 주황 = 열린 집합, 빨강 = 벽.

휴리스틱을 0으로 바꾸면 Dijkstra와 같은 동작을 합니다 — 닫힌 집합이 훨씬 더 많이 펼쳐지는 걸 볼 수 있어요.

5. 주차별 핵심 한 장 정리

W1 — SAE 등급과 파이프라인

  • Lv0(전수동) — Lv1(보조 1축) — Lv2(보조 2축) — Lv3(조건부, 책임 전환) — Lv4(영역 무인) — Lv5(완전).
  • Perception → Planning → Control → Actuator. 본 강의는 가운데 두 개.

W2 — 좌표계 & Bicycle

$\dot x = v\cos\psi,\;\dot y = v\sin\psi,\;\dot\psi = \dfrac{v}{L}\tan\delta$. 후륜 기준이 Planning 표준. 변환: $T^W_S = T^W_B\,T^B_S$.

W3 — 동역학

$F_y = C_\alpha\alpha$ (선형 영역, $|\alpha|\le 5°$). $K_{us} > 0$ → Understeer(안전), < 0 → Oversteer(위험). $A$ 행렬에 $v_x$가 들어가 속도별 동역학이 다름.

W4 — 적분

Euler $O(\Delta t)$, RK4 $O(\Delta t^4)$. 자율주행 실무 $\Delta t = 10\sim 50$ ms. 큰 $\Delta t$는 발산 위험.

W5 — 지도와 Frenet

Occupancy Grid + 거리 변환 + 인플레이션 = Costmap. Frenet $(s,d)$: $s$ 진행거리, $d$ 횡오차. $d>0$이면 경로 왼쪽.

W6 — RRT

5단계 = S→N→S→C→A. Goal Bias 5–10%. RRT*는 Choose Parent + Rewire로 점근 최적.

W7 — A*

$f = g + h$. Admissible($h \le h^*$)이면 최적 보장. Manhattan(4-방향), Euclidean/Octile(8-방향). 대각 비용은 1이 아닌 $\sqrt 2$.

7. 예시 문제 (해답 토글)

Q1. SAE Level 3의 설명으로 가장 올바른 것은?
① 운전자가 모든 상황을 감시한다 / ② 차량이 조건부로 운전 주체가 되고 요청 시 운전자가 인수한다 / ③ 특정 영역 무인 / ④ 모든 환경 완전 주행.
정답: ②. Lv3은 시스템이 운전 주체가 되는 첫 등급으로, 책임 주체 전환이 핵심 경계입니다.
Q2. 휠베이스 $L = 2.8$ m, 조향각 $\delta = 20°$일 때 순간 회전반경 $R$은?
$R = L/\tan\delta = 2.8/\tan 20° \approx 2.8/0.364 \approx 7.69$ m
Q3. Occupancy Grid 셀 한 변 0.2 m, 격자 50×50의 실제 면적은?
한 변 = 50 × 0.2 = 10 m. 면적 = 100 m².
Q4. Goal Bias = 100%로 설정하면 RRT는 어떻게 동작하는가?
매번 목표점만 샘플로 사용 → 목표 직선 방향으로만 트리가 자람. 장애물이 있으면 우회를 못 해 실패합니다.
Q5. A*에서 휴리스틱 $h(n) = 0$이라면 어떤 알고리즘과 같은가?
Dijkstra. $f = g$가 되어 균일한 확장이 됩니다.
Q6. $v = 5$ m/s, $\delta = 15°$, 초기 $(0,0,0)$, $L = 2.8$, $\Delta t = 0.1$, 1스텝 후 $(x,y,\psi)$는?
$x = 0 + 5\cos 0\cdot 0.1 = 0.5$, $y = 0$, $\psi = 0 + (5/2.8)\tan 15°\cdot 0.1 \approx 0.0479$ rad ≈ 2.74°.

9. 추가 손풀이 — 시험 직전 도전

Q7. 차체 좌표계에서 $(2, 1)$인 점이 차량 위치 $(5, 3)$, 헤딩 $90°$일 때 World 좌표는?
회전 90° 적용 후 평행이동:
$R(90°)\begin{bmatrix}2\\1\end{bmatrix} = \begin{bmatrix}-1\\2\end{bmatrix}$, 더하기 $(5,3)$ → World 좌표 $(4, 5)$.
Q8. RRT 알고리즘 5단계를 순서대로 나열하시오.
① Sample(랜덤 점) → ② Nearest(트리에서 가장 가까운 노드) → ③ Steer(작은 걸음) → ④ Collision Check(장애물 통과 여부) → ⑤ Add(트리에 추가). 약자로 "S-N-S-C-A".
Q9. Frenet 좌표에서 $(s, d) = (10, -1.5)$의 의미는?
시작점에서 경로 따라 10 m 진행한 후, 경로 오른쪽으로 1.5 m 떨어진 위치. 차선 폭 3.5 m 기준으로는 차선 안에 있음.
Q10. A*에서 8-방향 이동 시 Manhattan 휴리스틱 사용 시 문제는?
대각 이동 비용 ≈ 1.41인데 Manhattan은 (수평 + 수직 = 2)로 과대평가 → admissibility 위반 → 최적 경로 보장 X. 8-방향엔 Euclidean 또는 Octile heuristic 사용.
Q11. Understeer 차량 ($K_{us}>0$) 이 고속에서 보이는 거동은?
속도 ↑ 일수록 같은 곡선을 돌기 위해 조향을 더 많이 줘야 함. "차가 둔해지는" 느낌. 이 특성이 안전 설계로 양산차 표준.
Q12. Cubic Spline의 장점을 한 줄로?
이산 waypoint 들을 부드럽게 연결 (1·2계 도함수 연속) → 차량이 따라갈 수 있는 매끈한 경로. 곡률 계산도 가능.

10. 서술형 모범답안 골자

S1. "Kinematic vs Dynamic Bicycle 모델 비교"

핵심 4가지로 답변
  1. 가정 차이: Kinematic은 슬립 무시, Dynamic은 선형 타이어 모델 포함
  2. 상태 변수: K = 3개 ($x,y,\psi$), D = 5개 (+ $v_y, \dot\psi$)
  3. 유효 영역: K = 저속·완만 조향, D = 고속·급조향
  4. 용도: K = Planning, 저속 제어 / D = 고속 추종, 안정성 분석

S2. "Occupancy Grid → A* → Spline 파이프라인 설명"

3단계 흐름
  1. Grid: 환경을 격자로 분할, 0(빈 공간)/1(장애물)로 표시. Distance transform 으로 Costmap 생성.
  2. A*: 시작·목표 셀 사이 최단 경로. $f = g + h$, 8-방향 이웃 + Euclidean h. 출력은 격자 셀 시퀀스.
  3. Spline: 격자 셀들을 Cubic Spline으로 평활화 → 차량이 따라갈 수 있는 부드러운 곡선. 곡률 계산도 가능.

각 단계의 입출력 명시 + 그림 1개 = 만점.

S3. "RRT vs A* 어느 상황에 사용할까"

기준RRT 유리A* 유리
공간고차원·연속저차원·격자

11. 시험 직후 — 후반부 마음가짐

시험은 지금까지 배운 걸 점검하는 거울일 뿐, 끝이 아닙니다. 9주차부터의 후반부는 1~7주차 위에 쌓아올리는 구조라서:

  • 2주차 Bicycle 모델 → 10주차 Pure Pursuit/Stanley 의 입력
  • 3주차 Dynamic 모델 → 13주차 MPC 의 플랜트
  • 5주차 Frenet → 11주차 통합 시뮬의 좌표계
  • 7주차 A* → 11주차 통합 시뮬의 전역 경로

전반부에서 막혔던 부분은 후반부 실습을 통해 자연스럽게 더 깊이 이해됩니다. 시험에서 틀린 문제 = 후반부에 다시 만날 친구!

13. 자기 진단 — 어떤 주차를 더 봐야 할까?

아래 표에서 본인이 막혔던 항목에 체크해 보세요. 가장 많이 체크된 주차가 시험 D-1에 다시 봐야 할 주차입니다.

항목관련 주차막혔다면 다시 볼 곳
회전행렬 / 동차 변환을 자유롭게 못 쓴다W2W2 §3 회전행렬
$\dot\psi = (v/L)\tan\delta$ 의 직관이 흐릿하다W2W2 §6 Bicycle 모델
Understeer / Oversteer 부호가 헷갈린다W3W3 §3
오일러 $\Delta t$ 선택 기준을 모르겠다W4W4 §4 안정성
Frenet $(s, d)$ 의 $d$ 부호가 헷갈린다W5W5 §5
RRT 의 5단계 순서가 안 떠오른다W6W6 §3
A* 손풀이를 처음부터 못 하겠다W7W7 §4 + 본 페이지 §4 인터랙티브
Admissible heuristic 조건이 무엇인지 모른다W7W7 §3

3개 이상 체크되면 D-2 ~ D-1 에 해당 주차 페이지의 인터랙티브 1번씩 만져보고 핵심 수식 손으로 적기.

14. 후반부 학습 동기 — 지금 이걸 왜 외우나?

1~7주차 내용은 대부분 후반부 실습에서 다시 등장합니다. 시험 직후에 잊어도 OK, 후반부에서 자연스럽게 복습됩니다.

전반부 개념후반부에서 어떻게 쓰이는가
Bicycle 모델 (W2)Pure Pursuit/Stanley 의 조향 공식 유도 기반 (W10)
Dynamic 모델 (W3)MPC 의 플랜트 (W13), Rajamani 오차 동역학
Euler 적분 (W4)모든 시뮬레이터의 핵심 (W11~14)
Costmap (W5)A* 및 통합 시뮬에서 안전 경로 유도 (W7, W11)
Frenet (W5)차선 주행 표현, MPC 오차 좌표 (W11, W13)
RRT (W6)시험엔 나오지만 후반부엔 적게 사용. 단, "랜덤 탐색" 직관은 RL(W14)에서
A* (W7)통합 시뮬의 전역 경로 (W11), 연구 분야 표준

그래서 시험은 "후반부에 사용할 도구를 점검하는 자리" 라고 생각하면 됩니다. 통과 후엔 곧 도구로 쓰게 돼요.
