Week 12. 진동 입문

한 학기 동안 배운 모든 것이 만나는 곳. 동역학에서 가장 자주 마주치는 운동의 패턴.

1. 단순조화운동(SHM)

복원력이 변위에 비례할 때 — $F = -kx$ — 운동방정식은 다음과 같습니다.

$$ m\ddot x = -kx \quad\Rightarrow\quad \ddot x + \omega_n^2 x = 0, \quad \omega_n = \sqrt{k/m} $$

이 식의 해는 사인 함수입니다.

$$ x(t) = A\cos(\omega_n t + \phi) $$

$A$는 진폭(초기 조건이 정함), $\omega_n$은 고유진동수(시스템의 성질만으로 정해짐). 주기는 $T = 2\pi/\omega_n = 2\pi\sqrt{m/k}$.

진동의 보편성 — 안정한 평형점 근처라면 거의 모든 시스템이 SHM처럼 행동합니다(테일러 전개의 1차항이 복원력을 만들어주니까요). 그래서 이 식이 학부 시절 가장 자주 만나는 미분방정식입니다 — 진자, 스프링, RLC 회로(W7 기전공학), 분자 진동, 우주선 자세, 음파, 다리의 흔들림 모두.

🎮 데모 1: 질량-스프링 — k와 m이 주기를 어떻게 정하나

질량과 스프링 상수를 슬라이더로 바꿔보세요. 주기 $T = 2\pi\sqrt{m/k}$ — $m$이 4배가 되면 주기가 2배(느림), $k$가 4배가 되면 주기가 절반(빠름). 시간 그래프와 위상 평면이 함께 보여집니다.

2. 단진자 — 작은 각도에서만 SHM

길이 $L$의 단진자 운동방정식은 $\ddot\theta = -(g/L)\sin\theta$. 작은 $\theta$에서는 $\sin\theta \approx \theta$로 근사해 SHM이 되고 $\omega_n = \sqrt{g/L}$. 그러나 큰 진폭에서는 비선형이 강해져 주기가 진폭에 의존하기 시작합니다.

3. 감쇠 진동 — 마찰이 들어오면

운동에 비례하는 저항(점성 감쇠) $-c\dot x$가 추가되면 식이 다음과 같아집니다.

$$ m\ddot x + c\dot x + k x = 0 \quad\Rightarrow\quad \ddot x + 2\zeta\omega_n \dot x + \omega_n^2 x = 0 $$

$\zeta$는 감쇠비(damping ratio), $c/(2\sqrt{mk})$. 이 한 무차원 수가 응답의 모양을 결정합니다.

  • $\zeta < 1$ 부족 감쇠 — 진동하면서 진폭이 점점 줄어듦. 자동차 서스펜션, 종 소리.
  • $\zeta = 1$ 임계 감쇠 — 진동 없이 가장 빠르게 평형으로 복귀. 문 닫힘 장치의 이상적 튜닝.
  • $\zeta > 1$ 과 감쇠 — 천천히, 진동 없이 복귀. 점성이 큰 액체 속의 추.

🎮 데모 2: 감쇠 비교 — 같은 시스템, 다른 감쇠

같은 $\omega_n$을 가진 세 시스템에 같은 초기 조건. 감쇠비 $\zeta$만 다릅니다. 부족·임계·과 감쇠가 어떻게 다르게 평형으로 돌아가는지 시간 응답으로 봅니다. 자동제어 1·2차 시스템에서 그대로 나오는 그림.

4. 흔한 오개념과 한 학기 정리

  • "질량이 클수록 진자가 더 빨리 흔들린다" — 단진자에서 질량은 식에서 약분돼 사라집니다. 주기는 $L$과 $g$만으로 정해집니다.
  • "감쇠가 크면 평형까지 빨리 간다" — 일정 수준 이상으로 감쇠를 키우면 오히려 느려집니다(과 감쇠). 가장 빠른 건 임계 감쇠 $\zeta=1$.
  • SHM은 동역학을 한 학기 배운 사람이 보는 첫 "고전 명작" — 운동학(W2)에서 사인 함수를 만든 적분, F=ma(W6)에서 자유물체도, 에너지 보존(W7), 강체의 회전(W11)이 모두 한 식에서 만납니다.
다음 학기로 가는 다리 — 이 식 $\ddot x + 2\zeta\omega_n \dot x + \omega_n^2 x = 0$은 자동제어의 2차 시스템 단위계단 응답으로 다시 등장합니다. RLC 회로(기전공학기초 W7)도 정확히 같은 형태의 미분방정식이에요. 같은 식이 기계·전기·제어 어디서나 나온다는 사실이 공학의 매력입니다.
