Week 2. 이산 상태공간 모델

MPC는 매 샘플 시간마다 "내부 시뮬레이터"로 미래를 예측합니다. 그 시뮬레이터의 표준 형태가 이산시간 상태공간 모델이에요. 모델을 어떻게 얻고, 어떻게 검증하고, 무엇을 보장하는지 한 페이지에 정리합니다.

목차

  1. 왜 상태공간인가 — 전달함수와 비교
  2. 이산시간 모델의 형태 (A, B, C, D)
  3. 연속 → 이산 변환 (ZOH, Tustin)
  4. 가제어성·가관측성 — MPC가 동작하기 위한 조건
  5. 모델 식별 — 데이터에서 (A, B) 뽑기
  6. 외란·미스매치 — 적분 외란 모델

1. 왜 상태공간인가

한 줄 비유부터

전달함수 $G(s)$는 "입력→출력의 주파수 관계"를 한 식에 담는 SISO용 도구입니다. 스마트폰의 음향 EQ 같은 신호 처리에는 충분합니다. 그런데 자율차의 종방향·횡방향 동역학처럼 "입력 여러 개 → 출력 여러 개 + 내부 상태가 서로 연결"인 시스템에는 부족해요. 상태공간은 시스템의 내부 변수들이 서로 어떻게 영향을 주고받는지를 행렬로 펼쳐서 보여줍니다.

이산시간 표준형

MPC가 가장 자주 쓰는 형태:

$$ x_{k+1} = A x_k + B u_k + E w_k $$ $$ y_k = C x_k + D u_k + v_k $$
  • $x_k \in \mathbb{R}^n$ — 상태 (시스템의 "기억"). 위치, 속도, 온도, 농도 등.
  • $u_k \in \mathbb{R}^m$ — 제어 입력. 모터 전압, 밸브 개도, 토크 등.
  • $y_k \in \mathbb{R}^p$ — 측정 출력. 센서로 직접 잴 수 있는 양.
  • $w_k, v_k$ — 프로세스/측정 외란. 평균 0, 공분산 알려진 가정.
  • $A, B, C, D, E$ — 상수 행렬 (선형 시불변, LTI). 모델링 또는 식별로 결정.

$D = 0$인 경우가 많습니다 (출력에 입력이 직접 포함되지 않음). MPC에서는 일반적으로 출력 추적 $y \to r$이 목표지만, 수식 단순화를 위해 자주 $y = x$ (상태 직접 측정)로 두는 경우도 있습니다.

2. 연속시간 → 이산시간 변환

물리 모델은 보통 연속시간 형태로 나옵니다:

$$ \dot x(t) = A_c x(t) + B_c u(t) $$

이걸 샘플링 시간 $T_s$로 이산화합니다. 가장 정확하면서 흔한 방법은 Zero-Order Hold (ZOH) — "한 샘플 간격 동안 입력 $u$가 일정하다"는 가정:

$$ A = e^{A_c T_s},\quad B = \int_0^{T_s} e^{A_c \tau} d\tau \cdot B_c $$

다른 방법으로 Forward Euler($A = I + A_c T_s$, 단순하지만 큰 $T_s$에서 불안정), Tustin (Bilinear)($A = (I + \tfrac{T_s}{2}A_c)(I - \tfrac{T_s}{2}A_c)^{-1}$, 안정성·주파수 보존이 좋음)이 있습니다.

실전 팁 — Python에서는 scipy.signal.cont2discrete((Ac, Bc, Cc, Dc), Ts, method='zoh')로 한 줄. MATLAB은 c2d(sys, Ts, 'zoh'). ZOH는 거의 항상 옳고, Tustin은 주파수 응답을 정확히 보존하고 싶을 때 좋습니다.

예시 — 1차 RC 회로

연속시간: $\dot x = -\tfrac{1}{RC} x + \tfrac{1}{RC} u$. $RC = 1$, $T_s = 0.1$일 때 ZOH 결과는 $A \approx 0.905$, $B \approx 0.095$. 즉 이산 모델 $x_{k+1} = 0.905\, x_k + 0.095\, u_k$.

3. 가제어성 (Controllability)과 가관측성 (Observability)

MPC가 의미 있게 동작하려면 두 가지 구조적 성질이 필요합니다.

가제어성 — "어디든 갈 수 있는가"

임의의 초기 상태 $x_0$에서 임의의 목표 $x_f$로 유한 스텝 안에 도달 가능한가? 판정 행렬:

$$ \mathcal{C} = [B \quad AB \quad A^2 B \quad \cdots \quad A^{n-1} B] \in \mathbb{R}^{n \times nm} $$

$\text{rank}(\mathcal C) = n$이면 시스템은 가제어. 그렇지 않으면 어떤 모드는 입력으로 절대 제어할 수 없습니다 — MPC도 LQR도 그런 모드는 손댈 수 없어요.

가관측성 — "내부를 다 볼 수 있는가"

출력만 보고 모든 상태를 복원할 수 있는가? 판정 행렬:

$$ \mathcal{O} = \begin{bmatrix} C \\ CA \\ \vdots \\ CA^{n-1} \end{bmatrix} \in \mathbb{R}^{np \times n} $$

$\text{rank}(\mathcal O) = n$이면 가관측. MPC는 보통 상태 $x$를 직접 안다고 가정하지만, 실제로는 출력 $y$만 측정 가능하므로 관측기(observer)로 $\hat x$를 추정해 사용합니다 (보통 칼만 필터, L4 1D 칼만 참조).

약화된 조건 — 가제어성 대신 안정화 가능성(stabilizability)이면 충분합니다 ("불안정 모드는 모두 제어 가능"). 마찬가지로 검출 가능성(detectability) ("불안정 모드는 모두 관측 가능")이면 관측기 설계 가능.

4. 모델 식별 — 데이터에서 (A, B) 추정

물리 1원리로 $A, B$를 유도하기 어려운 시스템(생물·경제·복잡 기계)은 입출력 데이터로 직접 식별합니다.

최소제곱 ARX

$N$ 스텝의 입출력 $\{u_k, y_k\}$에서 다음을 최소화:

$$ \min_{A, B}\ \sum_{k=0}^{N-1} \|y_{k+1} - (A y_k + B u_k)\|^2 $$

이는 결국 단일 큰 최소제곱 문제 — 머신러닝 W3 회귀의 정규방정식과 동일한 구조입니다. SciPy의 numpy.linalg.lstsq 한 줄.

고급 기법

  • Subspace identification (N4SID, MOESP) — 출력 데이터의 SVD로 상태 차원과 행렬을 한 번에. 산업 표준.
  • Prediction Error Method (PEM) — 잡음 모델까지 함께 추정. MATLAB idtfest, n4sid.
  • Sparse identification (SINDy) — 비선형 시스템에서 sparse 표현 추정.

🎮 인터랙티브: 시스템 응답 시각화

2차 시스템 $x_{k+1} = A x_k + B u_k$의 행렬 원소를 슬라이더로 바꾸며 임펄스 응답이 어떻게 변하는지 봅니다. $A$의 고윳값 절댓값이 1보다 작으면 안정, 1을 넘으면 발산해요.

5. 외란과 모델 미스매치

실제 시스템은 식별된 모델과 정확히 일치하지 않습니다. MPC가 정상상태에서도 오차를 남기는 가장 흔한 원인은 일정 외란(constant disturbance)입니다. 해법은 적분 외란 모델(integrating disturbance model):

$$ x_{k+1} = A x_k + B u_k + B_d d_k,\quad d_{k+1} = d_k $$

외란 $d$를 상태에 추가해 함께 추정하면 MPC가 자동으로 보상하게 됩니다. 이는 PID의 적분기 ($I$ 항)와 같은 역할.

6. 핵심 정리

  • 이산시간 상태공간 $x_{k+1} = Ax_k + Bu_k$가 MPC의 표준 모델 형식.
  • 연속 → 이산은 ZOH가 거의 항상 정답 (scipy/MATLAB 한 줄).
  • 가제어성·가관측성 (또는 약화 버전)이 MPC가 동작하기 위한 구조적 조건.
  • $x$를 직접 못 보면 칼만 필터로 $\hat x$ 추정해 사용.
  • 정상상태 오차를 없애려면 적분 외란 모델로 $d$를 상태에 추가.

📚 보강 1 — 상태공간 모델을 직관으로

상태공간 표현은 시스템의 모든 정보를 "상태 벡터" $x$로 압축합니다. 한 시점의 $x$만 알면 미래 동작이 결정 — 이게 마르코프 성질.

1.1 친절 비유

  • 자전거 — 상태 = (위치, 속도, 기울기, 핸들 각도). 이 5~6개로 자전거의 미래가 결정됨.
  • 요리 — 상태 = (재료 양, 온도, 시간). 다음 상태는 현재 상태 + 동작(불 조절).
  • 은행 잔액 — 상태 = 잔액. 다음 잔액 = 현재 잔액 + 입출금.

1.2 왜 이산화하나

컴퓨터는 연속 시간을 다룰 수 없음. 일정 간격 $T_s$로 샘플링하여 $x_k = x(kT_s)$로 표기. ZOH(Zero-Order Hold)가 표준 — 입력을 한 샘플 동안 일정값으로 유지.

📚 보강 2 — 단계별 예제 (이산화)

2.1 예제 — 1차 시스템 이산화

$\dot x = -2x + u$. 샘플링 $T_s = 0.1$ s. 이산 모델?

$A = -2, B = 1$. ZOH: $A_d = e^{AT_s} = e^{-0.2} \approx 0.819$. $B_d = (A_d - 1)/A \approx 0.0905$.
이산 모델: $x_{k+1} = 0.819 x_k + 0.0905 u_k$.

2.2 예제 — 2차 시스템

스프링-질량 시스템 $\ddot q + \omega^2 q = u/m$. 상태 $x = [q, \dot q]^T$.

$A = \begin{pmatrix} 0 & 1 \\ -\omega^2 & 0 \end{pmatrix}$, $B = \begin{pmatrix} 0 \\ 1/m \end{pmatrix}$. 이산화는 scipy.signal.cont2discrete 한 줄.

2.3 예제 — 가제어성 점검

$A = \begin{pmatrix} 1 & 0 \\ 0 & 2 \end{pmatrix}$, $B = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$. 가제어?

제어성 행렬 $\mathcal{C} = [B, AB] = \begin{pmatrix} 1 & 1 \\ 0 & 0 \end{pmatrix}$. 랭크 1 < 2. 가제어 안 됨. 두 번째 상태는 제어 불가.

📚 보강 3 — 흔한 오답

  1. 이산 모델 $A_d \ne A T_s$. ZOH 식 사용 필요.
  2. 샘플링이 너무 느리면 Nyquist 위반 → 정보 손실.
  3. 이산화 후 가제어성/가관측성 재검사 필요.
  4. 모든 상태를 측정 가능하다고 가정. 현실은 일부만 측정 → 옵저버 필요.
  5. 외란/모델 오차 무시 → 정상상태 오차 발생.

📚 보강 5 — FAQ

Q1. 왜 굳이 상태공간?

전달함수는 SISO 위주. 상태공간은 MIMO 자연스럽고 비선형 확장 쉬움. MPC는 모두 상태공간.

Q2. 모델이 부정확하면?

적분 외란 모델 + 칼만 필터로 모델 오차 추정·보상.

Q3. 가제어성이 안 되면?

가제어 부분공간만 제어. 또는 액추에이터 추가/위치 변경 검토.

📚 보강 8 — 추가 워크북 5문제

워크 1

$\dot x = -3x + 2u$, $T_s = 0.2$. 이산 $A_d, B_d$?

$A_d = e^{-0.6} \approx 0.549$. $B_d = 2(1 - 0.549)/3 \approx 0.301$.

워크 2

차량 적분기 $\dot x = u$ 이산화?

$A = 0$이므로 $A_d = 1, B_d = T_s$.

워크 3

가제어성 행렬 $\mathcal{C}$ 차원?

$n \times nm$ ($n$ = 상태 수, $m$ = 입력 수).

워크 4

$x_{k+1} = 1.5 x_k + u_k$ 시스템 안정?

$|A| = 1.5 > 1$ → 불안정. MPC로 안정화 가능 (제어성 OK이면).

워크 5

이산화 시 $T_s$ 너무 크면?

샤논 한계 위반 — 빠른 동특성 누락. 일반적으로 시상수의 1/10 이하.

📚 보강 9 — 한 페이지 정리

항목식
연속$\dot x = Ax + Bu$, $y = Cx + Du$
이산$x_{k+1} = A_d x_k + B_d u_k$, $y_k = Cx_k + Du_k$
ZOH$A_d = e^{AT_s}$, $B_d = \int_0^{T_s} e^{A\tau} d\tau\, B$
제어성$\text{rank}\,[B, AB, \dots, A^{n-1}B] = n$
관측성$\text{rank}\,[C^T, A^TC^T, \dots]^T = n$
안정성이산: $|\lambda(A_d)| < 1$

📚 보강 10 — 실습 — Python으로 1차원 시스템

import numpy as np
from scipy.signal import cont2discrete

A = np.array([[-2.0]])
B = np.array([[1.0]])
C = np.array([[1.0]])
D = np.array([[0.0]])
Ts = 0.1

Ad, Bd, Cd, Dd, _ = cont2discrete((A, B, C, D), Ts, method='zoh')
print(Ad, Bd)  # [[0.8187]] [[0.0906]]

📚 보강 17 — 칼만 필터 입문

측정에서 일부 상태만 알 수 있을 때 미측정 상태를 추정하는 표준 도구.

$$\hat x_{k+1} = A\hat x_k + Bu_k + L(y_k - C\hat x_k)$$

여기서 $L$은 칼만 게인. 노이즈 통계로 결정. MPC와 결합 시 separation principle로 안정성 분리.

📚 보강 18 — 모델 단순화 트릭

  • 비선형 → 선형: 평형점 주변 선형화.
  • 시상수 빠른 모드 제거: 특이섭동(singular perturbation).

📚 보강 23 — 상태공간 표현 변환

같은 시스템이 여러 상태 좌표로 표현될 수 있습니다. $\tilde x = T x$로 변환하면:

$$\tilde A = T A T^{-1},\ \tilde B = TB,\ \tilde C = CT^{-1}$$

고유값(즉 동특성)은 변하지 않습니다. 좌표만 바꿈.

📚 보강 26 — 비선형 모델의 선형화

$\dot x = f(x, u)$를 평형점 $(x^*, u^*)$ 주변에서 테일러 전개:

$$\delta\dot x = A \delta x + B \delta u,\ A = \frac{\partial f}{\partial x}\Big|_*,\ B = \frac{\partial f}{\partial u}\Big|_*$$

이게 선형 MPC의 시작점.

📚 보강 32 — 추가 워크북 5문제

워크 6

적분기 시스템 $\dot x = u$, $T_s = 0.1$. 이산 모델?

$A_d = 1, B_d = 0.1$. 적분 시스템 그대로.

워크 7

2x2 시스템 가관측성 행렬?

$\mathcal{O} = [C^T, A^TC^T]^T$, $2 \times 2$ 행렬.

워크 8

$A = \text{diag}(0.5, 1.5)$. 안정?

두 번째 모드 $|\lambda| = 1.5 > 1$ → 불안정.

워크 9

비선형 $\dot x = -x^2 + u$를 $x = 0$ 주변 선형화?

$A = -2x|_{x=0} = 0$, $B = 1$. 선형: $\dot x = u$.

워크 10

같은 시스템 $x = 1$ 주변 선형화?

$A = -2 \cdot 1 = -2$, $B = 1$. 선형: $\delta\dot x = -2\delta x + \delta u$.

📚 보강 41 — 모델링 사례 — 1축 드론

1축 자세 모델: $\ddot \theta = \tau / I$. 상태 $x = [\theta, \dot\theta]^T$.

  • $A = \begin{pmatrix}0 & 1 \\ 0 & 0\end{pmatrix}$, $B = \begin{pmatrix}0 \\ 1/I\end{pmatrix}$.
  • 이산화: $T_s = 0.01$ s.
  • $A_d = I + AT_s = \begin{pmatrix}1 & 0.01 \\ 0 & 1\end{pmatrix}$.
  • 가제어성 OK ($\mathcal{C}$ 랭크 2).
  • $\lambda(A_d) = 1$ (이중) → 적분기 → 안정 한계.
  • MPC로 안정화 가능.

    📚 보강 47 — 상태 변수 선택 가이드

    같은 시스템도 어떤 변수를 상태로 잡느냐에 따라 풀이 난이도가 달라집니다.

    • 물리적으로 의미 있는 변수 우선 (위치·속도·온도).

    📚 보강 48 — 시간지연 처리

    입력 또는 출력에 시간지연 $\tau_d$가 있으면 상태에 추가:

    • $N_d = \tau_d / T_s$ 스텝 만큼의 입력 이력을 상태에 포함.
    • 차원 증가하지만 표준 LMPC로 처리 가능.
    • 화학공정·통신 시스템의 흔한 문제.

    📚 보강 49 — 외란 모델의 종류

    유형모델예
    상수 외란$d_{k+1} = d_k$오프셋, 바이어스

    📚 보강 50 — Augmented Model

    외란 추정을 위해 원래 상태에 외란 상태 추가:

    $$\tilde x = \begin{pmatrix} x \\ d \end{pmatrix},\quad \tilde A = \begin{pmatrix} A & E \\ 0 & A_d \end{pmatrix},\quad \tilde B = \begin{pmatrix} B \\ 0 \end{pmatrix}$$

    증강된 시스템에 칼만 필터 적용 → 외란까지 추정.

    📚 보강 51 — 정상상태 오차 제거

    적분 외란 모델 + 옵저버 → 정상상태에서 외란 추정값이 출력 오차를 0으로 만듦. 이 트릭이 사실상 산업 MPC의 표준.

    📚 보강 52 — Reference Tracking

    비용 함수에 $(x_k - x_{ref})^TQ(x_k - x_{ref})$ 또는 $(y_k - y_{ref})^T Q_y (y_k - y_{ref})$. Reference도 상태 일부로 다룰 수 있음.

    📚 보강 53 — 비최소 상태 표현

    때로 측정값을 그대로 상태로 사용. 차원이 최소가 아니지만 식별이 쉬움. ARX, ARMAX 모델이 그 예.

    📚 보강 54 — 다중 시간 스케일

    한 시스템에 빠른 모드와 느린 모드가 같이 있으면:

    • 분리 — 빠른 모드는 내부 루프, 느린 모드는 외부 루프.
    • singular perturbation으로 빠른 모드 제거.
    • cascade MPC — 두 단계 분리.

    📚 보강 55 — 제약과 상태공간

    대부분의 제약은 다음 형태로 표현됩니다.

    • 입력 한계: $u_{\min} \le u_k \le u_{\max}$.
    • 상태 한계: $x_{\min} \le x_k \le x_{\max}$.
    • 출력 한계: $y_{\min} \le Cx_k \le y_{\max}$.
    • 입력 변화율: $|\Delta u_k| \le \Delta u_{\max}$ — 추가 상태로 처리.

    📚 보강 56 — Stage Cost vs Terminal Cost

    유한 호라이즌 비용:

    $$J = \sum_{k=0}^{N-1} \underbrace{l(x_k, u_k)}_{\text{stage}} + \underbrace{V_f(x_N)}_{\text{terminal}}$$

    Stage cost = 매 스텝 누적, Terminal cost = 끝점 페널티 (안정성·근사 무한 호라이즌). W6에서 자세히.

    📚 보강 57 — 이산화 후 변경되는 것 / 변경 안 되는 것

    특성이산화로 변하나?
    가제어성일반적으로 보존 (특수 $T_s$ 제외)
    가관측성일반적으로 보존
    안정성이산 안정성으로 매핑됨
    고유값 위치$z = e^{sT_s}$ 매핑
    전달함수 영점일반적으로 변함 (sample zero 발생 가능)

    이산화: $A_d = e^{AT_s}$, $B_d = \int_0^{T_s} e^{A\tau}d\tau B$
    가제어성: $\text{rank}\,[B, AB, \dots, A^{n-1}B] = n$
    가관측성: $\text{rank}\,[C^T, A^TC^T, \dots]^T = n$
    이산 안정성: $|\lambda(A_d)| < 1$

    📚 보강 79 — 자주 쓰는 행렬 식

    • $e^A = \sum_{k=0}^\infty \frac{A^k}{k!}$ (행렬 지수).
    • $e^{(A+B)t} = e^{At} e^{Bt}$ ⇔ $AB = BA$.
    • $\det(e^A) = e^{\text{tr}(A)}$.
    • $\frac{d}{dt} e^{At} = A e^{At} = e^{At} A$.

    📚 보강 105 — 추가 학습 — 비교를 위한 전달함수

    같은 시스템의 전달함수 표현 $G(z) = C(zI - A)^{-1}B + D$. SISO에 적합. MPC는 상태공간이 표준.

    📚 보강 106 — Z-변환 정리

    $x_{k+1} = Ax_k + Bu_k$의 Z-변환: $zX(z) - zx(0) = AX(z) + BU(z)$. 여기서 $X(z) = (zI - A)^{-1}(zx(0) + BU(z))$.

    📚 보강 108 — 사례 연구 — 자율주행 차량

    대표적 차량 모델 (kinematic bicycle):

    $$\dot x = v\cos\psi,\ \dot y = v\sin\psi,\ \dot\psi = v\tan\delta/L,\ \dot v = a$$

    비선형. 평형점(직진) 주변 선형화 후 MPC로 차선 유지·추월. Tesla, 현대 ADAS 등에서 표준.

    📚 보강 109 — 사례 연구 — 4륜 드론
