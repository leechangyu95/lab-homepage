Week 9. 부분공간·일차독립

"공간을 본다" 의 첫 걸음. 부분공간(원점 통과 + 닫힘), 생성(span), 일차독립/종속 판정. 행렬과 도형이 만나는 지점.

🔁 전반부 정리

우리가 익힌 도구들

  • 벡터의 표현·연산·내적·외적 (2~4강)
  • 행렬·연산·곱셈 (5강)
  • 가우스 소거법 (6강)
  • 역행렬·행렬식 (7강)

후반부의 질문

  • "벡터들이 만들어내는 공간" 의 모양이 어떤가?
  • 벡터들이 서로 겹치는가, 아니면 새 방향을 만드는가?
  • "공간의 차원" 이라는 말이 정확히 무슨 뜻인가?
전반부가 "계산 도구"였다면, 후반부는 "구조 이해"입니다. 9~10강은 그 첫 두 걸음.

🎯 오늘의 학습 목표

🔹 $\mathbb{R}^n$ 의 부분공간의 정의를 이해한다 (덧셈·스칼라곱에 닫힘)

🔸 직선·평면이 원점을 지나야만 부분공간이 됨을 알고, 예/반례를 구별한다

🔸 벡터들의 생성(span) 의 의미를 그림과 식으로 설명한다

🔸 일차독립과 일차종속을 정의로 판정할 수 있다

🔸 일차독립 판정에 행렬식이나 가우스 소거를 사용할 수 있다

🌐 직관 — "공간" 이라는 말

$\mathbb{R}^n$ 은 무엇인가

  • $\mathbb{R}^2$ : 평면 위 모든 점.
  • $\mathbb{R}^3$ : 3차원 공간의 모든 점.
  • $\mathbb{R}^n$ : 실수 $n$ 개로 이루어진 모든 벡터의 모임.

부분공간(subspace)

  • $\mathbb{R}^n$ 안의 "작지만 자기 완결적인" 공간.
  • 예: $\mathbb{R}^3$ 안의 어떤 평면 — 평면 위 두 벡터를 더해도 그 평면 위, 스칼라곱해도 그 평면 위.
🌱 핵심 직관: "안에서 더하고 곱해도 절대 밖으로 빠져나가지 않는 부분" — 그게 부분공간입니다.

📐 부분공간의 정의

정의

$\mathbb{R}^n$ 의 부분집합 $W$ 가 다음 세 조건을 모두 만족하면 부분공간.

  1. 영벡터 포함: $\mathbf{0} \in W$
  2. 덧셈 닫힘: $\mathbf{u},\mathbf{v} \in W \Rightarrow \mathbf{u}+\mathbf{v} \in W$
  3. 스칼라곱 닫힘: $\mathbf{u} \in W,\ c \in \mathbb{R} \Rightarrow c\mathbf{u} \in W$

한 마디로

"안에서 일차결합이 닫힌다"

예

  • $W = \{\mathbf{0}\}$ → 가장 작은 부분공간 (영부분공간).
  • $W = \mathbb{R}^n$ → 가장 큰 부분공간 (자기 자신).

✅ 부분공간의 예 (1) — 원점 지나는 직선

예

$W = \{(t,\,2t) : t \in \mathbb{R}\}$ ⊂ $\mathbb{R}^2$ — 원점 지나는 기울기 2인 직선.

검증

  • 영벡터: $t = 0$ → $(0, 0) \in W$ ✓
  • 덧셈: $(t_1, 2t_1) + (t_2, 2t_2) = (t_1+t_2,\,2(t_1+t_2)) \in W$ ✓
  • 스칼라곱: $c(t, 2t) = (ct,\,2ct) \in W$ ✓

→ 부분공간!

❌ 부분공간 아닌 예 — 원점 안 지나는 직선

예

$W = \{(t,\,2t + 1) : t \in \mathbb{R}\}$ — 기울기 2, $y$ 절편 1.

검증

  • 영벡터? $t = 0 \to (0, 1) \neq \mathbf{0}$. 원점 안 지남! ✗

→ 부분공간 아님.

$\mathbb{R}^n$ 의 부분공간은 반드시 원점을 지나야 합니다. 평면도 마찬가지!

✅ 부분공간의 예 (2) — 원점 지나는 평면

예

$W = \{(x, y, z) : x + y + z = 0\}$ ⊂ $\mathbb{R}^3$.

검증

  • 영벡터: $(0,0,0)$ → $0+0+0 = 0$ ✓
  • 덧셈: $(x_1+x_2)+(y_1+y_2)+(z_1+z_2) = 0+0 = 0$ ✓
  • 스칼라곱: $cx + cy + cz = c\cdot 0 = 0$ ✓

→ 부분공간!

반례

$W' = \{(x,y,z) : x + y + z = 1\}$ → 영벡터 미포함. 부분공간 아님.

🌱 일반화: $A\mathbf{x} = \mathbf{0}$ (동차 시스템)의 해집합은 항상 부분공간. $A\mathbf{x} = \mathbf{b}$ ($\mathbf{b} \neq \mathbf{0}$)의 해집합은 부분공간이 아님.

🌟 생성 (Span)

정의

벡터들 $\mathbf{v}_1, \ldots, \mathbf{v}_k$ 의 생성 = 모든 일차결합의 모임.

$$\text{span}(\mathbf{v}_1, \ldots, \mathbf{v}_k) = \{c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k : c_i \in \mathbb{R}\}$$

핵심 사실

$\text{span}(\mathbf{v}_1, \ldots, \mathbf{v}_k)$ 은 항상 부분공간.

예시

  • $\mathbf{v}_1 = (1,0)$ → span = $x$ 축 전체.
  • $\mathbf{v}_1 = (1,0),\ \mathbf{v}_2 = (0,1)$ → span = $\mathbb{R}^2$ 전체.
  • $\mathbf{v}_1 = (1,0,0),\ \mathbf{v}_2 = (0,1,0)$ → span = $xy$ 평면.
  • $\mathbf{v}_1 = (1,1),\ \mathbf{v}_2 = (2,2)$ → span = 원점 지나는 한 직선 (둘이 평행하므로).
"생성" = "이 벡터들로 도달할 수 있는 모든 곳". 도달 가능한 영역의 모양이 곧 부분공간의 정체.

🟰 일차독립 (Linear Independence)

정의

벡터들 $\mathbf{v}_1, \ldots, \mathbf{v}_k$ 가 일차독립이라는 것은:

$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$$

의 해가 $c_1 = c_2 = \cdots = c_k = 0$ 뿐 임을 뜻함.

일차종속(linearly dependent)

일차독립이 아니면 종속. 즉 어떤 $c_i$ 들이 모두 0은 아닌 채로 위 식이 성립.

직관

  • 독립 = 어떤 벡터도 다른 벡터들의 일차결합으로 표현 안 됨 → "각자 다른 방향".
  • 종속 = 한 벡터가 다른 벡터들의 일차결합 → "겹치는 방향이 있음".

🖼️ 일차독립 — 그림으로

🌱 한 줄 룰: $n$차원 공간에서 벡터가 $n+1$ 개 이상이면 무조건 일차종속. (도달할 수 있는 방향이 $n$ 개뿐이니까.)

🧮 일차독립 판정 — 정의로

절차

$c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$ 을 풀어 본다.

  • 해가 $c_i = 0$ 뿐이면 독립.
  • 해가 비자명(nonzero)이면 종속.

예

$\mathbf{v}_1 = (1,2),\ \mathbf{v}_2 = (3,4)$

$$c_1\binom{1}{2} + c_2\binom{3}{4} = \binom{0}{0}$$

$$\begin{cases} c_1 + 3c_2 = 0 \\ 2c_1 + 4c_2 = 0 \end{cases}$$

가우스 소거 → $c_2 = 0,\ c_1 = 0$ (유일해) → 독립 ✓

예 (종속)

$\mathbf{v}_1 = (1,2),\ \mathbf{v}_2 = (2,4)$

$$c_1 + 2c_2 = 0 \to c_1 = -2c_2$$

$c_2 = 1,\ c_1 = -2$ 같은 비자명 해 존재 → 종속 ✗

🧮 일차독립 판정 — 행렬식 / 가우스 소거

정사각인 경우 — 행렬식이 가장 빠름

벡터들을 열로 모아 행렬 $V$ 만들기.

$$\det V \neq 0 \;\Longleftrightarrow\; \text{독립}$$

예

$$V = \begin{pmatrix} 1 & 3 \\ 2 & 4 \end{pmatrix},\ \det V = 4 - 6 = -2 \neq 0 \to \text{독립}$$

일반 ($k$ 벡터, $n$ 차원, $k \neq n$ 가능) — 가우스 소거

$V$ 를 RREF 로 만들어 피벗의 수가 $k$ 개면 독립, 모자라면 종속.

도구적용
행렬식정사각만 ($n=k$). 가장 빠름.
가우스 소거모든 경우. 일반적.

🌟 동치 명제 정리

$n\times n$ 행렬 $A$ 에 대해 (7강 동치 6가지 + 추가)

다음은 모두 동치:

  1. $A$ 가역
  2. $\det A \neq 0$
  3. $A\mathbf{x} = \mathbf{b}$ 가 모든 $\mathbf{b}$ 에 유일해
  4. $A\mathbf{x} = \mathbf{0}$ 의 유일해는 $\mathbf{x} = \mathbf{0}$
  5. $A$ 의 RREF = $I$
  6. $A$ 의 열들이 일차독립
  7. $A$ 의 행들이 일차독립
  8. $A$ 의 열들이 $\mathbb{R}^n$ 을 생성
선형대수에서 가장 중요한 정리. "가역"과 "독립", "생성" 은 사실 같은 한 가지 사실의 다른 옷입니다.

🧮 종합 예제

예제

$\mathbf{v}_1 = (1, 0, 1),\ \mathbf{v}_2 = (2, 1, 0),\ \mathbf{v}_3 = (3, 1, 1)$ 의 일차독립 여부?

풀이 (행렬식)

$$\det\begin{pmatrix} 1 & 2 & 3 \\ 0 & 1 & 1 \\ 1 & 0 & 1 \end{pmatrix} = 1\cdot(1\cdot 1 - 1\cdot 0) - 2\cdot(0\cdot 1 - 1\cdot 1) + 3\cdot(0\cdot 0 - 1\cdot 1) = 1 + 2 - 3 = 0$$

$\det = 0$ → 종속!

종속 관계 찾기

관찰: $\mathbf{v}_3 = \mathbf{v}_1 + \mathbf{v}_2$. 즉 $\mathbf{v}_1 + \mathbf{v}_2 - \mathbf{v}_3 = \mathbf{0}$ → 비자명 일차결합으로 0이 됨.

🌍 응용 — 해 공간의 차원

동차 시스템 $A\mathbf{x} = \mathbf{0}$ 의 해

  • 항상 자명해 $\mathbf{x} = \mathbf{0}$ 가짐.
  • 그 외 해가 더 있으면 → 자유변수 존재 → 해 공간이 직선/평면/고차원.
  • 해 공간의 "차원" = 자유변수의 개수 (10주차 정리).

예

$A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \end{pmatrix}$

  • RREF: $\begin{pmatrix} 1 & 2 & 3 \\ 0 & 0 & 0 \end{pmatrix}$
  • 자유변수 2개 → 해 공간 = 2차원 부분공간.
자유변수의 개수가 곧 "해를 결정하는 데 필요한 매개변수 수" = "그 부분공간의 차원". 다음 강의에서 이를 정리합니다.

📝 정리 — 오늘의 핵심

1️⃣ 부분공간

  • 영벡터 포함, 덧셈·스칼라곱에 닫힌 부분집합.
  • $\mathbb{R}^n$ 의 부분공간은 반드시 원점을 지난다.

2️⃣ 생성

  • $\text{span}(\mathbf{v}_1, \ldots, \mathbf{v}_k)$ = 모든 일차결합 → 항상 부분공간.

3️⃣ 일차독립 / 종속

  • 독립: $c_1\mathbf{v}_1 + \cdots = \mathbf{0}$ 의 유일해가 모두 0.
  • 판정: 행렬식 (정사각) 또는 가우스 소거 (일반).
  • $n$ 차원에 $n+1$ 개 이상 벡터는 무조건 종속.

4️⃣ 핵심 동치

"가역 ↔ 행렬식 ≠ 0 ↔ 열 일차독립 ↔ 열이 공간 생성"
