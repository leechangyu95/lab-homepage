1. 핵심 공식 한눈에 (W9~W14)

1.1 부분공간·일차독립 (W9)

  • 부분공간 조건: ① $\mathbf{0} \in W$ ② 덧셈 닫힘 ③ 스칼라곱 닫힘.
  • $\mathbb{R}^n$ 의 부분공간은 반드시 원점을 지난다.
  • $\text{span}(\mathbf{v}_1, \ldots, \mathbf{v}_k)$ = 모든 일차결합 → 항상 부분공간.
  • 일차독립: $c_1\mathbf{v}_1+\cdots+c_k\mathbf{v}_k = \mathbf{0}$ 의 유일해가 모두 0.
  • 판정: 정사각이면 행렬식, 일반이면 가우스 소거(피벗 수 = 벡터 수).

1.2 기저·차원 (W10)

  • 기저 = 생성 + 일차독립.
  • 표준기저: $\{\mathbf{e}_1, \ldots, \mathbf{e}_n\}$.
  • $\dim W$ = 기저 벡터 개수. 같은 공간이면 어떤 기저든 개수 동일.
  • $\dim \mathbb{R}^n = n$. $W \subseteq \mathbb{R}^n$ 이면 $0 \le \dim W \le n$.

1.3 선형변환 (W11)

  • 선형성: $T(\mathbf{u}+\mathbf{v}) = T(\mathbf{u})+T(\mathbf{v})$, $T(c\mathbf{u}) = cT(\mathbf{u})$.
  • $T(\mathbf{x}) = A\mathbf{x}$. $A$ 의 $j$ 번째 열 = $T(\mathbf{e}_j)$.
  • 회전: $R_\theta = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$.
  • $x$ 축 반사 $\begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$, $y$ 축 반사 $\begin{pmatrix} -1 & 0 \\ 0 & 1 \end{pmatrix}$, $y=x$ 반사 $\begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$.
  • 합성 = 곱셈 (오른쪽부터 적용, 비가환).

1.4 핵·치역·계수 (W12)

$$\text{rank}(A) + \text{nullity}(A) = n \quad (n = \text{입력 차원})$$
  • $\text{Col}(A)$ = 치역 = $A$ 의 열들이 만드는 부분공간.
  • $\text{Null}(A)$ = 핵 = $A\mathbf{x}=\mathbf{0}$ 의 해.
  • rank = 피벗 수, nullity = 자유변수 수.
  • $A\mathbf{x}=\mathbf{b}$ 해 존재 ↔ $\mathbf{b} \in \text{Col}(A)$ ↔ $\text{rank}(A) = \text{rank}([A|\mathbf{b}])$.
  • 일반해 = 특수해 + 영공간 전체.

1.5 고윳값·고유벡터 (W13)

$$A\mathbf{v} = \lambda\mathbf{v},\ \mathbf{v}\ne\mathbf{0} \;\;\Longleftrightarrow\;\; \det(A - \lambda I) = 0$$
  • 특성다항식 풀어 $\lambda$ → 각 $\lambda$ 마다 $(A-\lambda I)\mathbf{v}=\mathbf{0}$ 의 영공간 = 고유공간.
  • 검산: $\sum\lambda_i = \text{tr}(A)$, $\prod \lambda_i = \det A$.
  • $A^k$ 의 고윳값 = $\lambda^k$, 고유벡터 동일.
  • $A$ 가역 ↔ 고윳값에 0 이 없음.

1.6 대각화·정사영 (W14)

$$A = PDP^{-1} \quad\Longrightarrow\quad A^k = P D^k P^{-1}$$ $$\text{proj}_\mathbf{v}(\mathbf{u}) = \frac{\mathbf{u}\cdot\mathbf{v}}{\mathbf{v}\cdot\mathbf{v}}\mathbf{v}$$
  • 대각화 가능 ↔ $n$ 개 일차독립 고유벡터 존재 (충분조건: 고윳값 모두 다름 / 대칭행렬).
  • 정사영 = 한 벡터의 다른 벡터(부분공간) 위 그림자. $\mathbf{u} - \text{proj}_\mathbf{v}(\mathbf{u}) \perp \mathbf{v}$.
  • 최소제곱: $M^T M \mathbf{x} = M^T \mathbf{y}$ (정상방정식).

2. 풀이 패턴 정리

패턴 A — 부분공간 판정

  1. $\mathbf{0}$ 포함 확인 → 안 들어 있으면 즉시 "아님".
  2. 임의 두 원소 더한 결과가 다시 그 집합에 속하는지.
  3. 임의 원소에 스칼라 곱한 결과가 다시 그 집합에 속하는지.

패턴 B — 기저·차원 찾기 (열공간)

  1. 벡터들을 열로 모아 $V$ 행렬.
  2. $V$ 를 RREF 로 변환.
  3. 피벗 열에 대응하는 원래 벡터 = 기저 (RREF 의 열 자체가 아님!).
  4. $\dim$ = 피벗 수.

패턴 C — 변환의 표현행렬 만들기

  1. $T(\mathbf{e}_1), T(\mathbf{e}_2), \ldots$ 를 각각 계산.
  2. 이 결과를 열로 나란히 → 표현행렬 $A$.
  3. 검산: $A\mathbf{e}_j$ 가 $T(\mathbf{e}_j)$ 와 같은가.

패턴 D — rank·nullity 계산

  1. $A$ 를 RREF 로.
  2. 피벗 수 = rank, 자유변수 수 = nullity.
  3. 차원정리 $\text{rank}+\text{nullity}=n$ 으로 검산.
  4. 영공간 기저: 자유변수에 1, 다른 자유변수 0 을 대입한 해 벡터들.

패턴 E — 고윳값·고유벡터

  1. $\det(A - \lambda I) = 0$ 푼다 → $\lambda_1, \ldots, \lambda_n$.
  2. 각 $\lambda$ 에 대해 $(A - \lambda I)\mathbf{v} = \mathbf{0}$ 가우스 소거.
  3. 영공간의 0 아닌 해 → 고유벡터.
  4. 검산: $A\mathbf{v} = \lambda\mathbf{v}$ 한 항만 확인. trace/det 도 추가 검산.

패턴 F — 대각화 및 $A^k$

  1. 고윳값·고유벡터 모두 구하기.
  2. $P$ = 고유벡터 열로, $D$ = 고윳값 대각.
  3. $A^k = P D^k P^{-1}$ — $D^k$ 는 대각 원소 $k$ 제곱만 하면 됨.

3. 기출 유형 6문제

문제 1 (부분공간 판정)

$W = \{(x, y, z) \in \mathbb{R}^3 : x + 2y - z = 0\}$ 가 부분공간인지 판정하시오.

풀이

$\mathbf{0}$: $0+0-0=0$ ✓. 덧셈: 두 해 $(x_1,y_1,z_1),(x_2,y_2,z_2)$ 의 합은 $x+y$ 의 좌변이 $0+0=0$ ✓. 스칼라곱도 동일 ✓. → 부분공간. 차원 2 (3변수 - 1제약).

문제 2 (일차독립)

$\mathbf{v}_1 = (1, 0, 1),\ \mathbf{v}_2 = (2, 1, 0),\ \mathbf{v}_3 = (3, 1, 1)$ 의 일차독립 여부를 판정하시오.

풀이

열벡터 행렬의 $\det = 0$ (관찰: $\mathbf{v}_3 = \mathbf{v}_1 + \mathbf{v}_2$). → 일차종속.

문제 3 (rank·nullity)

$A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \\ 0 & 1 & 1 \end{pmatrix}$ 의 rank, nullity, 영공간 기저를 구하시오.

풀이

$R_2 \to R_2 - 2R_1$ → 2행 모두 0. $R_3$ 와 자리 바꾸기 후 RREF: $\begin{pmatrix} 1 & 0 & 1 \\ 0 & 1 & 1 \\ 0 & 0 & 0 \end{pmatrix}$. rank = 2, nullity = 1. 자유변수 $z=t$ → $x = -t,\ y = -t$. 영공간 기저: $\{(-1,-1,1)\}$.

문제 4 (선형변환·합성)

$A$ = 90° 반시계 회전, $B$ = $y$ 축 반사. $AB$ 와 $BA$ 를 각각 구하고, $\binom{1}{0}$ 에 적용한 결과를 비교하시오.

풀이

$A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix},\ B = \begin{pmatrix} -1 & 0 \\ 0 & 1 \end{pmatrix}$. $AB = \begin{pmatrix} 0 & -1 \\ -1 & 0 \end{pmatrix},\ BA = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$. $AB\binom{1}{0} = \binom{0}{-1},\ BA\binom{1}{0} = \binom{0}{1}$ — 결과 다름 (비가환).

문제 5 (고윳값·고유벡터)

$A = \begin{pmatrix} 5 & 4 \\ 4 & 5 \end{pmatrix}$ 의 고윳값과 고유벡터를 구하시오.

풀이

$\det(A - \lambda I) = (5-\lambda)^2 - 16 = \lambda^2 - 10\lambda + 9 = (\lambda-1)(\lambda-9) = 0$ → $\lambda = 1, 9$.

$\lambda = 9$: $\begin{pmatrix} -4 & 4 \\ 4 & -4 \end{pmatrix}\mathbf{v} = \mathbf{0}$ → $\mathbf{v} = (1, 1)$.

$\lambda = 1$: $\begin{pmatrix} 4 & 4 \\ 4 & 4 \end{pmatrix}\mathbf{v} = \mathbf{0}$ → $\mathbf{v} = (1, -1)$.

검산: trace = 10 = 1+9 ✓, det = 25-16 = 9 = 1·9 ✓.

문제 6 (정사영)

$\mathbf{u} = (2, 3)$ 의 $\mathbf{v} = (1, 2)$ 위로의 정사영을 구하시오.

풀이

$\mathbf{u}\cdot\mathbf{v} = 2 + 6 = 8$, $\mathbf{v}\cdot\mathbf{v} = 5$. $\text{proj}_\mathbf{v}(\mathbf{u}) = \tfrac{8}{5}(1, 2) = (\tfrac{8}{5}, \tfrac{16}{5})$. 직교 성분: $\mathbf{u} - \text{proj} = (\tfrac{2}{5}, -\tfrac{1}{5})$. 검산: 직교 성분 · $\mathbf{v} = \tfrac{2}{5} - \tfrac{2}{5} = 0$ ✓.

4. 한 학기 흐름 — 도구상자 정리

  1. 벡터 (W2–W4): 화살표 + 내적·외적 + 직선·평면. 기하 직관의 토대.
  2. 행렬 (W5–W7): 데이터·연립방정식·역행렬·행렬식. 가장 강력한 계산 도구상자.
  3. 구조 (W9–W10): 부분공간·일차독립·기저·차원. "공간을 보는 눈".
  4. 변환 (W11–W12): 행렬 = 변환. 핵·치역·rank/nullity 로 시스템 해 구조 분석.
  5. 고유 (W13–W14): 변환의 본질적 축. 대각화로 거듭제곱·동역학 한 번에. 정사영으로 최소제곱까지.

5. 다음 학기 강의로의 연결

  • 자동제어 — 상태공간 $\dot{\mathbf{x}}=A\mathbf{x}$. $A$ 의 고윳값으로 안정성 판정 (W13 그대로 사용).
  • 메카트로닉스 — 회로 노드 분석은 가우스 소거 (W6) 의 직접 적용.
  • 경로계획·추종 · 로봇공학특론 L2 — 회전 행렬·좌표 변환 (W11).
  • 머신러닝 · 선형 회귀 — 정사영·최소제곱 (W14) 가 회귀의 정체.
  • 신호처리·푸리에 — 푸리에 변환 = "주파수 기저로의 좌표 변환" (W10 의 일반화).
  • SVD, PCA — 대각화 (W14) 의 직사각 일반화. 데이터 압축의 핵심.

6. 시험 직전 체크리스트

  • 부분공간 판정 3 조건을 즉답할 수 있는가?
  • "독립 vs 종속" 판정에 행렬식 / RREF 모두 사용 가능한가?
  • 회전 행렬·반사 행렬을 외워서 쓸 수 있는가?
  • 차원정리 $\text{rank}+\text{nullity}=n$ 의 의미를 그림으로 설명할 수 있는가?
  • 특성다항식 → 고윳값 → 고유벡터 한 사이클 (2x2) 을 3분 안에 풀 수 있는가?
  • 대각화 $A=PDP^{-1}$ 에서 $P, D$ 가 무엇인지 즉답?
  • 정사영 공식 $\dfrac{\mathbf{u}\cdot\mathbf{v}}{\mathbf{v}\cdot\mathbf{v}}\mathbf{v}$ — 분모 분자 의미를 그림으로?
