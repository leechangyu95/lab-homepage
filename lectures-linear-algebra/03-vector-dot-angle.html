Week 3. 벡터 (2) — 내적·길이·각도

벡터의 길이(노름), 내적의 두 가지 정의(좌표·기하), 각도와 직교 판정. 내적 한 줄로 거의 모든 도형 문제가 풀린다.

🔁 복습 — 지난 시간

벡터 = 화살표 = 숫자 묶음

  • 표현: $\mathbf{v} = (v_1, v_2)$ 또는 $\begin{pmatrix} v_1 \\ v_2 \end{pmatrix}$
  • 두 가지 연산:
  • 덧셈: 같은 자리끼리 → 삼각형/평행사변형 법칙
  • 스칼라곱: 모든 성분에 $c$ 곱 → 길이 조절

아직 못 한 것

  • 벡터의 길이가 얼마인가?
  • 두 벡터 사이 각도가 몇 도인가?
  • 두 벡터가 수직(직교) 인지 어떻게 알지?
오늘 핵심 도구 = 내적(dot product). 길이·각도·수직 판정을 모두 한 식으로 해결합니다.

🎯 오늘의 학습 목표

🔹 벡터의 길이(노름) 를 계산할 수 있다 — 피타고라스의 일반화

🔸 내적을 좌표(대수적)와 그림(기하적) 두 가지로 정의·계산할 수 있다

🔸 내적으로 두 벡터 사이 각도를 구할 수 있다

🔸 두 벡터가 직교인지 판정할 수 있다 ($\mathbf{a}\cdot\mathbf{b} = 0$)

🔸 응용: 일(work), 유사도(similarity) 같은 실제 예시에 연결할 수 있다

📏 벡터의 길이 (노름, norm)

2D — 피타고라스

$\mathbf{v} = (v_1, v_2)$ 의 길이는 직각삼각형의 빗변.

$$\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2}$$

nD — 일반화

$$\|\mathbf{v}\| = \sqrt{v_1^2 + v_2^2 + \cdots + v_n^2}$$

"노름(norm)" = "길이"의 정식 명칭. 표기는 $\|\mathbf{v}\|$ (절댓값 두 줄). $|v|$ 와 헷갈리지 말 것.

📏 단위벡터로 정규화하기

정규화 (normalization)

0이 아닌 벡터 $\mathbf{v}$ 를 그 길이로 나누면 길이 1짜리 단위벡터가 된다.

$$\hat{\mathbf{v}} = \frac{\mathbf{v}}{\|\mathbf{v}\|}$$

예

$\mathbf{v} = (3,\,4),\ \|\mathbf{v}\| = 5$

$$\hat{\mathbf{v}} = \tfrac{1}{5}(3,\,4) = (0.6,\,0.8)$$

확인: $0.6^2 + 0.8^2 = 0.36 + 0.64 = 1$ ✓

🌱 왜 정규화?: "방향만 알고 싶고 크기는 표준화하고 싶을 때". 예를 들어 GPS에서 "북쪽 방향" 처럼 단위 길이로 표현된 방향만 필요할 때.

📐 두 점 사이 거리

점 $A,\,B$ 사이 거리

변위벡터 $\overrightarrow{AB} = B - A$ 의 길이.

$$d(A,\,B) = \|B - A\| = \sqrt{(b_1 - a_1)^2 + (b_2 - a_2)^2}$$

예

$A = (1,\,2),\ B = (4,\,6)$

$$d = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = 5$$

🌱 고등학교에서 본 두 점 사이 거리 공식이 사실은 "변위벡터의 노름" 이었습니다.

✖️ 내적 (Dot Product) — 정의

대수적 정의 (좌표로 계산)

같은 자리끼리 곱한 후 모두 더한다. 결과는 숫자(스칼라).

$$ \mathbf{a} \cdot \mathbf{b} = \begin{pmatrix} a_1 \\ a_2 \end{pmatrix} \cdot \begin{pmatrix} b_1 \\ b_2 \end{pmatrix} = a_1 b_1 + a_2 b_2 $$

일반 (n차원)

$$\mathbf{a} \cdot \mathbf{b} = a_1 b_1 + a_2 b_2 + \cdots + a_n b_n$$

예

$\mathbf{a} = (3,\,2),\ \mathbf{b} = (4,\,-1)$

$$\mathbf{a}\cdot\mathbf{b} = 3\cdot 4 + 2\cdot(-1) = 12 - 2 = 10$$

결과는 스칼라 한 개. 벡터가 아닙니다! 외적(다음 강의)과 헷갈리지 마세요.

✖️ 내적 — 기하학적 의미

또 하나의 식 (그림으로 본 정의)

$$\boxed{\mathbf{a}\cdot\mathbf{b} = \|\mathbf{a}\|\,\|\mathbf{b}\|\,\cos\theta}$$

여기서 $\theta$ 는 두 벡터 사이의 각도 (0 ≤ θ ≤ π).

두 정의는 완전히 동치. 그래서 내적은 "좌표 계산" 과 "각도·길이" 라는 서로 다른 두 세계를 연결하는 다리 역할을 합니다.

🎮 인터랙티브: 두 정의가 정말 같은지 직접 확인

두 화살표를 마우스로 끌어서 움직여 보세요. 좌측 그림 위쪽의 좌표식 $a_1b_1+a_2b_2$ 값과 아래쪽의 기하식 $\|\mathbf{a}\|\|\mathbf{b}\|\cos\theta$ 값이 항상 같다는 걸 눈으로 확인할 수 있습니다. 또한 각도가 90°에 가까워지면 두 값이 어떻게 0으로 수렴하는지도 보입니다.

파랑·주황 화살표의 끝점을 마우스로 끌어 보세요.

📐 내적으로 각도 구하기

공식 뒤집기

$$\cos\theta = \frac{\mathbf{a}\cdot\mathbf{b}}{\|\mathbf{a}\|\,\|\mathbf{b}\|}$$

절차

  1. 좌표로 $\mathbf{a}\cdot\mathbf{b}$ 계산
  2. $\|\mathbf{a}\|,\,\|\mathbf{b}\|$ 계산
  3. 위 식에 대입 → $\cos\theta$ → $\theta = \arccos(\cdots)$

예

$\mathbf{a} = (1,\,0),\ \mathbf{b} = (1,\,1)$

  • $\mathbf{a}\cdot\mathbf{b} = 1\cdot 1 + 0\cdot 1 = 1$
  • $\|\mathbf{a}\| = 1,\ \|\mathbf{b}\| = \sqrt{2}$
  • $\cos\theta = \dfrac{1}{1\cdot\sqrt{2}} = \dfrac{1}{\sqrt{2}}$
  • $\theta = 45°$ ✓

⊥ 직교 (orthogonality)

핵심 사실

  • $\theta = 90°$ ↔ $\cos\theta = 0$ ↔ $\mathbf{a}\cdot\mathbf{b} = 0$.

$$\boxed{\mathbf{a} \perp \mathbf{b} \iff \mathbf{a}\cdot\mathbf{b} = 0}$$

예

  • $\mathbf{a} = (1,\,0),\ \mathbf{b} = (0,\,1)$ → $\mathbf{a}\cdot\mathbf{b} = 0$ → 직교 ✓
  • $\mathbf{a} = (1,\,2),\ \mathbf{b} = (2,\,-1)$ → $\mathbf{a}\cdot\mathbf{b} = 2 - 2 = 0$ → 직교 ✓
  • $\mathbf{a} = (3,\,2),\ \mathbf{b} = (1,\,1)$ → $\mathbf{a}\cdot\mathbf{b} = 5$ → 직교 ✗
"직교 판정"은 선형대수 전체에서 가장 자주 쓰는 도구 중 하나. 내적이 0인지만 확인하면 끝.

🔄 내적과 각도의 부호

부호$\cos\theta$$\theta$의미
$\mathbf{a}\cdot\mathbf{b} > 0$$> 0$$0 \le \theta < 90°$같은 방향에 가깝다
$\mathbf{a}\cdot\mathbf{b} = 0$$= 0$$\theta = 90°$수직
$\mathbf{a}\cdot\mathbf{b} < 0$$< 0$$90° < \theta \le 180°$반대 방향에 가깝다

🧮 내적의 성질

성질 (보통 수의 곱셈과 비슷)

  1. 교환법칙: $\mathbf{a}\cdot\mathbf{b} = \mathbf{b}\cdot\mathbf{a}$
  2. 분배법칙: $\mathbf{a}\cdot(\mathbf{b}+\mathbf{c}) = \mathbf{a}\cdot\mathbf{b} + \mathbf{a}\cdot\mathbf{c}$
  3. 스칼라 결합: $(c\mathbf{a})\cdot\mathbf{b} = c(\mathbf{a}\cdot\mathbf{b})$
  4. 자기 자신과 내적 = 길이 제곱: $\mathbf{a}\cdot\mathbf{a} = \|\mathbf{a}\|^2$

4번 확인

$$\mathbf{a}\cdot\mathbf{a} = a_1^2 + a_2^2 + \cdots + a_n^2 = \|\mathbf{a}\|^2 \;\;\checkmark$$

노름은 사실 "자기 자신과의 내적의 제곱근" $\|\mathbf{a}\| = \sqrt{\mathbf{a}\cdot\mathbf{a}}$ 입니다. 노름과 내적이 분리된 개념이 아님!

📊 코시–슈바르츠 부등식 (직관)

식

$$|\mathbf{a}\cdot\mathbf{b}| \le \|\mathbf{a}\|\,\|\mathbf{b}\|$$

🌱 한 줄 직관 — "내적은 길이의 곱이 한계다"

두 화살표가 완전히 같은 방향일 때 내적값이 가장 커집니다. 이때 $\cos\theta = 1$ 이므로 내적은 정확히 $\|\mathbf{a}\|\|\mathbf{b}\|$. 방향이 어긋날수록 코사인이 작아져서 내적값도 줄어듭니다. 즉 "길이의 곱"이 내적이 가질 수 있는 최댓값의 한계입니다. 위 인터랙티브에서 두 벡터를 평행하게 만들어 보면 좌표식 값이 정확히 두 길이의 곱이 됩니다.

왜 성립하나 (정식 증명)

  • $\mathbf{a}\cdot\mathbf{b} = \|\mathbf{a}\|\|\mathbf{b}\|\cos\theta$ 에서 $|\cos\theta| \le 1$ 이므로 자명.
  • 등호 $|\mathbf{a}\cdot\mathbf{b}| = \|\mathbf{a}\|\|\mathbf{b}\|$ 는 두 벡터가 평행(같은 방향 또는 반대 방향)일 때만 성립.

의미

  • 내적값은 절대 두 길이의 곱을 넘을 수 없다.
  • "두 벡터 사이 코사인은 ±1을 넘을 수 없다"의 다른 표현.
  • 유사도 측정에서 코사인 유사도 $\dfrac{\mathbf{a}\cdot\mathbf{b}}{\|\mathbf{a}\|\|\mathbf{b}\|}$ 가 항상 [−1, +1]에 머무는 보장이 바로 이 부등식.
이 부등식은 단순해 보이지만, 통계의 상관계수가 항상 [−1, +1] 범위에 있는 이유의 출발점이고, 딥러닝 임베딩의 코사인 유사도가 의미를 가지는 근거이기도 합니다.

🧮 종합 예제

예제

$\mathbf{a} = (2,\,1,\,2),\ \mathbf{b} = (1,\,-2,\,2)$

  1. $\mathbf{a}\cdot\mathbf{b}$
  2. $\|\mathbf{a}\|,\,\|\mathbf{b}\|$
  3. 두 벡터 사이 각도
  4. 직교 여부

풀이

  1. $\mathbf{a}\cdot\mathbf{b} = 2\cdot 1 + 1\cdot(-2) + 2\cdot 2 = 2 - 2 + 4 = 4$
  2. $\|\mathbf{a}\| = \sqrt{4+1+4} = 3$, $\|\mathbf{b}\| = \sqrt{1+4+4} = 3$
  3. $\cos\theta = \dfrac{4}{3\cdot 3} = \dfrac{4}{9} \approx 0.444$ → $\theta \approx 63.6°$
  4. $\mathbf{a}\cdot\mathbf{b} = 4 \neq 0$ → 직교 아님

🌍 응용 (1) — 일(work)

물리 시간에 본 일 공식

물체에 일정한 힘 $\mathbf{F}$ 가 작용하여 변위 $\mathbf{d}$ 만큼 이동했다면, 한 일은:

$$W = \|\mathbf{F}\|\,\|\mathbf{d}\|\,\cos\theta = \mathbf{F}\cdot\mathbf{d}$$

직관

  • 힘과 이동 방향이 같으면 ($\theta = 0°$) → $W$ 가 최대.
  • 힘이 이동 방향에 수직이면 ($\theta = 90°$) → $W = 0$ (예: 가방을 들고 수평으로 걷기).
  • 힘이 반대 방향이면 → $W < 0$ (마찰력의 일).
고등학교 물리의 "일 = 힘 × 거리 × cosθ" 공식이 사실은 벡터의 내적이었습니다.

🌍 응용 (2) — 추천시스템의 "유사도"

사람의 영화 취향을 벡터로

  • 5개 영화 각각에 1~5점 평점.
  • 한 사람의 취향 = 5차원 벡터, 예: $\mathbf{u}_A = (5,\,3,\,1,\,4,\,2)$.

두 사람 취향이 비슷한가?

코사인 유사도 = 두 벡터 사이 각도의 코사인.

$$\text{sim}(\mathbf{u}_A,\,\mathbf{u}_B) = \frac{\mathbf{u}_A \cdot \mathbf{u}_B}{\|\mathbf{u}_A\|\,\|\mathbf{u}_B\|}$$

  • 1에 가까우면 취향이 비슷 → 같은 영화 추천.
  • 0에 가까우면 무관.
  • −1에 가까우면 정반대 취향.
넷플릭스·유튜브 추천의 가장 단순한 버전이 바로 이 공식. "당신과 비슷한 사람들이 좋아한 영화" 의 수학적 정의.

🌍 응용 (3) — "이 두 신호가 닮았는가?"

신호 = 시간에 따라 측정된 숫자열

  • 1초씩 100번 측정하면 100차원 벡터.
  • 두 신호가 닮았는지 = 두 벡터 사이 코사인 유사도.

활용

  • 음성 인식: 입력 음성과 사전 단어 음성의 유사도.
  • 이미지 검색: 두 사진의 픽셀 벡터 유사도.
  • 챗봇: 질문 문장과 답변 후보 문장의 유사도.
"닮았다"는 모호한 개념을 내적 한 줄로 수치화할 수 있다는 것이 선형대수의 위력.

📝 정리 — 오늘의 핵심

1️⃣ 길이 (노름)

$$\|\mathbf{v}\| = \sqrt{v_1^2 + \cdots + v_n^2} = \sqrt{\mathbf{v}\cdot\mathbf{v}}$$

2️⃣ 내적 (두 가지 정의, 같은 값)

$$\mathbf{a}\cdot\mathbf{b} = a_1 b_1 + \cdots + a_n b_n = \|\mathbf{a}\|\|\mathbf{b}\|\cos\theta$$

3️⃣ 각도

$$\cos\theta = \frac{\mathbf{a}\cdot\mathbf{b}}{\|\mathbf{a}\|\,\|\mathbf{b}\|}$$

4️⃣ 직교 판정

$$\mathbf{a} \perp \mathbf{b} \iff \mathbf{a}\cdot\mathbf{b} = 0$$

다음 강의에서는 외적(3D 전용) 으로 평면의 법선벡터·평행사변형 면적을 구하고, 직선·평면의 방정식을 다룹니다.

🌍 응용 (4) — 그래픽스에서의 조명 계산

램버트 모델 (Lambertian shading)

3D 모델 표면이 빛을 받았을 때 얼마나 밝게 보이는가? 표면 법선 $\mathbf{n}$ 과 빛 방향 $\mathbf{l}$ 의 내적으로 계산:

$$I = \max(0,\ \hat{\mathbf{n}} \cdot \hat{\mathbf{l}})$$

(단위벡터로 정규화한 후 내적)

해석

  • $\hat{\mathbf{n}} \cdot \hat{\mathbf{l}} = 1$: 빛이 정면에서 → 가장 밝음.
  • $\hat{\mathbf{n}} \cdot \hat{\mathbf{l}} = 0$: 빛이 표면에 평행 → 그림자.
  • $\hat{\mathbf{n}} \cdot \hat{\mathbf{l}} < 0$: 표면이 빛 반대편 → 0으로 클램핑.

왜 cosθ?

같은 양의 빛이라도 표면이 비스듬하면 더 넓은 영역에 분산되어 단위 면적당 밝기가 줄어듭니다 (코사인 법칙). 실제 게임 엔진의 셰이더 코드 한 줄이 정확히 이 내적입니다.

3D 게임의 모든 음영(shadow)·반사·하이라이트의 가장 단순한 형태가 "법선 · 빛 방향 = 내적".

🌍 응용 (5) — 임베딩 벡터와 의미 검색

단어를 벡터로 (Word Embedding)

  • 현대 NLP는 단어 하나를 100~1024차원 실수 벡터로 표현.
  • 의미가 비슷한 단어 → 가까운 벡터 (코사인 유사도 큼).
  • "왕 - 남자 + 여자 ≈ 여왕" 같은 벡터 연산이 실제 가능.

검색 시스템

  • 질문 문장 → 임베딩 벡터.
  • 답변 후보 문장들의 임베딩 벡터들과 코사인 유사도 계산.
  • 가장 유사도 높은 답변 반환.
ChatGPT 같은 LLM 서비스의 "관련 문서 검색" 기능이 본질적으로 이 한 줄 — 내적 한 번이 전부입니다.
