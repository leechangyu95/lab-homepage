Week 1. 자동제어 개요

자동제어란 무엇이고, 왜 우리에게 필요한가? 개회로(open-loop)와 폐회로(closed-loop)의 차이를 일상 예시·인터랙티브 데모로 직감부터 잡는 첫 시간.

1. 자동제어란 무엇인가?

한 줄 비유부터

샤워기에서 따뜻한 물이 나오게 하는 일을 떠올려 보세요. 손등을 물줄기에 슬쩍 대보고 "조금 차갑네" 싶으면 빨강 쪽으로 살짝, "어이쿠 너무 뜨겁다" 싶으면 다시 파랑 쪽으로. 우리는 매 순간 원하는 온도(목표)와 지금 손에 닿는 온도(측정값) 사이의 차이를 보고 수도꼭지를 조금씩 돌립니다.

이 단순한 행동이 자동제어의 본질입니다. 누군가가 또는 무엇인가가 원하는 값(목표 reference)과 현재 값(출력 output)의 차이를 줄이려고 입력(actuator)을 조절하는 일 — 이걸 기계가 사람 대신 자동으로 하게 만드는 학문이 자동제어(Automatic Control)입니다.

왜 사람이 직접 안 하고 자동으로 하나?

  • 빠르다 — 사람의 반응 시간은 0.2초 내외. 드론 회전이나 모터 위치 제어에는 너무 느립니다.
  • 정확하다 — 사람은 주의가 흐트러지지만 제어기는 매 5 ms마다 같은 일을 빠짐없이 합니다.
  • 지치지 않는다 — 24시간 돌아가는 공장 라인, 위성, 발전소를 사람이 일일이 조작할 수 없습니다.
  • 위험한 환경 — 원자로, 우주, 심해. 사람이 가지 못하는 곳에서도 시스템은 스스로 균형을 잡아야 합니다.
자주 듣는 오해 — "AI/딥러닝이 결국 다 대체하는 거 아닌가요?" 아닙니다. 학습 기반 제어가 등장한 지금도 실제로 모터를 돌리고 액추에이터를 움직이는 가장 안쪽 루프는 여전히 PID·LQR 같은 고전 제어가 담당합니다. 강화학습이 "어디로 갈까"를 정해도, "그 명령을 실제로 안정적으로 실행"하는 건 자동제어 이론입니다. 이 강의를 마치면 그 둘의 경계가 또렷해집니다.

2. 제어시스템의 구성 요소

자동제어를 처음 배울 때 가장 헷갈리는 것이 용어입니다. 사실 다섯 개만 외우면 끝납니다.

기호이름의미샤워 예시
$r(t)$기준입력 / 목표값"이 값이 됐으면" 하는 출력원하는 물 온도 40℃
$y(t)$출력 / 제어량실제 측정되는 시스템 상태지금 손에 닿는 물 온도
$e(t)=r-y$오차 (error)목표와 실제의 차이"3℃ 더 따뜻해야 함"
$u(t)$제어입력 (control input)제어기가 액추에이터에 보내는 명령"수도꼭지 빨강 쪽으로 5° 회전"
플랜트제어 대상 (plant)우리가 다루는 물리 시스템 자체보일러 + 배관

이 다섯 개만 정확히 구분해도 자동제어 책의 절반은 읽힙니다. 나머지는 "어떻게 좋은 $u$를 계산할까?"에 대한 여러 답일 뿐이에요.

3. 개회로 vs 폐회로 — 두 가지 전략

한 줄 비유부터

같은 샤워라도 두 가지 방식이 있습니다.

  • 개회로(Open-loop): "어제 빨강 쪽으로 30도 돌렸을 때 딱 좋았으니까 오늘도 그렇게 하자" 하고 손도 안 대본 채 들어갑니다. 보일러 상태가 어제와 다르거나 옆집이 뜨거운 물을 쓰는 순간이라면 그대로 화상.
  • 폐회로(Closed-loop): 손등으로 물을 만져보고 그 결과를 보면서 수도꼭지를 다시 조절합니다. 매 순간 측정해서 어긋난 만큼 보정하므로, 모델이 살짝 틀려도 외란이 와도 결국 목표에 닿습니다.

식으로 쓰면

플랜트의 동특성을 $G$, 제어기를 $C$, 외란을 $d$라고 하면 두 구조는 각각 다음과 같이 쓸 수 있습니다.

  • 개회로: $u = C\, r$, $\;\;y = G\, u + d = G\, C\, r + d$
  • 폐회로: $u = C(r - y)$, $\;\;y = \dfrac{G\, C}{1 + G\, C}\, r + \dfrac{1}{1 + G\, C}\, d$

핵심 관찰: 폐회로의 외란 항 $\frac{1}{1+GC}$는 $|GC|$가 클수록 작아집니다. 즉 충분한 게인만 가지면 외란이 출력에 미치는 영향을 자동으로 줄여줍니다. 이것이 폐회로의 마법입니다.

그럼 그냥 폐회로만 쓰면 되겠네? — 절반 맞습니다. 다만 폐회로에는 안정성이라는 새 문제가 따라옵니다. 게인을 너무 키우면 보정이 과해서 진동하다 발산할 수도 있어요. 샤워에서 너무 성격 급하게 수도꼭지를 휙휙 돌리면 뜨거움-차가움이 반복되는 것과 같습니다. 그래서 폐회로 설계의 핵심 질문은 "얼마나 세게 보정할까(게인)"가 됩니다. 이 질문에 답하는 도구가 바로 W2~W11에서 배울 라플라스·전달함수·근궤적·Bode 선도입니다.

4. 인터랙티브: 외란이 있을 때 누가 살아남는가

🎮 인터랙티브: 개회로 vs 폐회로 — 외란과 모델 오차에 누가 강한가

같은 1차 시스템 두 대(예: 가열기로 방 데우기)에 같은 목표 온도 $r$을 주었습니다. 개회로(주황)은 공칭 모델만 보고 미리 계산한 입력 $u_{ff} = r/\hat{K}$을 그대로 보냅니다. 폐회로(파랑)은 측정된 출력으로 $u = K_p(r-y)$를 매 순간 갱신합니다. 외란(창문 열림)과 모델 게인 오차(난방기 효율 변화)를 흔들어 보세요.

꼭 만져볼 시나리오:

  • 외란 d ≠ 0 (창문 살짝 열림): 개루프는 영구적인 잔류 오차가 생깁니다. 폐루프는 입력을 스스로 늘려서 다시 끌어옵니다.
  • 모델 오차 ΔK ≠ 0 (난방기 효율이 가정과 다름): 개루프는 엉뚱한 위치에 정착, 폐루프는 실제 플랜트가 어떻든 목표에 수렴합니다.
  • Kp를 너무 키우면?: 폐루프도 빠르게 따라가지만, 시스템이 더 복잡해지면 진동이 나타납니다. "어디까지 키워도 되는가"가 W8~W11의 핵심 질문입니다.
  • 둘 다 0: 개루프도 그럭저럭 도달합니다. 모델이 완벽하고 외란이 없을 때만 개루프가 통한다는 뜻이에요.

7. 코드로 만나기 — 첫 번째 시뮬레이션

위 인터랙티브 데모와 똑같은 1차 시스템 τẏ + y = Ku + d를 Python으로 직접 시뮬레이션해 보세요. numpy와 matplotlib만 있으면 됩니다.

import numpy as np
import matplotlib.pyplot as plt

# 1차 플랜트:  tau * dy/dt + y = K * u + d
tau, K_act = 2.0, 1.0       # 실제 플랜트
K_nom      = 1.0             # 개루프가 가정하는 모델
Kp         = 3.0             # 폐루프 P 게인
dt, T      = 0.04, 12.0
N          = int(T/dt)

t  = np.arange(N) * dt
r  = np.where(t < 6.0, 3.0, -3.0)         # 6초 후 목표 반전
d  = np.where((t > 3.0) & (t < 9.0), 1.0, 0.0)  # 외란 펄스

y_open, y_closed = 0.0, 0.0
log_open, log_closed = [], []

for k in range(N):
    # 개루프: 모델 역연산 기반 피드포워드
    u_open   = r[k] / K_nom
    # 폐루프: P 제어
    u_closed = Kp * (r[k] - y_closed)
    # 1차 플랜트 적분 (오일러)
    y_open   += dt * ((K_act*u_open   + d[k] - y_open)   / tau)
    y_closed += dt * ((K_act*u_closed + d[k] - y_closed) / tau)
    log_open.append(y_open); log_closed.append(y_closed)

plt.plot(t, r, 'r--', label='reference')
plt.plot(t, log_open,   label='open-loop')
plt.plot(t, log_closed, label='closed-loop (P)')
plt.xlabel('time [s]'); plt.ylabel('output y'); plt.legend(); plt.grid()
plt.show()

한 줄씩 바꿔보면서 직감을 키워 보세요. K_act를 1.5로 바꾸면 개루프만 어긋나고, Kp를 50으로 키우면 폐루프가 진동을 시작합니다. 이 진동이 바로 W8 안정성 분석으로 이어집니다.

8. 핵심 정리

  • 자동제어 = 측정 → 오차 → 입력 갱신을 자동으로 반복하는 일.
  • 개회로는 모델이 완벽할 때만 통하고, 폐회로는 외란·모델 오차에 강하다.
  • 폐회로의 무기는 측정과 게인. 게인이 클수록 외란 억제력은 커지지만 안정성은 위험해진다.
  • W2부터는 "얼마나 큰 게인까지 안정한가"를 정확히 계산하는 수학 도구를 쌓는다.
