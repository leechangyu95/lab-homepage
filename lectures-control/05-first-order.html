Week 5. 1차 시스템 시간응답

시정수 τ 하나로 모든 게 결정되는 가장 단순한 동적 시스템. RC 필터·온도·1차 모터 — 자동제어에서 가장 자주 만나는 형태입니다.

목차

  1. 1차 시스템의 표준형
  2. 계단응답과 63% 규칙
  3. 임펄스응답
  4. 시정수와 5τ 규칙
  5. 인터랙티브: τ와 K 슬라이더

1. 1차 시스템의 표준형

1차 시스템은 미분방정식이 1차인 시스템입니다.

$$ \tau\, \dot{y}(t) + y(t) = K\, u(t) $$

라플라스 변환:

$$ \boxed{\, G(s) = \frac{Y(s)}{U(s)} = \frac{K}{\tau s + 1} \,} $$
  • $K$ (DC 게인): 입력이 1로 영원히 유지될 때 출력이 정착하는 값.
  • $\tau$ (시정수, time constant): 시스템이 얼마나 빠르게 반응하는지의 척도. 단위는 시간(초).
  • 극점: $\tau s + 1 = 0 \;\Rightarrow\; s = -1/\tau$ (실수, 좌반평면) → 항상 안정.

2. 계단응답과 63% 규칙

$u(t)$가 단위 스텝(0 → 1)일 때 W2 예제에서 본 결과:

$$ \boxed{\, y(t) = K\bigl(1 - e^{-t/\tau}\bigr) \,} $$
$t$$y(t)/K$해석
$\tau$$1 - e^{-1} \approx 0.632$63.2% ← 시정수의 정의
$2\tau$$\approx 0.865$86.5%
$3\tau$$\approx 0.950$95.0%
$4\tau$$\approx 0.982$98.2%
$5\tau$$\approx 0.993$99.3% — 실무에서 "정착"으로 본다
왜 63%일까? $\tau$의 정의 자체가 "응답이 최종값의 $1 - 1/e \approx 63.2\%$에 도달하는 시간"입니다. 이 사실 한 줄이면 측정된 응답 곡선에서 자를 대고 시정수를 바로 읽을 수 있습니다 — 6절 시스템 식별의 핵심.

3. 임펄스응답

$u(t) = \delta(t)$일 때 (수학적 임펄스, 짧고 강한 자극):

$$ y(t) = \frac{K}{\tau} e^{-t/\tau} $$

$t=0$에서 갑자기 $K/\tau$로 뛰었다가 시정수 $\tau$로 지수 감쇠합니다. 자연 응답(natural response) 모드를 순수하게 보여주는 시험이지만, 실제 시스템에 진짜 임펄스를 가하기는 어려워서 보통은 계단응답으로 대체합니다.

4. 시정수와 5τ 규칙

실무 한 문장: "1차 시스템은 5τ 정도면 정착했다고 본다." 이 한 줄 때문에 모터·필터·온도제어를 설계할 때 "필요한 응답 시간 = 5τ" 식으로 거꾸로 시정수를 정합니다.

예: 0.5초 안에 응답을 정착시키고 싶다 → $\tau \leq 0.1$초가 되어야 함 → 그에 맞춰 회로의 RC 값이나 제어기의 게인을 결정.

5. 인터랙티브: τ와 K 슬라이더

🎮 인터랙티브: 1차 시스템의 계단응답 만져보기

$\tau$와 $K$를 바꿔보면서 응답이 어떻게 변하는지 관찰하세요. 파란 점은 $t = \tau$, $2\tau$, $3\tau$, $5\tau$ 위치.

관찰:

  • $\tau$를 키우면 응답이 느려진다 (정착까지 더 오래).
  • $K$는 최종 정착 값만 바꾼다 — 응답의 모양(시간 척도)은 건드리지 않는다.
  • $t = \tau$에서 항상 $0.632\,K$를 지나간다. 자로 그 점을 잡으면 미지의 시스템에서도 $\tau$를 측정할 수 있다.

6. 실험으로 시스템 식별 (System ID)

실제 모터·온도제어기·드론 데이터를 받았을 때 1차 모델로 근사하는 가장 빠른 방법:

  1. 입력에 단위 스텝을 가한다 (예: 모터에 일정 전압 5V 인가).
  2. 출력이 정착하는 값을 측정 → $K = y_{\infty} / u_{\infty}$.
  3. 출력이 $0.632\, K$가 되는 시간을 측정 → 그게 $\tau$.

이걸 그래프 한 장에서 자로 잡으면 $G(s) = K/(\tau s + 1)$ 모델 완성. 공정 제어 분야에서 60년 넘게 쓰이고 있는 클래식한 방법입니다 (Cohen-Coon, IMC 등의 출발점).

import numpy as np
import matplotlib.pyplot as plt

# 임의의 측정 데이터 (예: 모터 속도 응답)
t = np.linspace(0, 10, 200)
y_meas = 3.0 * (1 - np.exp(-t/1.5)) + 0.05*np.random.randn(len(t))  # K=3, τ=1.5

# 1) DC 게인 추정: 마지막 N 샘플 평균
K_hat = y_meas[-30:].mean()
# 2) 0.632*K 지점 찾기
idx = np.argmax(y_meas >= 0.632 * K_hat)
tau_hat = t[idx]

print(f'추정값:  K ≈ {K_hat:.2f},  τ ≈ {tau_hat:.2f} s')

# 모델 응답과 비교
y_model = K_hat * (1 - np.exp(-t/tau_hat))
plt.plot(t, y_meas, '.', label='측정', alpha=.5)
plt.plot(t, y_model, 'r-', label=f'모델 K={K_hat:.2f}, τ={tau_hat:.2f}')
plt.axhline(0.632*K_hat, ls='--', color='gray')
plt.xlabel('t [s]'); plt.ylabel('y'); plt.legend(); plt.grid(); plt.show()

7. 핵심 정리

  • 1차 시스템: $G(s) = K/(\tau s + 1)$, 극점 하나가 $-1/\tau$. 항상 안정.
  • 계단응답 $y(t) = K(1 - e^{-t/\tau})$, $t = \tau$에서 63.2%.
  • 5τ ≈ 99% 정착 — "1차 시스템 정착 시간"의 실무 기준.
  • 측정된 응답에서 $K$와 τ를 바로 읽을 수 있다 → 가장 단순한 시스템 식별.
