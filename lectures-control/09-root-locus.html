Week 9. 근궤적 (Root Locus)

제어기 게인 K가 0에서 무한대까지 움직일 때 폐루프 극점이 s-평면에서 그리는 궤적. "K를 얼마나 키우면 진동 시작?", "어떤 ζ에 도달할 수 있나?"를 한 그림으로 답해줍니다.

1. 왜 근궤적이 필요한가

W8에서 Routh로 "특정 K에서 안정한가"는 판정할 수 있었습니다. 그런데 실제 설계에서는 더 풍부한 질문이 옵니다.

  • K를 키우면 폐루프 극점이 어디로 움직이는가?
  • K를 어디까지 올려야 ζ = 0.7 (오버슈트 ~5%)에 도달하는가?
  • 특정 K에서 자연주파수 ωₙ은 얼마인가?

근궤적은 "K가 0 → ∞"라는 한 매개변수의 변화를 한 장의 그림으로 보여줍니다. W. R. Evans가 1948년에 제안한, 컴퓨터가 없던 시대의 시각적 설계 도구입니다.

2. 크기·각도 조건

특성방정식 $1 + K G(s)H(s) = 0$, 즉 $K G(s)H(s) = -1$. 복소수 등식이므로 두 조건으로 나뉩니다.

  • 각도 조건: $\angle G(s)H(s) = \pm 180^\circ \cdot (2k+1)$ — 이게 궤적의 모양을 결정
  • 크기 조건: $K = 1/|G(s)H(s)|$ — 궤적 위 점에 해당하는 K 값을 알려줌

각도 조건만으로 궤적이 그려지고, 크기 조건은 그 위 점에 K 라벨을 다는 데 쓰인다고 생각하면 됩니다.

3. 작도 규칙 7가지

$K G(s)H(s) = K \dfrac{N(s)}{D(s)}$, 영점 $m$개·극점 $n$개라 합시다 ($n \geq m$).

  1. 출발과 종착: 궤적은 $K = 0$일 때 개루프 극점에서 시작해, $K \to \infty$일 때 개루프 영점 또는 무한대로 간다.
  2. 가지 수: 가지 수 = $\max(n, m)$ = 일반적으로 $n$.
  3. 실축 부분: 실축 위 한 점이 궤적에 속하려면 그 점의 오른쪽에 있는 실축 위 극점·영점의 개수의 합이 홀수여야 한다.
  4. 점근선: 무한대로 가는 가지의 점근선은 $$ \sigma_a = \frac{\sum p_i - \sum z_i}{n - m}, \quad \theta_a = \frac{(2k+1) \cdot 180^\circ}{n - m} $$ $\sigma_a$는 점근선이 만나는 실축의 점, $\theta_a$는 각도.
  5. 실축 이탈/진입점: $dK/ds = 0$의 실수 해. 두 극점 사이에서 궤적이 갈라지거나 합쳐지는 지점.
  6. 허수축 교차점: 특성방정식에 $s = j\omega$를 대입해 풀면 임계 K와 ω를 얻는다 (W8 Routh 경계와 동일).
  7. 출발 각도 (복소 극점에서): $\theta_d = 180^\circ - \sum \angle (p_d - p_i) + \sum \angle (p_d - z_i)$.
실무 팁 — 위 규칙을 모두 외울 필요는 없습니다. 1~4번이 80%를 그리고, 5~7번은 정밀도를 높일 때만. Python control.root_locus(G)면 한 줄로 그려주니, 시험을 위해서가 아니라면 손계산은 1~4번까지면 충분합니다.

4. 예제 — 손으로 그려보기

$G(s)H(s) = \dfrac{1}{s(s+1)(s+5)}$ (W8 예제 동일).

  1. 극점 $\{0, -1, -5\}$, 영점 없음 → $n=3, m=0$.
  2. 가지 수 3개. 모두 무한대로 발산.
  3. 실축 부분: $-1 < s < 0$의 오른쪽엔 극점 1개(홀수 ✓), $-5 < s < -1$의 오른쪽엔 2개(짝수 ✗), $s < -5$의 오른쪽엔 3개(홀수 ✓). 따라서 실축 위 궤적은 $[-1, 0]$과 $(-\infty, -5]$.
  4. 점근선 중심 $\sigma_a = (0 + (-1) + (-5)) / 3 = -2$, 각도 $\theta_a = 60^\circ, 180^\circ, 300^\circ$.
  5. $dK/ds = 0$ 풀면 이탈점 $s \approx -0.47$. 두 극점 0과 −1 사이에서 갈라져 60°, 300° 점근선을 따라 무한대로.
  6. 허수축 교차: $s = j\omega$ 대입 후 실·허부 분리하면 $\omega = \sqrt{5}$, $K = 30$ (W8과 일치).

5. 인터랙티브: 게인 K 슬라이더로 궤적 따라가기

🎮 인터랙티브: $1/[s(s+1)(s+5)]$의 근궤적

회색 곡선이 K가 0 → 60일 때 그리는 전체 궤적, 빨간 X가 현재 K에서의 폐루프 극점. 슬라이더로 K를 움직이면서 극점이 좌반평면 → 허수축 → 우반평면으로 옮겨가는 모습을 보세요.

관찰:

  • K 작을 때 (예: 0.5): 세 극점이 거의 개루프 극점 자리에서 시작.
  • K가 커지면 두 극점이 갈라져서 점근선을 따라 위·아래 방향으로 이동.
  • K = 30에서 두 극점이 정확히 허수축 위 (±j√5).
  • K > 30에서 우반평면으로 → 발산.

6. 설계에 활용하기

설계 명세 → s-평면 영역 → 근궤적과의 교차점 → K 결정 흐름:

  1. "오버슈트 ≤ 5%, 정착 ≤ 1초" → ζ ≥ 0.7, $\zeta\omega_n \geq 4$.
  2. s-평면에서 ζ ≥ 0.7 (각도 ≤ 45°)과 $\sigma \leq -4$를 만족하는 부채꼴 영역을 그린다.
  3. 근궤적이 그 영역을 지나는 점에서 K를 결정 (크기 조건 $K = 1/|G(s)|$).

이게 가능한 시스템도 있고 불가능한 시스템도 있습니다. 후자라면 P 단독으로는 명세를 못 맞추니 PD나 PID로 바꾸거나(W12), Lead/Lag 보상기로 영점을 추가해서 궤적의 모양 자체를 바꿔야 합니다.

import numpy as np
import matplotlib.pyplot as plt
import control as ctl

# G(s) = 1 / [s(s+1)(s+5)]
G = ctl.tf([1], [1, 6, 5, 0])
ctl.root_locus(G, kvect=np.linspace(0.01, 60, 300), grid=True)
plt.title('근궤적:  K = 0 → 60')
plt.show()

# 특정 K에서 폐루프 극점 확인
for K in [5, 20, 30, 40]:
    cl = ctl.feedback(K * G, 1)
    poles = ctl.poles(cl)
    print(f'K={K:2d}  poles = {np.round(poles, 3)}')

7. 핵심 정리

  • 근궤적 = K가 변할 때 폐루프 극점이 그리는 길.
  • 각도 조건이 모양을, 크기 조건이 K 값을 결정.
  • 가장 자주 쓰는 작도 규칙 4가지: 출발/종착, 가지 수, 실축 위 부분, 점근선.
  • 설계 명세 → s-평면 영역 → 궤적과의 교차점 → K 결정.
  • 실무에선 손계산보다 control.root_locus() 한 줄.
