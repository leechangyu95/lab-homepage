Week 8. 안정성 — Routh-Hurwitz

고차 다항식의 근을 구하지 않고도 "모든 근이 좌반평면에 있는가?"를 판정하는 19세기의 영리한 알고리즘. 매개변수 $K$의 안정 영역을 손으로 찾는 가장 빠른 방법입니다.

목차

  1. BIBO 안정성과 극점 위치
  2. 필요조건 — 모든 계수의 부호가 같아야 한다
  3. Routh 표 만들기
  4. 예제: 3차·4차 시스템
  5. 매개변수 안정 영역 (가장 실용적인 사용)
  6. 인터랙티브: K를 돌려서 안정 경계 찾기

1. BIBO 안정성과 극점 위치

BIBO 안정 (Bounded-Input Bounded-Output): 유한한 입력에 대해 출력이 유한히 머무르는 시스템. LTI 시스템의 경우, 다음과 동치입니다.

$$ \text{시스템이 안정} \iff \text{전달함수의 모든 극점이 좌반평면(Re < 0)에 있음} $$

그런데 시스템이 4차, 5차, 그 이상이 되면 극점을 손으로 풀기가 어렵습니다. Routh-Hurwitz 판별법은 다항식의 계수만 보고도 "양의 실수부 근의 개수"를 알려줍니다.

2. 필요조건 — 부호 일치

특성다항식 $a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0 = 0$이 안정하려면:

  1. 모든 계수가 존재해야 한다 (중간에 0이 없어야 함)
  2. 모든 계수의 부호가 같아야 한다 (모두 양수 또는 모두 음수)

이 두 가지 중 하나라도 위반되면 그 시스템은 즉시 불안정. 충분조건은 아니라서 통과해도 다음 단계(Routh 표)를 거쳐야 합니다.

3. Routh 표 만들기

특성다항식 $s^n$의 계수부터 표를 만듭니다. 예: $s^4 + 3s^3 + 5s^2 + 4s + 2 = 0$

$s^4$152
$s^3$340
$s^2$$b_1$$b_2$
$s^1$$c_1$
$s^0$$d_1$

아래 행은 위 두 행에서 다음 공식으로 계산합니다.

$$ b_1 = \frac{(3)(5) - (1)(4)}{3} = \frac{11}{3}, \quad b_2 = \frac{(3)(2) - (1)(0)}{3} = 2 $$ $$ c_1 = \frac{(b_1)(4) - (3)(b_2)}{b_1} = \frac{(11/3)(4) - 6}{11/3} = \frac{44/3 - 6}{11/3} = \frac{26/3}{11/3} = \frac{26}{11} $$ $$ d_1 = \frac{(c_1)(b_2) - (b_1)(0)}{c_1} = b_2 = 2 $$

완성된 표:

$s^4$152
$s^3$34
$s^2$11/32
$s^1$26/11
$s^0$2

판정 규칙: 1열의 부호 변화 횟수 = 우반평면 근의 개수. 위에서 1열은 1, 3, 11/3, 26/11, 2 → 모두 양수 → 부호 변화 0 → 안정.

특수 상황 두 가지 —
  • 1열에 0이 나오면: 그 자리를 작은 양수 ε로 두고 계산을 이어간 뒤 ε → 0의 부호를 본다.
  • 한 행 전체가 0이면: 그 위 행으로 보조다항식 $A(s)$를 만들어 미분 $dA/ds$ 계수로 그 행을 채운다 (허수축에 극점이 있음 — 경계 안정).

4. 예제: 폐루프 안정성

플랜트 $G(s) = \dfrac{1}{s(s+1)(s+5)}$, 비례 제어기 $C(s) = K$, 단위 피드백.

특성방정식 $1 + KG(s) = 0$ → $s(s+1)(s+5) + K = 0$ → $s^3 + 6s^2 + 5s + K = 0$.

$s^3$15
$s^2$6K
$s^1$(30 − K)/6
$s^0$K

안정 조건 (1열이 모두 양수):

$$ K > 0 \quad \text{and} \quad \frac{30 - K}{6} > 0 \;\Rightarrow\; K < 30 $$ $$ \boxed{\, 0 < K < 30 \,} $$

$K = 30$이 정확히 안정 경계. 이때 보조다항식 $6s^2 + 30 = 0$에서 진동 주파수 $\omega = \sqrt{30/6} = \sqrt{5} \approx 2.24$ rad/s를 바로 읽을 수 있습니다 — Ziegler-Nichols 튜닝(W12)의 임계 진동수가 바로 이것입니다.

5. 매개변수 안정 영역

현실에서 Routh-Hurwitz의 가장 실용적인 쓰임은 "이 시스템이 안정한 게인 K의 범위는?"입니다. 특성방정식의 한 계수에 K를 두고 표를 만들어 1열 양수 조건을 풀면 그게 답입니다.

두 변수($K_p$, $K_i$ 등)가 동시에 들어가면 평면 위 영역으로 표현됩니다. 이게 PID 튜닝의 '안정한 영역에서만 게인을 골라야 한다'는 안전망 역할을 합니다.

6. 인터랙티브: K를 돌려서 안정 경계 찾기

🎮 인터랙티브: 위 예제 시스템에 K를 가변

플랜트 $G(s) = 1/[s(s+1)(s+5)]$, P 제어기 K, 단위 피드백. K를 움직이면서 폐루프 극점이 s-평면에서 어디로 가는지, 시간응답이 어떻게 변하는지 관찰하세요. Routh가 예측한 임계값 K = 30에서 정확히 진동 시작.

꼭 확인할 것:

  • K = 5: 안정, 약간 진동하면서 수렴.
  • K = 25: 여전히 안정, 그러나 진동이 심해짐.
  • K = 30: 정확히 경계 — 극점이 허수축 위에 → 지속 진동.
  • K > 30: 우반평면 극점 → 발산.

7. Python으로 자동화

import numpy as np
from numpy.polynomial.polynomial import Polynomial

def routh(coeffs):
    """coeffs: [a_n, a_{n-1}, ..., a_0]  →  Routh table 1열만 반환"""
    n = len(coeffs) - 1
    rows = (n + 1) // 2
    table = np.zeros((n + 1, rows + 1))
    table[0, :len(coeffs[0::2])] = coeffs[0::2]
    table[1, :len(coeffs[1::2])] = coeffs[1::2]
    for i in range(2, n + 1):
        for j in range(rows):
            a = table[i-2, 0]; b = table[i-1, 0]
            if b == 0: b = 1e-9
            table[i, j] = (b * table[i-2, j+1] - a * table[i-1, j+1]) / b
    return table[:, 0]

# 예: s^3 + 6 s^2 + 5 s + K = 0,   K를 휘저으면서 안정 K 범위 찾기
for K in [1, 10, 29, 30, 31, 50]:
    col1 = routh([1, 6, 5, K])
    sign_changes = np.sum(np.diff(np.sign(col1)) != 0)
    print(f'K={K:3d}  1열={np.round(col1,3)}  부호변화={sign_changes}  → '
          + ('안정' if sign_changes==0 else f'불안정 (우반평면 근 {sign_changes}개)'))

8. 핵심 정리

  • 안정 = 모든 폐루프 극점이 좌반평면에 있는 것.
  • Routh-Hurwitz: 다항식의 근을 직접 구하지 않고도 "우반평면 근의 개수"를 1열 부호 변화로 알 수 있다.
  • 실용 용도: 매개변수 K(또는 PID 게인)의 안정 영역을 손으로 찾을 때.
  • 경계점에서 보조다항식으로 임계 진동 주파수 ωu도 함께 얻는다 → Ziegler-Nichols 튜닝의 출발점.
