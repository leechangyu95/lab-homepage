Week 11. 안정여유 & 강인성

"지금은 안정하지만, 모델이 살짝 틀려도 안정할까?" 이득여유(GM)와 위상여유(PM)는 그 여유분을 한 숫자로 표현한 강인성의 척도입니다.

1. 왜 "여유"가 필요한가

안정성(W8)은 흑백 판정입니다 — 안정 또는 불안정. 그런데 현실에서는 위험합니다.

  • 플랜트 모델이 부정확해서 실제 게인이 가정보다 1.5배 클 수 있다.
  • 액추에이터·센서에 시간 지연(deadtime)이 있어 모델보다 위상이 더 늦을 수 있다.
  • 부품이 닳아 시정수가 변할 수 있다.

모델 위에서 "딱 안정 경계"에 설계해 두면 이런 변동 한 번에 시스템이 발산합니다. 모델 오차에 얼마나 견디는가를 정량화한 게 안정여유입니다.

2. 이득여유 GM과 위상여유 PM의 정의

한 줄 직관 — 왜 두 개를 모두 보는가?

폐루프가 발산하는 조건은 단 하나, $L(j\omega) = -1$입니다(나이퀴스트 점). 복소수 −1은 크기 1, 위상 −180°인 점이라서 두 가지 다른 경로로 거기에 도달할 수 있습니다.

  • 위상은 그대로인데 크기만 커지다 보면 크기가 1(즉 0 dB)에 닿는 순간 발산. 이때 "내 위상은 이미 −180°였는가?"를 봐서, 위상이 이미 −180°에 닿은 주파수($\omega_{pc}$)에서 크기가 0 dB까지 얼마 남았는지를 잰 것이 이득여유 GM.
  • 크기는 그대로인데 위상이 더 늦다 보면 위상이 −180°에 닿는 순간 발산. 이때 "내 크기는 이미 1이었는가?"를 봐서, 크기가 이미 0 dB인 주파수($\omega_{gc}$)에서 위상이 −180°까지 얼마 남았는지를 잰 것이 위상여유 PM.

현실의 모델 오차는 두 가지가 섞여 들어옵니다 — 부품 게인이 ±20% 어긋날 수도 있고, 액추에이터·필터·통신지연이 위상을 더 늦출 수도 있습니다. 그래서 두 방향 모두의 여유가 필요합니다.

개루프 전달함수 $L(s) = G(s)C(s)$의 Bode 선도에서:

  • 이득여유 (Gain Margin, GM): 위상이 −180°를 가로지르는 주파수 $\omega_{pc}$에서 크기가 0 dB까지 얼마나 남았는가. $$ \text{GM} = -|L(j\omega_{pc})|_{dB} \text{ (양수면 안정)} $$ "현재 게인을 몇 dB까지 더 올리면 발산하는가?"의 답.
  • 위상여유 (Phase Margin, PM): 크기가 0 dB을 가로지르는 주파수 $\omega_{gc}$에서 위상이 −180°까지 얼마나 남았는가. $$ \text{PM} = 180^\circ + \angle L(j\omega_{gc}) \text{ (양수면 안정)} $$ "현재 시스템에 위상 지연을 몇 도까지 더 추가해도 안정한가?"의 답.

둘 다 양수일 때 폐루프가 안정합니다. 0이면 안정 경계, 음수면 불안정.

📐 PM이 시간 지연으로 환산하면 얼마인가?

시간 지연 $T_d$의 위상 효과는 $-\omega T_d$ rad, 즉 $-\omega T_d \cdot 180/\pi$ 도. PM[°]만큼의 위상을 까먹으면 발산하니까, 견딜 수 있는 최대 지연은: $$ T_{d,\max} = \frac{\text{PM}[\text{rad}]}{\omega_{gc}} = \frac{\text{PM}[\degree]}{\omega_{gc}} \cdot \frac{\pi}{180} $$

예: PM = 60°, $\omega_{gc} = 5$ rad/s → $T_{d,\max} \approx 0.21$ s. 통신지연이 0.21초를 넘는 시스템은 발산. 분산 제어·원격조종에서 매우 중요한 숫자.

4. 실무 기준값

지표안전한 설계최소 권장
이득여유 GM≥ 6 dB (= 게인 2배 여유)≥ 3 dB
위상여유 PM≥ 45°≥ 30°

둘 다 너무 크면 응답이 느려지고, 너무 작으면 모델 오차에 취약. 보통 실무에서는 PM 45~60°를 목표로 합니다.

왜 PM이 GM보다 더 중요하게 다뤄지나? 시간 지연(deadtime), 액추에이터의 위상 지연, 센서 필터 등은 모두 위상에 영향을 줍니다. 그래서 실무에서는 위상여유가 더 자주 의사결정의 기준이 됩니다. GM은 "혹시 게인을 잘못 추정했을 때"의 안전망 정도로 보면 됩니다.

5. 인터랙티브: 게인 K 변화 → 마진 추적

🎮 인터랙티브: $L(s) = K/[s(s+1)(s+5)]$의 마진

K를 키우면 크기 곡선이 위로 평행 이동 → ωgc가 오른쪽으로 → PM 감소. K = 30이 W8에서 본 안정 경계 (PM = 0).

관찰:

  • K = 5: PM ≈ 50°, GM ≈ 16 dB. 안전한 설계.
  • K = 15: PM ≈ 25°, GM ≈ 6 dB. 경계 근처.
  • K = 30: PM ≈ 0°, GM ≈ 0 dB. 안정 경계 (Routh가 예측한 값과 정확히 일치).
  • K > 30: 둘 다 음수 → 불안정.

6. 위상여유와 감쇠비의 관계

2차 시스템의 경우 PM과 ζ 사이에 유용한 근사식이 있습니다.

$$ \zeta \approx \frac{\text{PM} [^\circ]}{100} \quad (\text{PM} \lesssim 60^\circ \text{에서 좋은 근사}) $$

예: PM = 45° → ζ ≈ 0.45 → 오버슈트 약 20%. PM = 60° → ζ ≈ 0.6 → 오버슈트 약 9%. Bode를 보면 시간응답을 짐작할 수 있다는 의미.

📐 왜 100ζ인가? — 표준 2차 시스템에서의 짧은 유도

표준 2차 개루프 $L(s) = \dfrac{\omega_n^2}{s(s + 2\zeta\omega_n)}$로 단위 피드백을 걸면 폐루프가 정확히 표준형이 됩니다. 이 $L$의 PM을 따져 봅니다.

  1. 이득 교차주파수: $|L(j\omega_{gc})| = 1$ 조건에서 $$ \omega_{gc}^2 = \omega_n^2\sqrt{4\zeta^4 + 1} - 2\zeta^2\omega_n^2 $$ 복잡해 보이지만 $\omega_{gc}/\omega_n \equiv u$로 두면 $u = \sqrt{\sqrt{4\zeta^4+1} - 2\zeta^2}$.
  2. 위상 계산: $\angle L(j\omega_{gc}) = -90° - \arctan(u/(2\zeta))$. 따라서 $$ \text{PM} = 90° - \arctan\!\bigl(u/(2\zeta)\bigr) = \arctan\!\bigl(2\zeta/u\bigr) $$
  3. 작은 ζ에 대한 전개: $\zeta$가 작으면 $u \approx 1$이라 $\text{PM} \approx \arctan(2\zeta)$. 라디안에서 $\arctan(2\zeta) \approx 2\zeta - (2\zeta)^3/3 + \cdots$ 이를 도(°)로 바꾸면 $\text{PM}[°] \approx (2\zeta)\cdot(180/\pi) \approx 114.6\,\zeta$.
  4. $\zeta < 0.6$ 영역에서 보정해 평균을 잡으면 표 값이 100ζ에 가까워집니다.

유효 범위 — PM ≲ 60° (즉 ζ ≲ 0.6)에서만 잘 맞고, PM이 더 커지면 실제 ζ는 더 빨리 1로 다가갑니다. 표로 보면:

PM10°30°45°60°70°
실제 ζ0.090.290.420.610.76
근사 PM/1000.100.300.450.600.70

또한 이 식은 2차 표준형에 한정. 시스템에 영점·시간지연·고차극이 있으면 PM과 ζ의 관계가 살짝 달라집니다. 그래도 "PM ≈ 100ζ"는 첫 어림에 매우 유용한 손어림입니다.

7. Python으로 마진 계산

import numpy as np
import matplotlib.pyplot as plt
import control as ctl

# 같은 시스템:  L(s) = K / [ s(s+1)(s+5) ]
for K in [5, 15, 30, 50]:
    L = ctl.tf([K], [1, 6, 5, 0])
    gm, pm, wpc, wgc = ctl.margin(L)
    print(f'K={K:2d}  GM={20*np.log10(gm):5.2f} dB  PM={pm:6.2f}°  '
          f'ωgc={wgc:.2f}  ωpc={wpc:.2f}')

# Bode + 마진 자동 표시
ctl.bode(ctl.tf([10], [1, 6, 5, 0]), dB=True, margins=True)
plt.show()

8. 핵심 정리

  • 안정여유 = 모델 오차에 견디는 강인성의 척도. 둘 다 양수일 때 안정.
  • GM: "게인을 몇 dB까지 더 올리면 발산?", PM: "위상이 몇 도 더 늦으면 발산?"
  • 실무 권장값 GM ≥ 6 dB, PM ≥ 45°. PM이 더 자주 결정 기준.
  • PM ≈ 100ζ — Bode에서 시간응답 감쇠비 짐작 가능.
