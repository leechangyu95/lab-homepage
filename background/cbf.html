3. 제어 장벽 함수 (Control Barrier Functions)

CBF는 시스템이 안전 집합 안에 영원히 머물도록 보장하는 수학적 도구입니다. "안전성을 부등식 제약 한 줄로 표현하고, 그것을 매 순간 만족시키도록 입력을 살짝 수정한다"는 것이 핵심 아이디어입니다.

목차

  1. 안전 집합과 불변성
  2. CBF의 정의
  3. CBF-QP: 안전 필터
  4. 고차 CBF (HOCBF)
  5. MPC와의 결합
  6. 코드 예제

3.1 안전 집합과 전방 불변성

안전 영역을 함수 $h(x)$ 의 0-superlevel set 으로 정의합니다.

$$\mathcal{C} = \{ x \in \mathbb{R}^n \mid h(x) \ge 0 \}$$

예: 장애물과의 거리가 $r$ 이상이어야 한다면 $h(x) = \|x - x_{\text{obs}}\|^2 - r^2$.

안전성 보장 = $\mathcal{C}$ 가 전방 불변(forward invariant) = 한 번 안전하면 영원히 안전.

3.2 CBF의 정의

제어 시스템 $\dot{x} = f(x) + g(x) u$ 에 대해, $h$ 가 제어 장벽 함수라는 것은 다음 조건을 만족하는 입력 $u$ 가 존재한다는 뜻입니다.

$$\underbrace{L_f h(x)}_{\partial h/\partial x \cdot f} + \underbrace{L_g h(x)}_{\partial h/\partial x \cdot g}\, u + \alpha(h(x)) \ge 0$$

여기서 $\alpha(\cdot)$ 는 클래스-$\mathcal{K}$ 함수 (대표적으로 $\alpha(h)=\gamma h,\ \gamma>0$).

직관: $\dot{h} \ge -\alpha(h)$. 즉 $h$ 가 0에 가까워질수록(위험할수록) $h$ 의 감소율이 강하게 제한되어, 절대 0 아래로 내려가지 못합니다.

3.3 CBF-QP: 안전 필터

이미 어떤 명목 제어기 $u_{\text{nom}}(x)$ 가 있다고 합시다 (예: PID, LQR, 강화학습 정책). CBF는 이것을 최소한으로 수정해서 안전하게 만듭니다.

$$ \begin{aligned} u^* = \arg\min_{u}\quad & \tfrac{1}{2}\|u - u_{\text{nom}}(x)\|^2 \\ \text{s.t.}\quad & L_f h(x) + L_g h(x) u + \alpha(h(x)) \ge 0 \\ & u \in \mathcal{U} \end{aligned} $$

이는 입력에 대해 선형 제약을 가진 QP 이므로 매우 빠르게(수십 마이크로초) 풀립니다.

3.4 고차 CBF (High-Order CBF)

$h$ 의 1차 미분에 입력 $u$ 가 직접 나타나지 않으면(상대 차수가 2 이상), 고차 CBF를 사용합니다. 예를 들어 위치만으로 정의된 $h$ 와 가속도 입력 차량의 경우

$$\ddot{h} + \alpha_1 \dot{h} + \alpha_2 h \ge 0$$

3.5 MPC + CBF

우리 연구실이 자주 사용하는 구조:

  • MPC: 장기 성능과 추종을 담당 (예: 차선 추종, 속도 프로파일).
  • CBF: 매 스텝 안전성을 수학적으로 보장 (예: 충돌 회피).

MPC의 각 예측 스텝에 CBF 부등식을 제약으로 추가하거나, MPC 출력을 CBF-QP로 후처리하는 방식이 일반적입니다.

3.6 코드 예제: CBF 안전 필터

import numpy as np
import cvxpy as cp

# 2D 단적분기 로봇: dx = u
x_obs = np.array([5., 5.])
r = 1.5
gamma = 2.0

def h(x):                       # 안전 함수
    return np.linalg.norm(x - x_obs)**2 - r**2

def grad_h(x):                  # ∂h/∂x
    return 2 * (x - x_obs)

# 명목 제어기: 목표점으로 직진
def u_nom(x, goal):
    return 1.0 * (goal - x)

def safe_filter(x, u_des):
    u = cp.Variable(2)
    cons = [grad_h(x) @ u + gamma * h(x) >= 0,
            cp.norm(u, 'inf') <= 2.0]
    prob = cp.Problem(cp.Minimize(cp.sum_squares(u - u_des)), cons)
    prob.solve()
    return u.value

x = np.array([0., 0.])
goal = np.array([10., 10.])
dt = 0.05
for k in range(400):
    u_d = u_nom(x, goal)
    u   = safe_filter(x, u_d)
    x   = x + u * dt
print("최종 위치:", x, "  h =", h(x))
위 코드는 명목 제어기가 장애물을 향해 돌진하더라도, CBF-QP가 입력을 자동으로 수정하여 절대 부딪히지 않게 만듭니다. 이것이 "안전 필터"의 핵심입니다.
