2. 모델 예측 제어 (Model Predictive Control)

MPC는 "미래를 예측하고, 그 예측을 바탕으로 현재 입력을 결정"하는 제어 기법입니다. 매 시간 스텝마다 유한 구간 최적화 문제를 풀고, 첫 번째 입력만 적용한 뒤 다음 스텝에서 다시 푸는 방식입니다 (receding horizon).

  1. 핵심 아이디어

2.1 핵심 아이디어

운전을 생각해 봅시다. 우리는 즉각적인 핸들 조작만 하는 것이 아니라, "앞으로 몇 초 동안 차가 어떻게 움직일지" 머릿속으로 시뮬레이션하면서 운전합니다. MPC는 정확히 이 과정을 수학적으로 구현합니다.

2.2 최적화 문제 정식화

이산 시간 시스템 $x_{k+1} = f(x_k, u_k)$ 에 대해 매 시간 $k$ 에서 다음 문제를 풉니다.

$$ \begin{aligned} \min_{u_{0|k},\dots,u_{N-1|k}} \quad & \sum_{i=0}^{N-1}\Big( \|x_{i|k}-x^{\text{ref}}_{i|k}\|_Q^2 + \|u_{i|k}\|_R^2 \Big) + \|x_{N|k}-x^{\text{ref}}_{N|k}\|_{Q_f}^2 \\ \text{s.t.}\quad & x_{i+1|k} = f(x_{i|k}, u_{i|k}) \\ & x_{0|k} = x_k \\ & u_{i|k} \in \mathcal{U},\ x_{i|k}\in \mathcal{X} \end{aligned} $$

여기서 $\|v\|_Q^2 = v^\top Q v$ 입니다. $N$ 은 예측 구간(prediction horizon), $Q,R,Q_f$ 는 가중치입니다.

2.3 Receding Horizon 원리

  1. 현재 상태 $x_k$ 측정
  2. $N$ 스텝 미래까지의 최적 입력 시퀀스 $u_{0|k}^*, \dots, u_{N-1|k}^*$ 계산
  3. 첫 번째 입력 $u_{0|k}^*$ 만 시스템에 적용
  4. 한 스텝 후 새 상태 $x_{k+1}$ 측정 → 1로 돌아감
매 스텝마다 다시 푸는 이유는 외란·모델링 오차를 보정하기 위해서입니다. 이것이 MPC가 강건한 핵심 이유입니다.

2.4 제약 조건의 처리

MPC의 가장 큰 강점은 제약을 명시적으로 다룰 수 있다는 점입니다.

  • 입력 제약: $u_{\min}\le u \le u_{\max}$ (예: 조향각 한계, 최대 토크)
  • 상태 제약: $x \in \mathcal{X}$ (예: 속도 한계)
  • 안전 제약: 장애물 회피 → 일반적으로 비볼록(non-convex). CBF로 처리하면 효율적입니다 (다음 챕터).

2.5 코드 예제: CVXPY로 선형 MPC

import numpy as np
import cvxpy as cp

# 시스템: 이산화된 1차원 차량
dt = 0.1
A = np.array([[1, dt],[0, 1]])
B = np.array([[0.5*dt*dt],[dt]])
nx, nu = 2, 1
N = 20  # 예측 구간

Q  = np.diag([10., 1.])
R  = np.array([[0.1]])
Qf = np.diag([20., 2.])

x0_param   = cp.Parameter(nx)
xref_param = cp.Parameter((nx, N+1))

x = cp.Variable((nx, N+1))
u = cp.Variable((nu, N))

cost = 0
constraints = [x[:, 0] == x0_param]
for k in range(N):
    cost += cp.quad_form(x[:,k] - xref_param[:,k], Q)
    cost += cp.quad_form(u[:,k], R)
    constraints += [x[:,k+1] == A @ x[:,k] + B @ u[:,k]]
    constraints += [cp.abs(u[:,k]) <= 3.0]   # 가속도 제한
cost += cp.quad_form(x[:,N] - xref_param[:,N], Qf)

prob = cp.Problem(cp.Minimize(cost), constraints)

# 시뮬레이션
x_now = np.array([0., 0.])
target = np.array([10., 0.])
for step in range(100):
    x0_param.value   = x_now
    xref_param.value = np.tile(target, (N+1,1)).T
    prob.solve(solver=cp.OSQP, warm_start=True)
    u_apply = u.value[:,0]
    x_now   = A @ x_now + B.flatten() * u_apply
print("최종 상태:", x_now)
