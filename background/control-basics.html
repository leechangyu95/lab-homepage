홈 제어 기초 MPC CBF 자율주행

1. 제어 기초 (Control Basics)

MPC와 CBF를 이해하려면 먼저 상태공간 표현, 안정성, LQR이라는 세 가지 토대를 알아야 합니다.

목차

  1. 상태공간 모델
  2. 선형화
  3. 안정성 (Lyapunov)
  4. LQR 최적 제어
  5. 코드 예제

1.1 상태공간 모델

제어 대상(시스템)을 수학적으로 표현하는 가장 일반적인 방법은 다음과 같은 미분방정식입니다.

$$\dot{x}(t) = f(x(t), u(t))$$

여기서 $x \in \mathbb{R}^n$ 은 상태 벡터(예: 위치, 속도, 자세), $u \in \mathbb{R}^m$ 은 제어 입력(예: 토크, 조향각)입니다.

선형 시스템의 경우 다음과 같이 표현됩니다.

$$\dot{x} = A x + B u$$

예시: 1차원 차량
상태 $x = [p, v]^\top$ (위치, 속도), 입력 $u$ = 가속도일 때 $$A = \begin{bmatrix}0 & 1\\ 0 & 0\end{bmatrix},\quad B = \begin{bmatrix}0\\1\end{bmatrix}$$

1.2 선형화

실제 로봇 시스템은 대부분 비선형입니다. 평형점 $(x^*, u^*)$ 근처에서 테일러 전개로 선형화하면

$$\dot{\delta x} \approx \left.\frac{\partial f}{\partial x}\right|_{*} \delta x + \left.\frac{\partial f}{\partial u}\right|_{*} \delta u$$

1.3 Lyapunov 안정성

시스템이 평형점 근처에서 "흩어지지 않는다"는 것을 증명하는 가장 강력한 도구가 Lyapunov 함수 $V(x)$ 입니다.

다음 조건을 만족하면 평형점은 점근 안정합니다:

$$V(x) > 0,\ \forall x \neq 0,\quad \dot{V}(x) < 0$$

1.4 LQR (Linear Quadratic Regulator)

선형 시스템 $\dot{x}=Ax+Bu$ 에 대해 다음 비용함수를 최소화하는 제어 입력을 찾는 문제입니다.

$$J = \int_0^\infty \left(x^\top Q x + u^\top R u\right) dt$$

해는 놀랍게도 상태 피드백 형태입니다.

$$u^* = -K x,\quad K = R^{-1} B^\top P$$

여기서 $P$ 는 대수 리카티 방정식(ARE) 의 해입니다.

$$A^\top P + P A - P B R^{-1} B^\top P + Q = 0$$

LQR은 MPC의 "무한 구간(infinite horizon)" 특수해라고 볼 수 있습니다. MPC를 이해하면 LQR과의 관계가 명확해집니다.

1.5 코드 예제: Python으로 LQR 풀기

import numpy as np
import scipy.linalg

# 1차원 차량: 상태 [위치, 속도], 입력 가속도
A = np.array([[0., 1.],
              [0., 0.]])
B = np.array([[0.],
              [1.]])

Q = np.diag([10., 1.])   # 위치 오차에 가중치 10
R = np.array([[0.1]])    # 입력 패널티

# 리카티 방정식 풀이
P = scipy.linalg.solve_continuous_are(A, B, Q, R)
K = np.linalg.inv(R) @ B.T @ P
print("LQR gain K =", K)

# 시뮬레이션
x = np.array([5., 0.])     # 초기: 5m 떨어진 곳
dt = 0.05
for t in range(200):
    u = -K @ x
    x = x + (A @ x + B.flatten() * u) * dt
print("최종 상태:", x)   # 원점 근처로 수렴
