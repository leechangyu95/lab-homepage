인터랙티브 데모: MPC + CBF 충돌 회피

아래 캔버스를 클릭하면 그 위치에 장애물이 추가됩니다. 슬라이더로 파라미터를 조절하며 명목 제어기(목표점으로 직진)와 CBF 안전 필터의 동작을 직접 관찰해보세요.

알고리즘 설명

이 데모는 2D 단적분기 로봇 ($\dot{x}=u$) 에 대해 다음을 매 프레임 푼다:

  1. 명목 입력: $u_{\text{nom}} = K(x_{\text{goal}}-x)$
  2. 각 장애물 $i$에 대해 $h_i = \|x-x_i\|^2 - r^2$, $\nabla h_i = 2(x-x_i)$
  3. CBF 제약: $\nabla h_i^\top u + \gamma h_i \ge 0$
  4. 여러 제약을 동시에 만족하는 $u^*$ 를 활성 제약 사영(active-set projection)으로 계산하여 적용

γ가 작을수록 더 일찍 회피하고, 클수록 장애물에 가까워질 때까지 직진합니다. CBF를 끄면 명목 제어기가 그대로 적용되어 장애물에 부딪힙니다.

이 데모는 단순 단적분기 로봇이지만, 실제 SAIL 연구에서는 같은 원리를 비선형 선박/쿼드로터 동역학과 Tube-MPC, HJ-Reachability 위에 결합합니다.
