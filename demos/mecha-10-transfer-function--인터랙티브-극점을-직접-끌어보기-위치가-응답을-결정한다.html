🎮 인터랙티브: 극점을 직접 끌어보기 — 위치가 응답을 결정한다

좌측 $s$ 평면의 빨간 ✕(극점)을 마우스로 끌어보세요. 위·아래 대칭(켤레쌍)으로 함께 움직이며, 우측에 시간 응답이 즉시 반영됩니다. 다음을 직접 확인해 보세요:

  1. 극점을 왼쪽으로 멀리 보내면 → 빠르게 정착.
  2. 극점이 허수축에 가까워지면(위·아래로 멀리) → 진동이 심해짐.
  3. 극점이 실수축 위(im=0)에 있으면 → 진동 없음(과/임계감쇠).
  4. 극점이 우반평면으로 넘어가면 → 응답이 발산(불안정).
실수부 σ = −1.0, 허수부 ω = 2.0
