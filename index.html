홈 제어 기초 MPC CBF 자율주행

새로 합류한 학생들을 위한 배경지식 가이드

우리 연구실은 모델 예측 제어(MPC)와 제어 장벽 함수(CBF)를 중심으로 안전하고 고성능의 로봇·자율주행 시스템을 연구합니다. 이 사이트는 처음 들어오는 학생이 연구 주제를 빠르게 따라잡을 수 있도록 개념·수식·그림·코드 예제를 단계별로 정리한 자료입니다.

1. 제어 기초

상태공간, 안정성, LQR — 모든 것의 출발점.

2. 모델 예측 제어 (MPC)

최적화 기반 제어의 핵심 아이디어와 구현.

3. 제어 장벽 함수 (CBF)

안전성을 수학적으로 보장하는 방법.

4. 자율주행 응용

차량 모델, 경로 추종, 충돌 회피.

학습 순서 추천

  1. 제어 기초 — 상태공간과 LQR을 먼저 익히세요.
  2. MPC — 최적화 기반 제어로 확장합니다.
  3. CBF — 안전 제약을 다루는 방법을 배웁니다.
  4. 자율주행 응용 — MPC + CBF를 차량에 적용합니다.
