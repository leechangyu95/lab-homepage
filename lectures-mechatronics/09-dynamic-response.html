Week 9. 동적 응답 — 1차·2차 시스템의 시간 응답

시스템 모델을 적었으니 이제 응답을 봐야죠. 같은 입력이라도 시스템마다 어떻게 다른지, $\zeta$가 어떻게 모양을 만드는지.

목차

  1. 표준 입력 4가지
  2. 1차 시스템 — 시정수 $\tau$
  3. 2차 시스템 — $\zeta$ 4가지 케이스
  4. 응답 성능 지표
  5. 정상상태 오차
  6. 주파수 응답 미리보기

1. 표준 입력 4가지

시스템의 동특성을 파악하려고 다음 표준 입력에 대한 응답을 봅니다.

이름식해석
임펄스(Impulse) $\delta(t)$매우 짧고 큰 입력시스템 고유 응답
스텝(Step) $u(t)$$t=0$부터 1로 일정가장 자주 쓰는 시험 입력
램프(Ramp) $r(t) = t$일정 속도로 증가추적 성능 시험
정현파 $\sin(\omega t)$주파수 시험주파수 응답 분석 (W10)

왜 이런 입력이 표준일까? 실제 시스템에 들어가는 입력은 무한히 다양하지만, 위 네 가지를 알면 거의 모든 입력에 대한 응답을 합성으로 알 수 있어요(선형 시스템의 중첩 원리). 게다가 임펄스·스텝은 실험실에서 가장 만들기 쉬운 신호 — 버튼 한 번 누르거나 갑자기 무게를 얹는 것만으로도 됩니다.

🎮 인터랙티브: 4가지 표준 입력 한눈에 비교

입력의 모양만 봐도 머릿속 직관이 다릅니다. 임펄스는 "한 방 때리기", 스텝은 "갑자기 일정값 유지", 램프는 "꾸준히 증가", 정현파는 "주기적으로 흔들기". 각 입력에 대한 응답이 다음 절부터 어떻게 다른지를 비교해 보세요.

2. 1차 시스템 — 시정수 $\tau$

가장 단순한 동적 시스템. 미분방정식과 표준형:

$$ \tau\dot{y} + y = K\,u(t) $$

예: RC 회로의 커패시터 전압, 1차 LPF, 온도 센서 응답. 단위 스텝 입력 $u(t) = 1$에 대한 응답:

$$ y(t) = K\left(1 - e^{-t/\tau}\right) $$
  • $\tau$(시정수): $y$가 최종값의 63.2%까지 도달하는 시간. 작을수록 빠른 시스템.
  • $K$(DC 이득): 정상상태 비례 — 단위 입력당 정상상태 출력.
  • $5\tau$이면 99% 도달 → 실무에서 "5τ면 끝났다".

이 식은 W3·W4에서 만난 RC 충전 응답과 정확히 같습니다. 1차 시스템의 모든 것은 $\tau$가 결정해요.

왜 1차 시스템은 진동이 없을까? 직관적으로, 1차 시스템에는 관성(에너지 저장 요소)이 하나뿐이라 한 번 평형을 지나칠 만한 "힘"이 없습니다. 진동하려면 에너지가 두 곳(예: 스프링과 질량)을 오가야 하는데, 1차 시스템에는 그 "두 곳"이 없어요. 그래서 응답은 항상 단조롭게 최종값에 가까워집니다.

🎮 인터랙티브: 1차 시스템의 스텝 응답

$\tau$를 0.2 → 2초로 옮겨보세요. 표시된 빨간 점이 정확히 ($\tau$, 63.2%), ($2\tau$, 86.5%), ($3\tau$, 95%), ($5\tau$, 99.3%)에 위치합니다. $\tau$가 작은 시스템 = 빠른 시스템. DC 이득 $K$는 정상상태 출력값을 결정해요(입력의 몇 배인지).

3. 2차 시스템 — $\zeta$의 4가지 얼굴

표준형:

$$ \ddot{y} + 2\zeta\omega_n\dot{y} + \omega_n^2 y = K\omega_n^2 u(t) $$

스텝 입력에 대한 응답은 $\zeta$ 값에 따라 4가지 형태로 갈립니다.

3.1 무감쇠 $\zeta = 0$

$$ y(t) = K\,(1 - \cos\omega_n t) $$

영원히 진동. 이상화 — 실제 시스템에는 마찰이 있어 거의 나타나지 않음.

3.2 저감쇠 $0 < \zeta < 1$

$$ y(t) = K\!\left[1 - \frac{e^{-\zeta\omega_n t}}{\sqrt{1-\zeta^2}} \sin(\omega_d t + \phi)\right], \quad \omega_d = \omega_n\sqrt{1-\zeta^2} $$

진동하면서 점점 안정화. 자동차 서스펜션, 빠른 제어 시스템이 보통 여기에 속합니다. $\omega_d$를 감쇠 진동수라고 합니다 — 실제로 관측되는 진동의 주파수는 $\omega_n$이 아니라 $\omega_d$.

3.3 임계감쇠 $\zeta = 1$

$$ y(t) = K\!\left[1 - (1 + \omega_n t)e^{-\omega_n t}\right] $$

진동 없이 가장 빠르게 정착. 기계식 측정기, 정밀 측정 장비의 이상.

3.4 과감쇠 $\zeta > 1$

두 개의 음의 실수 극점. 진동 없이 천천히 정착. 건물 출입문 댐퍼 같은 안전·완충 응용.

4. 응답 성능 지표

2차 시스템의 응답 모양을 정량화하는 표준 지표들. 제어 시스템 설계의 사양으로 자주 등장합니다. 한 그래프 위에 네 지표가 어디에 위치하는지를 먼저 보고 시작하면 식이 훨씬 쉬워져요.

4.1 상승시간(Rise time) $t_r$

출력이 최종값의 10%에서 90%까지 도달하는 시간. 빠른 시스템일수록 작음. "얼마나 빨리 반응했나".

4.2 첨두시간(Peak time) $t_p$

$$ t_p = \frac{\pi}{\omega_d} = \frac{\pi}{\omega_n\sqrt{1-\zeta^2}} $$

처음으로 최댓값에 도달하는 시간 (저감쇠일 때). "언제 가장 많이 튀었나".

4.3 최대 오버슈트(Overshoot) $M_p$

$$ M_p = e^{-\zeta\pi/\sqrt{1-\zeta^2}} \times 100\,\% $$

최종값을 얼마나 넘어가는지. $\zeta$만의 함수입니다 — $\omega_n$은 영향이 없어요. "얼마나 튀었나".

  • $\zeta = 0.7$: $M_p \approx 4.6\%$ (제어 설계의 기본 목표)
  • $\zeta = 0.5$: $M_p \approx 16\%$
  • $\zeta = 0.2$: $M_p \approx 53\%$ (너무 흔들림)

이 식이 어디서 왔는지 살짝 — 저감쇠 응답 곡선의 첫 번째 봉우리 높이를 미분으로 구하면 정확히 위 식이 나옵니다(유도는 W10 라플라스 변환 후가 더 깔끔함). 여기서 기억할 것은 $M_p$는 $\zeta$ 하나만 결정한다는 사실이에요.

4.4 정착시간(Settling time) $t_s$

$$ t_s \approx \frac{4}{\zeta\omega_n} \quad (\pm 2\% \text{ 기준}) $$

출력이 최종값의 ±2% 이내로 들어와 머무는 시간. 시스템의 "정리됨"을 정량화. "언제부터 안정됐나".

식의 직관 — 저감쇠 응답의 진폭은 $e^{-\zeta\omega_n t}$로 줄어듭니다. ±2%로 줄려면 $e^{-\zeta\omega_n t_s} = 0.02$, 즉 $\zeta\omega_n t_s \approx 4$. $\zeta\omega_n$의 곱이 응답이 잦아드는 속도입니다 — 이걸 실수부 감쇠라고 부르며 W10에서 극점의 위치로 다시 등장해요.

🎮 인터랙티브: 2차 시스템 스텝 응답 — 4가지 지표 함께 보기

$\zeta$와 $\omega_n$을 바꾸면 각 지표가 어떻게 변하는지 표시점이 따라 움직입니다. 핵심 발견 — $M_p$는 $\zeta$만 보고, $t_p$·$t_s$는 $\omega_n$도 같이 봅니다. 설계 사양($M_p < X\%$, $t_s < Y\,\text{s}$)을 만족시키려면 두 값을 별도로 잡아야 한다는 뜻이에요.

5. 정상상태 오차

충분한 시간이 흐른 뒤 출력과 목표의 차이.

$$ e_{ss} = \lim_{t \to \infty}(r(t) - y(t)) $$

왜 오차가 0이 안 될 수 있나? 가장 단순한 P 제어를 생각해 보세요. 출력 $y$가 목표 $r$에 정확히 도달하면 오차 $e=r-y=0$이 되고, 그러면 제어 명령 $u=K_p e=0$이 됩니다. 그런데 시스템에 마찰·중력 같은 일정한 외란이 있으면 명령이 0인 순간 출력이 다시 떨어져요. 결국 출력은 "명령이 외란을 정확히 상쇄하는 위치"에서 멈춥니다 — 이게 0이 아닌 일정 오차로 남는 이유입니다.

입력 형태와 시스템 종류(Type)에 따라 다릅니다. Type이란 시스템 전달함수 분모에 $s$(=원점 극점=적분기)가 몇 개 있는지를 가리킵니다.

  • Type 0(원점 극점 0개): 스텝 입력에 유한 오차 — P 제어로 줄임.
  • Type 1(원점 극점 1개, 적분기 1개): 스텝에 0 오차, 램프에 유한 오차 — 적분기가 필요한 이유.
  • Type 2(원점 극점 2개): 램프에도 0 오차 — 항법·추적 시스템.

적분기를 하나 추가할 때마다 한 단계 더 어려운 입력에도 0 오차가 됩니다. "정상상태 오차를 어떻게 0으로 만들까"가 PID의 I항(적분 제어)이 등장하는 핵심 동기예요. W11에서 PID로 직접 이 효과를 만들어 봅니다.

🎮 인터랙티브: P 제어의 정상상태 오차 — I 추가의 동기

1차 플랜트 $1/(\tau s+1)$에 P 제어만 적용하면, 이득 $K_p$를 키워도 오차가 완전히 0이 되지 않습니다(분수만 줄어들 뿐). I항을 켜면 누적 오차가 컨트롤러를 강제로 더 밀어붙여 결국 출력이 정확히 목표에 도달해요. 이게 W11 PID의 출발점입니다.

6. 주파수 응답 — 다음 주 미리보기

같은 시스템에 정현파 입력을 넣으면 출력도 정현파로 나오는데, 입력 주파수에 따라 진폭과 위상이 변합니다. 그 그래프를 보드 플롯(Bode plot)이라 하고 두 가지를 같이 봅니다.

  • 크기 응답 $|H(j\omega)|$ [dB]: 어느 주파수가 통과/감쇠되는지 (필터 분석).
  • 위상 응답 $\angle H(j\omega)$ [°]: 출력이 입력에 얼마나 뒤지는지 (안정성 분석).

이 분석을 라플라스 변환과 함께 다음 주(W10)에서 시작합니다.
