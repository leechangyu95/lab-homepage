Week 2. 센서와 트랜스듀서

물리 세계를 전기 신호로 번역하는 첫 단계. 측정의 정확도가 모든 후속 처리의 한계를 정합니다.

목차

  1. 센서 vs 트랜스듀서 — 같은 듯 다른 두 단어
  2. 센서 분류: 접촉/비접촉, 아날로그/디지털, 능동/수동
  3. 정적 특성 — 정확도·정밀도·민감도·이력
  4. 동적 특성 — 응답 시간과 시정수
  5. 대표 센서 — 포텐셔미터, 스트레인 게이지, 엔코더, IMU

1. 센서 vs 트랜스듀서

두 단어가 자주 혼용되지만 정확히는 다음과 같이 구분됩니다.

  • 센서(sensor) — 물리량(온도·압력·거리 등)을 감지하는 소자.
  • 트랜스듀서(transducer) — 한 형태의 에너지를 다른 형태로 변환하는 소자. 즉 센서 + 신호 변환 회로의 통칭.

예: 서모커플(thermocouple)은 온도(열에너지)를 전압(전기에너지)으로 변환하는 트랜스듀서입니다. 센서가 곧 트랜스듀서인 경우가 많아 일상에서는 거의 같은 의미로 씁니다.

2. 센서의 분류

2.1 접촉 vs 비접촉

  • 접촉형(Contact): 측정 대상에 직접 닿아야 함. 예) 리미트 스위치, 스트레인 게이지, 서모커플.
  • 비접촉형(Non-contact): 떨어져서 측정. 예) 적외선 거리센서, 카메라, 레이더, 라이다.

2.2 변위·위치·근접 센서

종류측정량대표 소자
변위(Displacement)이동 거리LVDT, 포텐셔미터
위치(Position)절대/상대 위치엔코더, GPS
근접(Proximity)가까이 있는지 (On/Off)유도형, 정전용량형, 광전

2.3 아날로그 vs 디지털

  • 아날로그: 출력이 연속적인 전압/전류. 잡음 영향 받음. ADC를 거쳐 디지털화 필요.
  • 디지털: 출력이 0/1 비트열. 잡음에 강함. MCU로 직접 읽기 가능.

3. 정적 특성(Static Characteristics)

입력이 일정한 정상상태에서의 센서 성능 지표들입니다.

3.1 Range, Span, Error

  • Range(범위): 측정 가능한 최소/최대값. 예) $-40 \sim 125\,°C$.
  • Span(스팬): 최대 − 최소. 위 예에서 $165\,°C$.
  • Error(오차): 측정값 − 참값. 참 24°C일 때 25°C가 나오면 오차 +1°C.

3.2 정확도(Accuracy) vs 정밀도(Precision)

두 단어를 자주 헷갈리는데, 다음 그림으로 설명됩니다.

  • 정확도: 측정값이 참값에 얼마나 가까운가 (전체 범위에 대한 % 오차로 표현).
  • 정밀도(Repeatability): 같은 입력을 반복 측정했을 때 결과가 얼마나 일정한가. 정확도와 별개로, 평균은 빗나가도 분포가 좁으면 정밀도는 높음.

🎮 인터랙티브: 바이어스와 산포 직접 만지기

슬라이더로 바이어스(평균이 참값에서 얼마나 벗어났는지)와 산포(측정 표준편차)를 바꿔보세요. 산포가 작으면 정밀도↑, 바이어스가 작으면 정확도↑. 두 축이 독립적이라는 점을 직접 확인하세요.

3.3 민감도(Sensitivity)

입력 변화에 대한 출력 변화의 비율. 예: $0.5\,\text{V}/°C$인 온도 센서는 1°C 올라가면 출력이 0.5V 증가.

$$ S = \frac{\Delta\,\text{output}}{\Delta\,\text{input}} $$

3.4 이력현상(Hysteresis)

같은 입력값이라도 입력이 증가하다 도달한 경우와 감소하다 도달한 경우 출력이 다른 현상. 강자성체나 일부 기계 부품의 마찰에서 흔히 나타납니다. 정밀 측정에서는 이력 곡선의 면적으로 정량화합니다.

🎮 인터랙티브: 이력 곡선 그려보기

입력 슬라이더를 좌우로 천천히 움직여 보세요. 입력을 키울 때와 줄일 때 출력이 다른 경로를 그리며, 닫힌 고리가 만들어집니다. 이력 폭이 클수록 같은 입력에 대한 출력 차이가 큽니다.

3.5 비선형 오차

이상적으로 출력이 입력에 직선 비례해야 하지만 실제로는 곡선. 그 편차의 최댓값을 풀스케일 대비 %로 표시합니다.

3.6 분해능(Resolution)·데드밴드·드리프트

  • 분해능: 식별 가능한 최소 변화량. 12-bit ADC로 0~5V를 측정하면 분해능 $\approx 1.22\,\text{mV}$.
  • 데드밴드: 입력이 변해도 출력이 반응하지 않는 구간. 예) 마우스 휠을 살짝 돌려도 안 움직이는 영역.
  • 데드타임: 입력 변화 후 출력 변화가 시작하기까지의 지연.
  • 제로 드리프트: 시간이 지나면서 0점이 천천히 이동하는 현상. 정기적 캘리브레이션이 필요한 이유.

4. 동적 특성(Dynamic Characteristics)

입력이 시시각각 변할 때 센서가 얼마나 빨리 반응하는지에 대한 지표. 입력 형태별로 다음을 봅니다.

  • 스텝 입력: 갑자기 한 값으로 변함 (실험에서 가장 자주 씀).
  • 램프 입력: 일정 속도로 변함.
  • 정현파 입력: 주파수 응답 측정용.

4.1 시정수 $\tau$

1차 시스템(가장 흔한 센서 모델)의 스텝 응답:

$$ y(t) = y_\infty\left(1 - e^{-t/\tau}\right) $$

$t = \tau$에서 최종값의 약 63.2%, $5\tau$이면 99% 도달. 이 $\tau$가 작을수록 빠른 센서입니다. 기전공학기초 W7의 RC 시정수와 정확히 같은 식이에요 — 모든 1차 시스템은 같은 수학으로 풀립니다.

4.2 상승시간(Rise Time)·정착시간(Settling Time)

  • 상승시간 $t_r$: 출력이 10%에서 90%까지 도달하는 시간.
  • 정착시간 $t_s$: 최종값의 ±2% 이내로 들어와 머무는 시간.

🎮 인터랙티브: 1차 센서의 스텝 응답과 시정수

$\tau$를 바꿔보세요. $t = \tau$에서 정확히 63.2%, $5\tau$에서 약 99% 도달. 참값(스텝)은 즉시이지만 센서 출력은 시정수만큼 늦게 따라갑니다. 빠른 변화를 측정하려면 $\tau$가 작은 센서가 필요합니다.

5. 대표 센서들

5.1 포텐셔미터(Potentiometer)

가변 저항. 회전 또는 직선 이동 → 저항값 변화 → 출력 전압 변화. 단순하지만 마찰·마모로 분해능과 수명에 한계.

5.2 스트레인 게이지(Strain Gauge)

금속 와이어가 늘어나면 길이는 증가, 단면적은 감소 → 저항이 증가. 미세한 저항 변화를 기전공학기초 W5의 휘트스톤 브릿지로 정밀하게 읽습니다.

$$ \frac{\Delta R}{R} = G \cdot \varepsilon $$

$G$는 게이지 인자(보통 ~2), $\varepsilon$은 변형률.

5.3 엔코더(Encoder)

  • 증분형(Incremental): 펄스를 세서 상대적 회전 측정. 전원 끄면 위치 정보 잃음.
  • 절대형(Absolute): 디스크에 패턴이 있어 절대 위치를 한 번에 읽음. 더 비쌈.

로봇 관절 제어에 거의 항상 쓰이는 센서.

5.4 IMU (관성 측정 장치)

  • 가속도계(Accelerometer): 선형 가속도 측정. 중력에서 기울기 계산 가능.
  • 자이로(Gyroscope): 각속도 측정. 적분으로 자세 추정 (드리프트 누적).
  • 자력계(Magnetometer): 지자기 방향. 절대 방위.

3축씩 합쳐 9-DOF IMU가 만들어지고, 그 신호를 W4 칼만 필터로 융합해 자세를 추정합니다.

5.5 거리 센서

종류원리특징
적외선(IR)반사광 강도 → 거리저가, 단거리(~80cm), 색·재질 영향
초음파음파 왕복시간(ToF)저가, 중거리(~5m), 각도 분해능 낮음
레이저(LiDAR)레이저 ToF / 위상고정밀, 고가, 자율주행 핵심
레이더전파 ToF / 도플러날씨 강건, 속도까지 측정
