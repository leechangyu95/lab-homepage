Week 10. CNN 2 — 전이학습 & 데이터 증강

남이 학습해 놓은 모델의 어깨 위에 올라타서, 적은 데이터로도 좋은 성능을 냅니다.

목차

  1. 학습된 필터 들여다보기
  2. 데이터 증강
  3. 전이학습 (Transfer Learning)
  4. ResNet과 잔차 연결
  5. 인터랙티브 데이터 증강 데모

1. 학습된 필터 들여다보기

학습이 끝난 CNN의 첫 층 필터를 그려보면, 보통 엣지 검출기나 색상 패치 같은 단순한 패턴이 나옵니다. 더 깊은 층의 활성화는 눈, 바퀴, 얼굴처럼 점점 추상적인 부품을 인식합니다.

PyTorch에서는 register_forward_hook으로 중간 출력을 캡처할 수 있습니다.

2. 데이터 증강 (Data Augmentation)

학습 데이터가 적을 때는 같은 이미지를 약간씩 변형해 가짜 데이터를 만듭니다.

  • 좌우 뒤집기 (RandomHorizontalFlip)
  • 약간 회전 (RandomRotation)
  • 모서리 자르기 (RandomCrop)
  • 색상 변화 (ColorJitter)

모델은 더 다양한 이미지를 보게 되어 일반화 성능이 올라갑니다.

3. 전이학습

ImageNet(120만 장)으로 미리 학습된 모델을 가져와, 마지막 분류층만 우리 데이터에 맞춰 재학습합니다.

$$ \theta_\text{pretrained} \xrightarrow{\text{fine-tune}} \theta_\text{new task} $$

처음부터 학습하는 것보다 보통 훨씬 적은 데이터·시간으로 더 높은 정확도를 얻습니다.

4. ResNet과 잔차 연결

층이 너무 깊어지면 기울기가 사라져 학습이 안 됩니다. ResNet은 이를 지름길(skip connection)로 해결합니다.

$$ \mathbf{y} = \mathcal{F}(\mathbf{x}) + \mathbf{x} $$

덧셈 한 줄 덕분에 100층, 1000층까지도 학습이 가능해졌습니다.

🎮 인터랙티브: 데이터 증강 미리보기

같은 원본 이미지에 다양한 변환이 적용된 결과를 동시에 보여줍니다.

5. PyTorch 코드

from torchvision import transforms, models
import torch.nn as nn

# 1) 데이터 증강
train_tf = transforms.Compose([
    transforms.RandomHorizontalFlip(),
    transforms.RandomCrop(32, padding=4),
    transforms.ColorJitter(0.2, 0.2, 0.2),
    transforms.ToTensor(),
    transforms.Normalize((0.49,0.48,0.45), (0.25,0.24,0.26)),
])

# 2) 사전학습된 ResNet18 불러와 마지막 층만 교체
model = models.resnet18(weights='IMAGENET1K_V1')
model.fc = nn.Linear(512, 10)             # CIFAR-10용

# 3) 새 fc 층만 빠르게 학습
opt = torch.optim.Adam(model.fc.parameters(), lr=1e-3)
