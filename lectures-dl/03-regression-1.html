"가장 잘 맞는 직선"을 수학으로 정의하고 컴퓨터로 찾아봅시다.

1. 회귀 vs 분류

분류는 카테고리, 회귀는 연속된 숫자를 예측합니다. "농어 길이로 무게 맞추기"가 회귀의 전형적 예입니다.

2. 손실 함수

예측값 $\hat{y}$가 정답 $y$에서 얼마나 어긋났는지를 한 숫자로 요약합니다.

$$ \text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2 $$ $$ \text{MAE} = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i| $$ $$ R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2} $$
  • MSE — 큰 오차에 더 큰 벌점. 가장 흔히 사용.
  • MAE — 절댓값 합. 이상치에 강함.
  • R² — 1에 가까우면 좋고, 0이면 평균값으로 답한 것과 같음.

3. 선형 회귀

가장 단순한 가설: 데이터가 직선 위에 있다고 가정합니다.

$$ \hat{y} = w x + b $$

$w$는 기울기, $b$는 y절편입니다. MSE를 최소로 만드는 $(w, b)$를 찾는 게 목표입니다.

4. 다항 회귀

직선으로 부족할 때는 곡선을 쓸 수 있습니다.

$$ \hat{y} = w_d x^d + w_{d-1} x^{d-1} + \dots + w_1 x + b $$

차수 $d$가 클수록 표현력이 강해지지만, 학습 데이터에 너무 맞춰지는 과적합이 생깁니다.

🎮 인터랙티브: 다항 회귀와 과적합

점들이 농어의 길이(가로) → 무게(세로)를 모방한 데이터입니다. 차수를 1부터 9까지 올려가며 어떻게 변하는지 보세요.

5. 코드 예제 (scikit-learn)

import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

X = np.array([8.4, 13.7, 15.0, 16.2, 17.4, 18.0, 18.7, 19.0, 22.0, 26.0]).reshape(-1,1)
y = np.array([5.9, 32.0, 40.0, 51.5, 70.0, 100.0, 78.0, 80.0, 110.0, 175.0])

# 선형 회귀
lr = LinearRegression()
lr.fit(X, y)
print(f"기울기 w = {lr.coef_[0]:.3f}, 절편 b = {lr.intercept_:.3f}")
print(f"R² = {lr.score(X, y):.3f}")

# 다항(2차) 회귀: x^2 특성 추가
X_poly = np.column_stack([X, X**2])
lr2 = LinearRegression()
lr2.fit(X_poly, y)
print(f"R² (deg 2) = {lr2.score(X_poly, y):.3f}")
