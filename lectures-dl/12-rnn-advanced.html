Week 12. RNN 심화 — GRU · BiLSTM · 시계열 예측

LSTM의 사촌들과 시계열 예측 응용을 살펴봅니다.

목차

  1. GRU — LSTM의 슬림 버전
  2. 양방향 LSTM (BiLSTM)
  3. 다층 LSTM
  4. 시계열 예측
  5. 인터랙티브 사인파 예측

1. GRU (Gated Recurrent Unit)

LSTM의 3개 게이트와 셀 상태를 단순화해, 게이트 2개와 hidden state 하나만 씁니다.

$$ r_t = \sigma(W_r [h_{t-1}, x_t]) \quad \text{(리셋)} $$ $$ z_t = \sigma(W_z [h_{t-1}, x_t]) \quad \text{(업데이트)} $$ $$ h_t = (1 - z_t)\, h_{t-1} + z_t\, \tilde{h}_t $$

파라미터가 LSTM보다 약 25–33% 적어 학습이 빠르고 메모리도 덜 듭니다. 성능 차이는 데이터에 따라 다릅니다.

2. 양방향 LSTM (BiLSTM)

문장을 왼쪽 → 오른쪽으로 한 번, 오른쪽 → 왼쪽으로 한 번 읽고 두 결과를 이어 붙입니다.

$$ h_t = [\,\overrightarrow{h_t}\,;\,\overleftarrow{h_t}\,] $$

"그는 _____을 좋아한다"의 빈칸을 채울 때 뒷문장도 활용할 수 있어 매우 유용합니다.

3. 다층 LSTM (Stacked LSTM)

한 LSTM의 출력을 다른 LSTM의 입력으로 넣어 층을 쌓습니다. 깊이가 늘어날수록 더 복잡한 시퀀스 패턴을 잡을 수 있지만 과적합 위험도 커지므로 보통 dropout=0.3을 함께 씁니다.

4. 시계열 예측

대표 응용은 "지난 30분의 값으로 다음 1분 예측"입니다.

$$ \hat{y}_{t} = f([x_{t-30}, x_{t-29}, \dots, x_{t-1}]) $$

슬라이딩 윈도우로 학습 샘플을 만들고, 마지막 hidden state를 FC 층에 통과시켜 한 값을 예측합니다.

모델파라미터속도장점
RNN적음빠름단순, 짧은 시퀀스
LSTM많음중간장기 의존성 강함
GRU중간빠름LSTM에 가까운 성능
BiLSTM2배느림양쪽 문맥 모두

5. PyTorch 코드

# 시계열 LSTM
class TSLSTM(nn.Module):
    def __init__(self, hidden=32):
        super().__init__()
        self.lstm = nn.LSTM(1, hidden, batch_first=True)
        self.fc = nn.Linear(hidden, 1)
    def forward(self, x):                # x: (B, T, 1)
        out, _ = self.lstm(x)
        return self.fc(out[:, -1, :]).squeeze(1)

# BiLSTM
self.lstm = nn.LSTM(embed_dim, hidden_dim,
                    bidirectional=True, batch_first=True)
self.fc   = nn.Linear(hidden_dim*2, 1)   # 양쪽 출력 결합

# Multi-layer LSTM with dropout
self.lstm = nn.LSTM(embed_dim, hidden_dim,
                    num_layers=2, dropout=0.3, batch_first=True)
