Week 8. 심층신경망과 역전파

한 층짜리 MLP를 깊게 쌓고, 역전파 수식을 직접 따라가 봅시다.

목차

  1. 왜 깊게 쌓는가
  2. 순전파 한 줄 정리
  3. 역전파의 핵심: 연쇄법칙
  4. 옵티마이저: SGD, Momentum, Adam
  5. 드롭아웃과 배치 정규화
  6. 인터랙티브 학습 데모

1. 왜 깊게 쌓는가?

은닉층 1개로도 이론적으로는 어떤 함수도 근사할 수 있지만(보편 근사 정리), 실제로는 폭이 폭발적으로 커야 합니다. 층을 깊게 쌓으면 같은 능력을 훨씬 적은 파라미터로 표현할 수 있습니다.

2. 순전파 (Forward Pass)

$$ \mathbf{a}^{[l]} = \sigma\!\bigl(W^{[l]} \mathbf{a}^{[l-1]} + \mathbf{b}^{[l]}\bigr) $$

입력 $\mathbf{a}^{[0]}$가 첫 층부터 마지막 층까지 차례대로 변환됩니다.

3. 역전파 (Backpropagation)

출력에서 손실 $\mathcal{L}$을 계산한 뒤, 연쇄법칙으로 모든 $W$에 대한 기울기를 한 번에 구합니다.

$$ \frac{\partial \mathcal{L}}{\partial W^{[l]}} = \frac{\partial \mathcal{L}}{\partial \mathbf{a}^{[l]}} \cdot \frac{\partial \mathbf{a}^{[l]}}{\partial \mathbf{z}^{[l]}} \cdot \frac{\partial \mathbf{z}^{[l]}}{\partial W^{[l]}} $$

핵심 직감: 마지막 층의 오차를 거꾸로(뒤에서 앞으로) 흘려보내면서 각 가중치가 손실에 얼마나 기여했는지 알아냅니다.

4. 옵티마이저

이름업데이트 규칙특징
SGD$w \leftarrow w - \alpha \nabla_w \mathcal{L}$단순, 느림
Momentum$v \leftarrow \beta v + \nabla_w$, $w \leftarrow w - \alpha v$관성으로 가속
Adam1차·2차 모멘트 적응대부분 디폴트

5. 과적합 방지

  • 드롭아웃 — 학습 중 뉴런을 확률 $p$로 끔. 앙상블 효과.
  • 배치 정규화 — 각 층 입력의 평균 0, 분산 1로 정규화.
  • 조기 종료 — 검증 손실이 다시 오르면 학습 중단.

6. PyTorch 코드

import torch.nn as nn, torch.nn.functional as F

class DNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 128)
        self.bn1 = nn.BatchNorm1d(128)
        self.drop = nn.Dropout(0.5)
        self.fc2 = nn.Linear(128, 64)
        self.fc3 = nn.Linear(64, 10)
    def forward(self, x):
        x = x.view(x.size(0), -1)
        x = F.relu(self.bn1(self.fc1(x)))
        x = self.drop(x)
        x = F.relu(self.fc2(x))
        return self.fc3(x)
