1. 퍼셉트론 (단일 뉴런)

$$ z = \sum_i w_i x_i + b, \quad \text{output} = \begin{cases}1 & z \ge 0 \\ 0 & z < 0\end{cases} $$

입력에 가중치를 곱해 합산한 뒤, 0보다 크면 1, 작으면 0을 출력하는 단순한 모델입니다. AND, OR 같은 선형 분리 가능 문제는 해결합니다.

2. XOR 문제

그러나 XOR은 직선 하나로는 절대 나눌 수 없습니다.

x₁x₂XOR
000
011
101
110

이 한 가지 사실 때문에 1970년대에 신경망 연구가 한 번 멈췄습니다. 해결책은 층을 여러 개 쌓는 것이었습니다.

3. 다층 퍼셉트론 (MLP)

은닉층 한두 개만 추가해도 임의의 비선형 함수를 표현할 수 있습니다(보편 근사 정리).

$$ \mathbf{a}^{[l]} = \sigma\!\bigl(W^{[l]} \mathbf{a}^{[l-1]} + \mathbf{b}^{[l]}\bigr) $$

각 층은 선형 변환 + 비선형 활성화의 반복입니다.

대표적 활성화 함수

$$ \text{ReLU}(z) = \max(0, z), \quad \sigma(z) = \frac{1}{1+e^{-z}} $$

ReLU는 단순하지만 깊은 네트워크에서 가장 잘 동작해 표준으로 쓰입니다.

4. 학습

손실 함수(분류는 교차 엔트로피, 회귀는 MSE)를 정의하고, 가중치 $w$를 손실 감소 방향으로 한 발씩 이동합니다.

$$ w \leftarrow w - \alpha \frac{\partial \mathcal{L}}{\partial w} $$

$\partial \mathcal{L}/\partial w$를 효율적으로 구하는 알고리즘이 역전파(backpropagation)입니다. PyTorch가 자동으로 해줍니다.

5. PyTorch 코드 (Fashion MNIST)

import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

# 데이터
tr = datasets.FashionMNIST('.', train=True, download=True,
                           transform=transforms.ToTensor())
ld = DataLoader(tr, batch_size=64, shuffle=True)

# 모델
model = nn.Sequential(
    nn.Flatten(),
    nn.Linear(28*28, 256), nn.ReLU(),
    nn.Linear(256, 128),  nn.ReLU(),
    nn.Linear(128, 10),
)
loss_fn = nn.CrossEntropyLoss()
opt = torch.optim.Adam(model.parameters(), lr=1e-3)

# 학습 루프
for epoch in range(5):
    for xb, yb in ld:
        pred = model(xb)
        loss = loss_fn(pred, yb)
        opt.zero_grad()
        loss.backward()       # 자동 역전파
        opt.step()
    print(f"epoch {epoch} loss {loss.item():.3f}")
여기까지 마치면 머신러닝의 핵심 흐름(데이터 → 손실 → 최적화 → 모델)을 모두 본 것입니다. 다음 단계는 로봇공학특론 강의의 신경망 활용편이나, CNN·Transformer 같은 심화 주제로 이어집니다.
