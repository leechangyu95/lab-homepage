1. 비지도학습이란

지도학습과 달리 정답 $y$가 없습니다. 데이터 $X$만 보고 비슷한 것끼리 묶거나, 핵심 축을 찾는 게 목표입니다.

2. K-Means

알고리즘:

  1. K개 중심점(centroid)을 무작위로 놓는다.
  2. 각 점을 가장 가까운 중심점에 할당.
  3. 각 군집의 평균으로 중심점을 갱신.
  4. 변화가 없을 때까지 2–3 반복.

최소화하려는 손실(이너시아):

$$ J = \sum_{i=1}^n \min_{j} \|\mathbf{x}_i - \mathbf{c}_j\|^2 $$

K를 모를 때는 엘보 방법: K를 1부터 늘려가며 $J$를 그리고, 꺾이는 지점을 K로 선택합니다.

3. DBSCAN

"점이 충분히 몰려 있는 곳을 군집"으로 봅니다.

  • eps: 이웃 반경
  • min_samples: 핵심 점이 되기 위한 최소 이웃 수

K-Means가 못 찾는 moons·circles 같은 비구형 군집도 잘 찾고, 이상치는 자동으로 라벨 -1로 표시합니다.

4. PCA (주성분 분석)

고차원 데이터의 정보를 가능한 한 보존하면서 차원을 줄이는 기법입니다.

$$ \mathbf{X}_\text{pc} = \mathbf{X} \mathbf{W}, \quad \mathbf{W} = [\mathbf{v}_1, \mathbf{v}_2, \dots] $$

$\mathbf{v}_i$는 분산이 가장 큰 방향(공분산 행렬의 고유벡터)입니다.

  • PC1 = 데이터가 가장 길게 늘어선 방향
  • PC2 = 그 다음으로 긴 직교 방향
  • 설명 분산 비율(explained variance ratio)로 PC가 얼마나 정보를 담는지 확인

🎮 인터랙티브: K-Means 라이브

K를 조절하고 한 스텝씩 진행해 보세요. 중심점(★)이 어떻게 이동하는지 관찰합니다.

5. 코드 예제

from sklearn.cluster import KMeans, DBSCAN
from sklearn.decomposition import PCA

# K-Means + 엘보
import matplotlib.pyplot as plt
inertias = []
for k in range(1, 11):
    km = KMeans(n_clusters=k, n_init=10, random_state=42).fit(X)
    inertias.append(km.inertia_)
plt.plot(range(1,11), inertias, 'o-'); plt.xlabel('K'); plt.show()

# DBSCAN — 비구형 군집 + 이상치
db = DBSCAN(eps=0.3, min_samples=5).fit(X)
print(set(db.labels_))   # {-1, 0, 1, 2}  (-1은 노이즈)

# PCA — 고차원 → 2차원 시각화
pca = PCA(n_components=2)
X_2d = pca.fit_transform(X)
print(pca.explained_variance_ratio_)   # 예: [0.72, 0.18]
