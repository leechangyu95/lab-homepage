Week 9. 합성곱 신경망 (CNN) 기초

이미지에는 격자 구조가 있습니다. CNN은 그 구조를 활용하는 신경망입니다.

목차

  1. 왜 MLP는 이미지에 약한가
  2. 합성곱 연산
  3. 패딩, 스트라이드, 출력 크기
  4. 풀링
  5. 인터랙티브 필터 데모

1. 왜 MLP가 이미지에 약한가

28×28 = 784개의 픽셀을 펼쳐 MLP에 넣으면 옆 픽셀끼리 가깝다는 사실이 사라집니다. 또한 같은 사물이 약간만 옆으로 이동해도 완전히 다른 입력으로 보입니다.

2. 합성곱(Convolution) 연산

작은 필터(예: 3×3)가 이미지 위를 슬라이딩하면서 원소별 곱의 합을 계산합니다.

$$ O[i,j] = \sum_u \sum_v K[u,v]\, I[i+u, j+v] $$

같은 필터가 모든 위치에 적용되므로 가중치 공유(weight sharing)가 일어나고, 이동 불변성이 자연스럽게 생깁니다.

3. 출력 크기 공식

$$ H_\text{out} = \left\lfloor \frac{H_\text{in} - K + 2P}{S} \right\rfloor + 1 $$
  • $K$: 커널 크기, $P$: 패딩, $S$: 스트라이드
  • 패딩을 $P=(K-1)/2$로 두면 출력 크기가 입력과 같아집니다("same padding").

4. 풀링 (Pooling)

2×2 영역의 최댓값을 뽑는 max pooling이 가장 흔합니다. 크기가 절반으로 줄고, 작은 위치 이동에 더 둔감해집니다.

🎮 인터랙티브: 합성곱 필터 만져보기

3×3 필터의 9개 값을 클릭으로 바꾸면 오른쪽 출력 이미지가 즉시 변합니다. 프리셋으로 엣지·블러를 시험해보세요.

중앙 셀 클릭: +1 / 우클릭: -1

관찰: 수직 엣지 필터(Sobel x)는 세로선만, 수평 엣지 필터는 가로선만 강하게 반응합니다. CNN은 이런 필터를 스스로 학습합니다.

5. PyTorch 코드

import torch.nn as nn, torch.nn.functional as F

class FashionCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 32, kernel_size=3, padding=1)
        self.pool  = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(32, 64, kernel_size=3, padding=1)
        self.fc1   = nn.Linear(64*7*7, 128)
        self.fc2   = nn.Linear(128, 10)
    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))   # 28→14
        x = self.pool(F.relu(self.conv2(x)))   # 14→7
        x = x.view(x.size(0), -1)
        x = F.relu(self.fc1(x))
        return self.fc2(x)
