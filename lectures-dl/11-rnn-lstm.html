Week 11. RNN과 LSTM — 시퀀스 모델

시간 순서가 중요한 데이터(문장, 음성, 주가)를 위한 신경망입니다.

목차

  1. 왜 시퀀스 모델인가
  2. RNN의 핵심 아이디어
  3. RNN의 한계: 장기 의존성
  4. LSTM과 게이트
  5. 텍스트 임베딩
  6. 인터랙티브 시퀀스 데모

1. 왜 시퀀스 모델인가

"나는 어제 영화를 보고 ___ 했다."의 빈칸을 채우려면 앞 문장 전체를 기억해야 합니다. 이미지처럼 한 번에 들어오는 데이터와 달리, 텍스트와 시계열은 순서가 정보입니다.

2. RNN (Recurrent Neural Network)

입력을 한 단어씩 처리하면서, 숨은 상태(hidden state) $h_t$를 다음 스텝으로 넘겨줍니다.

$$ h_t = \tanh(W_h h_{t-1} + W_x x_t + b) $$

마치 사람이 책을 읽으며 머릿속에 줄거리를 누적하는 것과 같습니다.

3. RNN의 한계

긴 문장을 처리하면 앞쪽 정보가 거의 사라집니다(기울기 소실). "10페이지 전에 나온 단서"는 잊어버립니다.

4. LSTM (Long Short-Term Memory)

LSTM은 hidden state 외에 셀 상태(cell state) $C_t$를 따로 두고, 세 개의 게이트로 정보를 조절합니다.

$$ f_t = \sigma(W_f [h_{t-1}, x_t] + b_f) \quad \text{(망각 게이트)} $$ $$ i_t = \sigma(W_i [h_{t-1}, x_t] + b_i) \quad \text{(입력 게이트)} $$ $$ o_t = \sigma(W_o [h_{t-1}, x_t] + b_o) \quad \text{(출력 게이트)} $$ $$ C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t $$

망각 게이트가 "잊을지 말지"를, 입력 게이트가 "새로 기억할지"를 결정합니다. 덕분에 수백 스텝 전 정보도 보존할 수 있습니다.

5. 단어 임베딩 (Embedding)

단어를 정수 인덱스로 바꾼 뒤, 학습 가능한 행렬에서 행을 뽑아 밀집 벡터로 만듭니다.

$$ \mathbf{e}_i = E[w_i], \quad E \in \mathbb{R}^{|V| \times d} $$

비슷한 의미의 단어는 학습 후 비슷한 벡터로 자리 잡습니다.

🎮 인터랙티브: LSTM 게이트 시각화

입력 시퀀스가 들어올 때 망각·입력·출력 게이트의 값이 어떻게 변하는지 보여줍니다. 시퀀스를 직접 입력해 보세요.

관찰: 망각 게이트 $f$가 0에 가까우면 셀이 리셋되고, 1에 가까우면 이전 정보가 유지됩니다.

6. PyTorch 코드 — 영화 리뷰 감성 분류

import torch.nn as nn

class LSTMClassifier(nn.Module):
    def __init__(self, vocab_size, embed_dim=128, hidden_dim=64):
        super().__init__()
        self.embedding = nn.Embedding(vocab_size, embed_dim, padding_idx=0)
        self.lstm = nn.LSTM(embed_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, 1)

    def forward(self, x):
        e = self.embedding(x)                  # (B, T, embed)
        out, (h, c) = self.lstm(e)             # h: (1, B, hidden)
        return self.fc(h.squeeze(0)).squeeze(1)

model = LSTMClassifier(vocab_size=10002)
loss  = nn.BCEWithLogitsLoss()
opt   = torch.optim.Adam(model.parameters(), lr=1e-3)
