Week 11. RNN과 LSTM

시간이 흐르는 데이터 — 말, 음악, 주가, 센서 신호 — 를 다루는 신경망. 그리고 그것을 실용화한 LSTM의 게이트 구조.

이번 주에 배우는 것

  1. 시퀀스 데이터의 특징
  2. vanilla RNN과 BPTT
  3. 장기 의존성 문제
  4. LSTM의 세 게이트
  5. GRU — 더 간단한 대안

1. 시퀀스 데이터

이미지는 위치가 중요하지만, 시퀀스는 순서가 중요합니다. "나는 학교에 갔다"와 "갔다 학교에 나는"은 단어가 같아도 의미가 다릅니다. 입력 길이가 가변적이라는 점도 큰 차이.

2. Vanilla RNN

핵심 아이디어는 은닉 상태 $h_t$를 다음 시점으로 넘기는 것. 같은 가중치를 매 시점 반복해 사용합니다.

$$ h_t = \tanh(W_h h_{t-1} + W_x x_t + b) $$ $$ y_t = W_y h_t $$

학습은 시간을 따라 펼친 뒤 보통 역전파를 적용합니다 — 이를 BPTT(Backpropagation Through Time)라고 부릅니다.

🎮 인터랙티브: RNN 펼침 시각화

같은 셀이 시간을 따라 5번 반복되는 모습을 봅니다. 슬라이더로 시점을 옮기면 그 순간의 은닉 상태가 강조됩니다.

3. 장기 의존성 문제

"나는 어제 도서관에서 친구들과 ... 책을 읽었다." 마지막 동사를 결정하는 정보(주어 "나는")가 아주 멀리 있을 수 있습니다. vanilla RNN은 같은 가중치를 반복 곱하므로, 시간 차이가 멀어지면 기울기가 지수적으로 사라지거나 폭발합니다.

4. LSTM

1997년 Hochreiter·Schmidhuber가 제안한 LSTM(Long Short-Term Memory)은 "정보의 고속도로"인 셀 상태 $C_t$를 추가하고, 세 개의 게이트로 그것을 제어합니다.

  • 망각 게이트(forget): 과거 셀 상태에서 무엇을 지울지.
  • 입력 게이트(input): 현재 정보 중 무엇을 새로 쓸지.
  • 출력 게이트(output): 셀 상태에서 무엇을 외부에 보일지.
$$ f_t = \sigma(W_f [h_{t-1}, x_t]), \quad i_t = \sigma(W_i [h_{t-1}, x_t]), \quad o_t = \sigma(W_o [h_{t-1}, x_t]) $$ $$ \tilde{C}_t = \tanh(W_C [h_{t-1}, x_t]), \quad C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t $$ $$ h_t = o_t \odot \tanh(C_t) $$

🎮 인터랙티브: LSTM 게이트 값 조절

세 게이트 값을 직접 움직이며 셀 상태가 어떻게 변하는지 보세요. 망각이 0이면 과거를 완전히 지우고, 입력이 0이면 새 정보를 받지 않습니다.

5. GRU — 더 간단한 대안

2014년 Cho 외가 제안한 GRU(Gated Recurrent Unit)는 LSTM의 세 게이트를 두 개로 줄였습니다. 성능은 비슷하면서 파라미터가 적어 작은 데이터셋에서 자주 쓰입니다.

7. 코드 예제 (PyTorch)

import torch.nn as nn

class LSTMNet(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.lstm = nn.LSTM(input_size, hidden_size, batch_first=True)
        self.fc = nn.Linear(hidden_size, output_size)
    def forward(self, x):
        out, (h, c) = self.lstm(x)
        return self.fc(out[:, -1, :])
