1. KNN 다시 보기

1주차에 만난 KNN을 한 단계 더 깊이 봅니다. KNN의 매력은 간단함이지만, 다음 한계도 명심해야 합니다.

  • 차원의 저주 — 특성이 많아질수록 모든 점이 비슷하게 멀어집니다.
  • 예측 비용 — 학습은 빠르지만 매 예측마다 모든 데이터와 거리를 재야 함.
  • 스케일 민감 — 표준화가 필수.

가중치 KNN을 쓰면 가까운 이웃에 더 큰 표를 줘서 K가 클 때의 둔감함을 완화할 수 있습니다.

2. SVM — 마진을 최대화하라

SVM(Support Vector Machine)은 두 클래스를 가르는 직선 중에서 가장 넓은 도로(margin)를 만드는 직선을 고릅니다. 도로 위에 닿는 점들이 서포트 벡터입니다.

$$ \min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^\top \mathbf{x}_i + b) \ge 1 $$

현실의 데이터는 깨끗하게 분리되지 않으므로 슬랙 변수를 허용한 소프트 마진 SVM을 씁니다. 파라미터 $C$가 위반을 얼마나 허용할지 정합니다 — 작으면 마진이 넓어지고(더 부드러움), 크면 마진이 좁아지며 과적합 위험이 커집니다.

🎮 인터랙티브: SVM 마진 (소프트 vs 하드)

두 가우시안 클러스터를 분리하는 SVM의 마진을 그립니다. C 슬라이더로 마진의 폭이 어떻게 달라지는지 봅니다(시각화 단순화 버전).

커널 트릭. 데이터가 직선으로 안 갈리면 RBF 같은 커널 함수로 고차원 공간에 매핑한 뒤 그 공간에서 직선을 긋습니다. 실제로 고차원으로 가지 않고 내적만 계산하기 때문에 "트릭"입니다.

3. 결정트리

"먼저 길이가 20cm 이상인가? 그렇다면 무게가 300g 이상인가? ..." 이런 식으로 질문을 가지치며 분류하는 모델입니다. 인간이 이해하기 쉽다는 큰 장점이 있습니다.

각 노드에서 어떤 질문이 가장 좋은지는 지니 불순도나 엔트로피로 측정합니다.

$$ G = 1 - \sum_k p_k^2, \quad H = -\sum_k p_k \log p_k $$

분할 후 불순도가 가장 많이 줄어드는 질문을 고릅니다(정보이득).

🎮 인터랙티브: 결정트리 영역 분할

2D 데이터를 단계별로 수직/수평선으로 나눕니다. 깊이가 깊어질수록 영역이 잘게 쪼개집니다.

4. 랜덤포레스트

나무 하나는 약하지만 백 그루가 모이면 숲이 됩니다. 랜덤포레스트는 다음 두 가지 무작위성을 더해 여러 트리를 만들고 다수결로 분류합니다.

  • 배깅(Bagging) — 데이터를 부트스트랩 샘플링.
  • 특성 무작위화 — 각 분할에서 일부 특성만 후보로.

과적합에 강하고, 별도 튜닝 없이도 괜찮은 성능을 내는 "기본기" 모델입니다.

5. 코드 예제

from sklearn.svm import SVC
from sklearn.tree import DecisionTreeClassifier
from sklearn.ensemble import RandomForestClassifier

svm = SVC(C=1.0, kernel='rbf').fit(X_tr, y_tr)
dt  = DecisionTreeClassifier(max_depth=4).fit(X_tr, y_tr)
rf  = RandomForestClassifier(n_estimators=100).fit(X_tr, y_tr)

for m in [svm, dt, rf]:
    print(type(m).__name__, m.score(X_te, y_te))
