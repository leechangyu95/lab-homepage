Week 4. 회귀 2 — 다중회귀와 로지스틱 회귀

여러 특성을 한 번에 다루고, 직선 회귀를 분류기로 변신시킵니다.

목차

1. 다중 회귀

특성이 한 개가 아니라 여러 개라면, 각 특성마다 가중치 $w_i$를 두면 됩니다.

$$ \hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_p x_p + b $$

벡터/행렬로 쓰면 $\hat{y} = \mathbf{w}^\top \mathbf{x} + b$로 단순합니다.

2. 규제(Regularization): Ridge & Lasso

특성이 많고 데이터가 적으면 모델이 가중치를 멋대로 키워 과적합합니다. 이를 막기 위해 손실함수에 가중치 크기 벌점을 더합니다.

$$ \text{Ridge: } J = \text{MSE} + \lambda \sum_i w_i^2 $$ $$ \text{Lasso: } J = \text{MSE} + \lambda \sum_i |w_i| $$
  • Ridge (L2) — 가중치를 작게 유지. 부드러움.
  • Lasso (L1) — 일부 가중치를 정확히 0으로 만들어 자동 특성 선택.

3. 시그모이드 함수

선형 회귀의 출력은 $-\infty \sim +\infty$ 어떤 값도 될 수 있습니다. 이걸 0과 1 사이의 확률로 짜내려면 시그모이드를 사용합니다.

$$ \sigma(z) = \frac{1}{1 + e^{-z}} $$

4. 로지스틱 회귀 (Logistic Regression)

이름엔 "회귀"가 들어가지만 실제로는 분류기입니다.

$$ P(y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x} + b) $$

출력이 0.5 이상이면 클래스 1, 미만이면 0으로 분류합니다. 손실 함수로는 교차 엔트로피를 씁니다.

$$ \mathcal{L} = -\bigl[y \log \hat{y} + (1-y)\log(1-\hat{y})\bigr] $$

🎮 인터랙티브: 시그모이드 & 로지스틱 결정 경계

왼쪽: 시그모이드 곡선($w$ 크기에 따라 가팔라짐). 오른쪽: 2D 데이터에서 로지스틱 회귀가 학습한 결정 경계입니다.

5. 코드 예제

from sklearn.linear_model import LogisticRegression, Ridge, Lasso

# 다중 회귀 + Ridge 규제
ridge = Ridge(alpha=1.0)
ridge.fit(X_train, y_train)
print(ridge.score(X_test, y_test))

# 로지스틱 회귀 (이진 분류)
log = LogisticRegression()
log.fit(X_train, y_train)
print(log.predict([[5.5, 30]]))                  # 0 또는 1
print(log.predict_proba([[5.5, 30]]))            # [P(0), P(1)]
print(log.coef_, log.intercept_)
