Week 4. 회귀 2: 다중회귀, 로지스틱, 소프트맥스

변수가 여러 개로 늘어나고, 출력이 0/1이 되면, 출력이 여러 클래스로 늘어나면 — 회귀의 세 가지 진화.

이번 주에 배우는 것

  1. 다중 변수 선형회귀
  2. 벡터화 표기와 행렬
  3. 이진 분류와 시그모이드

1. 다중 변수 선형회귀

입력이 한 개($x$)에서 여러 개($x_1, x_2, \dots, x_n$)로 늘어나도 본질은 같습니다.

$$ \hat{y} = w_1 x_1 + w_2 x_2 + \dots + w_n x_n + b = \mathbf{w}^\top \mathbf{x} + b $$

가중치 벡터 $\mathbf{w}$가 각 입력의 "중요도"를 학습합니다. 비용함수도 같은 MSE를 씁니다.

🎮 인터랙티브: 두 변수의 가중치 효과

$\hat{y} = w_1 x_1 + w_2 x_2$의 예측 평면을 색으로 그립니다. 두 가중치를 움직이며 평면이 어떻게 기우는지 보세요.

2. 시그모이드와 이진 분류

출력이 "예/아니오", "스팸/정상", "환자/정상"처럼 0이나 1이라면 직선 출력은 부적절합니다. 출력을 0~1로 짓이기는 함수가 필요한데, 가장 유명한 것이 시그모이드입니다.

$$ \sigma(z) = \frac{1}{1 + e^{-z}}, \quad z = \mathbf{w}^\top \mathbf{x} + b $$

$\sigma(z)$를 "1일 확률"로 해석합니다. 임계값 0.5를 기준으로 분류하면 결정경계는 $\mathbf{w}^\top \mathbf{x} + b = 0$인 직선이 됩니다.

🎮 인터랙티브: 시그모이드 곡선

$z = wx + b$의 $w$와 $b$를 움직이며 시그모이드의 모양과 위치 변화를 관찰하세요.

3. 로지스틱 회귀의 비용함수

로지스틱 회귀에 MSE를 그대로 쓰면 비용 표면이 울퉁불퉁해집니다(비볼록). 대신 크로스 엔트로피(log loss)를 씁니다.

$$ J(\mathbf{w}) = -\frac{1}{N} \sum_{i=1}^{N} \big[ y_i \log \hat{y}_i + (1-y_i) \log (1-\hat{y}_i) \big] $$

이 함수는 시그모이드 출력에 대해 볼록함수가 되어 경사하강이 골짜기를 찾기 쉬워집니다.

4. 2D 로지스틱 결정경계

🎮 인터랙티브: 2D 결정경계 학습

두 가우시안 클러스터(빨강/파랑)를 분리하는 직선을 로지스틱 회귀로 학습합니다. "학습" 버튼을 누르면 경사하강이 돌아갑니다.

5. 소프트맥스 — 여러 클래스 분류

클래스가 셋 이상이면 시그모이드 대신 소프트맥스를 씁니다. $K$개의 점수 $z_1, \dots, z_K$를 받아 합이 1인 확률 분포로 변환합니다.

$$ p_k = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}} $$

가장 큰 점수 클래스가 자연스럽게 가장 큰 확률을 갖되, 모든 클래스가 0보다 큰 부드러운 출력을 줍니다.

🎮 인터랙티브: 소프트맥스 막대그래프

세 클래스의 점수 $z$를 슬라이더로 조절하면 소프트맥스 확률이 어떻게 변하는지 보입니다.

6. 코드 예제

from sklearn.linear_model import LogisticRegression
import numpy as np

# 두 특성, 이진 라벨
X = np.array([[2,3],[3,3],[4,5],[1,1],[2,1],[3,2]])
y = np.array([1,1,1,0,0,0])

clf = LogisticRegression().fit(X, y)
print(clf.coef_, clf.intercept_)
print(clf.predict_proba([[3,3]]))  # [[0.07, 0.93]]
